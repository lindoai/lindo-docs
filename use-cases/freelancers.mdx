## A Day in the Life

### Morning: New Client Kickoff

**9:00 AM** — Client call. Sarah runs a photography studio. She needs a portfolio site.

**9:30 AM** — You have everything: business name, services, style preferences, a few reference sites she likes.

**9:35 AM** — Open Lindo.ai. Enter the prompt:

```
Portfolio website for Sarah Chen Photography.
Wedding and portrait photographer in Austin, Texas.
Elegant, minimal design with large image galleries.
Soft neutral colors, sophisticated typography.
```

**9:37 AM** — Full website generated. Hero, about, services, gallery, contact.

**10:30 AM** — After refining with AI chat (adjusting copy, tweaking gallery layout, adding testimonials), you send the preview link.

**11:00 AM** — Sarah loves it. A few minor tweaks.

**11:30 AM** — Published. Invoice sent. Done.

**Total time: 2.5 hours. Your rate: $500. Effective hourly: $200.**

## Three Ways to Deliver

<Tabs>
  <Tab title="You Maintain">
    **Best for:** Clients who want hands-off management
    
    - Keep client in your workspace
    - Handle all updates yourself
    - Charge monthly retainer ($99-299/mo)
    - Build predictable recurring revenue
    
    **Pros:** Steady income, full control
    **Cons:** Ongoing time commitment
  </Tab>
  <Tab title="Client Self-Service">
    **Best for:** Clients who want to make their own updates
    
    - Enable Agentic Mode
    - Set monthly credit limit (50-100)
    - Client edits with AI assistance
    - You handle major changes only
    
    **Pros:** Less support work, clients feel empowered
    **Cons:** Need to set clear boundaries
  </Tab>
  <Tab title="Export & Handoff">
    **Best for:** One-time projects
    
    - Build the site
    - Export as ZIP file
    - Client hosts elsewhere
    - Project complete, move on
    
    **Pros:** Clean break, no ongoing commitment
    **Cons:** No recurring revenue
  </Tab>
</Tabs>

## Pro Tips

<AccordionGroup>
  <Accordion title="Speed Up Every Project" icon="bolt">
    - **Use templates:** Start from industry-specific templates, customize from there
    - **Reference sites:** Use the URL tool to match client's inspiration
    - **Save your work:** Turn successful projects into reusable templates
    - **Batch similar clients:** Restaurant this week, real estate next week
  </Accordion>
  <Accordion title="Maximize Your Margins" icon="dollar-sign">
    - **Standardize packages:** Clear scope = fewer surprises
    - **Limit revisions:** Include specific number in every package
    - **Charge for extras:** Rush delivery, additional pages, custom features
    - **Upsell maintenance:** Every project should become a retainer
  </Accordion>
  <Accordion title="Manage Client Expectations" icon="comments">
    - **Be transparent:** "I use AI tools to work faster and deliver better results"
    - **Show the process:** Clients love seeing AI in action
    - **Set boundaries:** Define what's included before starting
    - **Document everything:** Use CRM notes for every client interaction
  </Accordion>
</AccordionGroup>
