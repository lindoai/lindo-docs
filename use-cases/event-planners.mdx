## Example: Wedding Planner

**Prompt:**
```
Website for Enchanted Events, luxury wedding planning in the Hamptons.
Full-service planning, partial planning, and day-of coordination.
Specializing in outdoor garden weddings and estate events.
Featured in Vogue Weddings and The Knot.
Packages starting at $15,000.
```

**Result:**
- Dreamy hero with wedding imagery
- Services breakdown
- Portfolio gallery
- Testimonials from couples
- Press features
- Inquiry form
- Instagram integration
