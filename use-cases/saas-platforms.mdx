## Technical Integration

### Workspace Embedding

```javascript
// Initialize embedded workspace
const workspace = new LindoEmbed({
  container: '#website-builder',
  workspaceId: 'your-workspace-id',
  userId: currentUser.id,
  theme: 'light', // or 'dark'
  features: {
    templates: true,
    aiChat: true,
    export: false // control what users can do
  }
});

// Listen for events
workspace.on('website.created', (data) => {
  // Update your database
  saveWebsiteToUser(currentUser.id, data.websiteId);
});

workspace.on('website.published', (data) => {
  // Trigger your workflows
  notifyUser(currentUser.id, 'Website published!');
});
```

### Webhook Events

```json
{
  "event": "website.published",
  "timestamp": "2024-01-15T10:30:00Z",
  "data": {
    "websiteId": "ws_abc123",
    "userId": "user_xyz789",
    "domain": "client-site.com",
    "pages": 5
  }
}
```
