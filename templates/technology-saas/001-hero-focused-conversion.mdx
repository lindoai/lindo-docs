## Overview

A conversion-focused landing page template designed for SaaS products. The minimal design keeps attention on your value proposition while the emerald color palette adds a professional, trustworthy feel.

## Key Highlights

- Minimal monochrome design system
- Hero section optimized for conversions
- Feature showcase with accordion layout
- Testimonial carousel with asymmetric design
- Light mode default theme
