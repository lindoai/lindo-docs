After generating your website, use the AI Chat and editor tools to make it perfect.

## Opening the Editor

1. In your workspace, find your website in the "Your Websites" grid
2. Click the website card
3. The Website Editor opens with AI Chat on the left and preview on the right

## Editing with AI Chat

The fastest way to make changes is through the AI Chat panel.

**Change text content:**
```
Update the hero headline to "Freshly Roasted, Locally Loved"
```

**Add a new section:**
```
Add a testimonials section with 3 customer reviews
```

**Modify styling:**
```
Make the buttons rounded with a hover animation
```

**Replace images:**
```
Change the hero image to show coffee beans being roasted
```

## Selecting Elements to Edit

For targeted edits:

1. Switch to **Select** mode in the preview header
2. Click on any element in the preview
3. The element is highlighted
4. Type your edit request in the AI Chat
5. The AI modifies just that element

## Using the Theme Panel

For global design changes:

1. Click the **Theme** icon in the preview header
2. Adjust settings:
   - **Typography** - Heading and body fonts
   - **Colors** - Primary, secondary, background, text
   - **Buttons** - Roundness, colors, borders
   - **Links** - Link colors and hover states
3. Changes apply across all pages

## Editing Page Settings

1. Click the **Page Settings** icon in the preview header
2. Configure:
   - Page URL path
   - SEO title and description
   - Social sharing image
   - Page lock (authentication)

## Undo and Redo

Made a mistake?
- **Undo**: Click the undo arrow or press `Cmd/Ctrl + Z`
- **Redo**: Click the redo arrow or press `Cmd/Ctrl + Shift + Z`

## Page History

View and restore previous versions:
1. Click the **History** icon in the preview header
2. See all saved versions with timestamps
3. Click any version to preview it
4. Click **Restore** to revert to that version

## Saving and Publishing

- **Save Draft**: Click **Save** to save without publishing
- **Publish**: Click **Publish** to make changes live

## Preview on Different Devices

Test responsiveness:
1. Click the device icons in the preview header
2. Switch between Desktop, Tablet, and Mobile views
3. Make sure your site looks good on all devices
