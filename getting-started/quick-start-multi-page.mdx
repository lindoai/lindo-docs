Build a complete website with multiple pages and navigation.

## What You'll Build

A multi-page website with:
- Home page with hero and key sections
- About page
- Services/Products page
- Contact page
- Navigation header linking all pages

<Steps>
  <Step title="Write Your Prompt">
    In the AI Prompt Builder, describe your full website:

    ```
    Website for [BUSINESS NAME], a [BUSINESS TYPE] in [LOCATION].

    Pages:
    - Home: Hero, services overview, testimonials, CTA
    - About: Company story, team, values
    - Services: Detailed service offerings with pricing
    - Contact: Contact form, location map, hours

    [DESIGN STYLE] design, [COLOR] colors.
    ```

    **Example:**
    ```
    Website for Greenleaf Landscaping, a landscaping company in Austin, Texas.

    Pages:
    - Home: Hero with garden photo, services overview, testimonials, CTA
    - About: Company story since 2010, team of 12, our values
    - Services: Lawn care, garden design, tree trimming with pricing
    - Contact: Contact info, service area map, business hours

    Modern clean design, green colors.
    ```
  </Step>
  <Step title="Select Multi-Page">
    1. Click the page structure dropdown
    2. Select **multi-page**
  </Step>
  <Step title="Add Tools">
    Click **(+)** and add:
    - **Mode**: Light (recommended for business sites)
    - **Color Palette**: Match your brand
  </Step>
  <Step title="Generate">
    Click **↑** to generate. Multi-page sites take slightly longer (1-2 minutes).
  </Step>
  <Step title="Review All Pages">
    1. Open your website in the editor
    2. Go to the **Pages** tab
    3. Click each page to review
    4. Check the navigation links work
  </Step>
  <Step title="Refine Each Page">
    Edit pages individually:

    **Home page:**
    ```
    Add a "Why Choose Us" section with 4 benefits
    ```

    **About page:**
    ```
    Add team member photos in a grid
    ```

    **Services page:**
    ```
    Add a pricing table comparing packages
    ```

    **Contact page:**
    ```
    Add business hours and phone number
    ```
  </Step>
  <Step title="Publish All Pages">
    1. Go to each page
    2. Click **Publish**
    3. Or publish all at once from Pages tab
  </Step>
</Steps>
