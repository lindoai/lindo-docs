Your workspace dashboard is the command center for all your Lindo.ai activities.

## Dashboard Layout

The dashboard has four main areas:

### Header (Top)

- **Logo** - Your brand logo (or Lindo.ai logo)
- **Workspace Info** - Your workspace name
- **Language Selector** - Change interface language (globe icon)
- **User Menu** - Account options, support, sign out

### Sidebar (Left)

Navigation to key features:
- **CRM** - Client management (agency plans)
- **Workflows** - Background automation
- **Analytics** - Aggregated website data
- **Settings** - Workspace configuration

The sidebar collapses on smaller screens.

### Main Area (Center)

**AI Prompt Builder**
- Large textarea for describing websites
- Toolbar with AI tools (+)
- Page structure selector (single/multi-page)
- Submit button

**Your Websites**
- Grid of website cards
- Search input to filter
- Click any card to open the editor

### Templates Panel (Right)

- Browse templates by category
- Preview before using
- Click to start from a template
- Expand button for full view

## Key Actions

**Create a Website**
1. Type in the AI Prompt Builder
2. Click submit (↑)

**Open a Website**
1. Find it in "Your Websites" grid
2. Click the card

**Use a Template**
1. Browse the Templates panel
2. Click a template to use it

**Access Settings**
1. Click Settings in the sidebar
2. Or use keyboard shortcut

## Website Card Actions

Hover over any website card to see actions:
- **Edit** - Open in editor
- **Rename** - Change website name
- **Clone** - Duplicate the website
- **Export** - Download as ZIP
- **Delete** - Remove permanently
- **Save as Template** - Create reusable template

## User Menu Options

Click your avatar in the header:
- **Support** - Get help
- **Feedback** - Submit feedback
- **Community** - Join Slack community
- **Hire Expert** - Get professional help
- **Sign Out** - Log out

## Keyboard Shortcuts

- `Cmd/Ctrl + 1-6` - Switch settings tabs
