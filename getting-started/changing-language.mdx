Lindo.ai supports 11 interface languages. Change the language to use the platform in your preferred language.

## Supported Languages

| Language | Code |
|----------|------|
| English | en |
| Spanish | es |
| French | fr |
| German | de |
| Italian | it |
| Portuguese | pt |
| Dutch | nl |
| Polish | pl |
| Turkish | tr |
| Arabic | ar |
| Japanese | ja |

## How to Change Language

1. Look for the **globe icon** in the header (top right area)
2. Click the globe icon
3. A dropdown appears with all available languages
4. Select your preferred language
5. The interface updates immediately

## What Changes

When you change the language:
- All interface text (buttons, labels, menus) changes
- Navigation items update
- Settings and modal text updates
- Error messages display in the new language

## What Doesn't Change

- Your website content (that's separate)
- AI-generated content language (use the Language tool for that)
- Client-facing content

## Content Language vs Interface Language

**Interface Language** (this setting)
- The language YOU see in the dashboard
- Buttons, menus, labels

**Content Language** (AI tool)
- The language of GENERATED website content
- Set when creating websites with the Language tool

## For Agencies

If you have clients:
- Set a default language in Settings → White Label → Appearance
- Clients see the interface in that default language
- Clients can change their own language preference

## RTL Support

Arabic and other right-to-left languages are fully supported:
- Interface flips to RTL layout
- Text alignment adjusts automatically
- All features work correctly
