Connect Lindo to Perplexity through the MCP Connector in the web app, the Comet browser, or Perplexity Spaces.

## Perplexity MCP Connector

Paste the remote MCP URL in Perplexity → Settings → Connectors. OAuth login — no API key needed.

```
https://mcp.lindo.ai/mcp
```

<a href="https://www.perplexity.ai/settings/connectors" target="_blank">Add Connector →</a>

## Comet Browser

Perplexity's AI browser supports MCP out of the box. Open Comet → Settings → Connectors and paste the same URL.

```
https://mcp.lindo.ai/mcp
```

Once connected, you can ask Comet to create websites, manage clients, and publish content while browsing the web — the AI has full Lindo access in any conversation.

## Perplexity Spaces

Create a Perplexity Space for your team, add the Lindo MCP connector, and share access. Everyone in the Space can build websites through Perplexity conversations using shared context and tools.

### Example Use Cases

- **Research → Website** — Ask Perplexity to research a business, then build a website based on the findings.
- **Competitive analysis → Pages** — Analyze competitors, then create an SEO page positioned against them.
- **Team knowledge base** — Keep a Space where your team can all build and manage client websites through conversation.

## Which Should I Use?

| Method | Best For |
|--------|----------|
| MCP Connector | Solo use in Perplexity web |
| Comet Browser | Research + build in one flow |
| Spaces | Team-wide access and collaboration |

## Authentication

All Perplexity integrations use OAuth. Click the Add Connector button, authorize with your Lindo account, and you're done — no API key needed.
