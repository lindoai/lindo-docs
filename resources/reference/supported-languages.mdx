Lindo.ai supports multiple languages for both the interface and content generation.

## Interface Languages

The workspace interface is available in 11 languages:

| Language | Code | Native Name |
|----------|------|-------------|
| English | en | English |
| Spanish | es | Español |
| French | fr | Français |
| German | de | Deutsch |
| Italian | it | Italiano |
| Portuguese | pt | Português |
| Dutch | nl | Nederlands |
| Polish | pl | Polski |
| Turkish | tr | Türkçe |
| Arabic | ar | العربية |
| Japanese | ja | 日本語 |

## Changing Interface Language

1. Click the globe icon in the header
2. Select your preferred language
3. Interface updates immediately

## Content Generation Languages

The AI can generate website content in virtually any language:

### Fully Supported

- English
- Spanish
- French
- German
- Italian
- Portuguese
- Dutch
- Polish
- Turkish
- Arabic
- Japanese
- Chinese (Simplified & Traditional)
- Korean
- Russian
- Hindi

### Also Available

The AI can generate content in most world languages. Quality may vary for less common languages.

## Setting Content Language

### Auto-Detect (Default)

The AI detects your prompt language and generates content in the same language.

### Manual Selection

1. Click **(+)** in the prompt builder
2. Select **Language**
3. Choose or type the language
4. AI generates in that language

## RTL Language Support

Right-to-left languages are fully supported:
- Arabic
- Hebrew
- Persian
- Urdu

Features:
- Interface flips to RTL
- Generated content uses RTL styling
- Text alignment handled automatically

## Multi-Language Websites

To create a website in multiple languages:

1. Create the primary language version
2. Clone the website
3. Use AI to translate:
```
Translate all content on this page to [Language]
```

## Language Best Practices

- **Be consistent** - Use one language per page
- **Review translations** - AI translations are good but not perfect
- **Consider SEO** - Use hreflang tags for multi-language sites
- **Test thoroughly** - Check all pages in target language
