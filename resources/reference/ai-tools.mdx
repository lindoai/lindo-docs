Reference guide for all AI tools available in the prompt builder and editor.

## Prompt Builder Tools

### Language Tool

Sets the content language for generated websites.

- **Location**: Toolbar (+) menu
- **Options**: Auto-detect or manual selection
- **Languages**: 11 supported languages
- **Effect**: All generated text in selected language

### Mode Tool

Sets the color theme for generated websites.

- **Location**: Toolbar (+) menu
- **Options**: Light, Dark
- **Effect**: Applies theme to entire website

### Color Palette Tool

Defines custom colors for the website.

- **Location**: Toolbar (+) menu
- **Options**: Primary, Secondary, Accent colors
- **Effect**: Colors applied throughout design

### Website URL Tool

Uses an existing website as design reference.

- **Location**: Toolbar (+) menu
- **Input**: Valid website URL
- **Effect**: AI considers reference in generation

## Editor AI Tools

### AI Chat

Send prompts to AI agents for editing.

- **Location**: Left panel in editor
- **Input**: Text prompts, optional images
- **Output**: Website modifications

### Prompt Enhancement

Improves your prompt for better results.

- **Location**: Chat input area
- **Trigger**: Enhance button
- **Effect**: Expands and clarifies prompt

### Reference Images

Attach images to guide AI generation.

- **Location**: Chat input area
- **Formats**: JPG, PNG, WebP
- **Effect**: AI uses images as reference

## Tool Combinations

### Effective Combinations

- Language + Mode + Prompt = Localized themed website
- Color Palette + Website URL = Branded design inspiration
- Reference Image + Block Creator = Visual-guided sections

### Best Practices

- Set language before generating
- Use color palette for brand consistency
- Reference images improve accuracy
- Enhance prompts for complex requests
