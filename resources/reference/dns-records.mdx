DNS record formats for domain setup.

## Required Records

### CNAME Records

For subdomains (www, app, sites):

| Field | Value |
|-------|-------|
| Type | CNAME |
| Name | www (or subdomain) |
| Value | cname.lindo.ai |
| TTL | 3600 (or Auto) |

### TXT Records

For domain verification:

| Field | Value |
|-------|-------|
| Type | TXT |
| Name | _verify |
| Value | [verification code] |
| TTL | 3600 (or Auto) |

## Record Examples

### Root Domain Setup

```
Type: A
Name: @
Value: [IP address]
TTL: 3600
```

### Subdomain Setup

```
Type: CNAME
Name: www
Value: cname.lindo.ai
TTL: 3600
```

### Verification

```
Type: TXT
Name: _verify
Value: lindo-verify-abc123xyz
TTL: 3600
```
