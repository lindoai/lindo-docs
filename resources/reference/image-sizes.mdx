Recommended sizes for different image types.

## Branding Images

| Image | Size | Format |
|-------|------|--------|
| Full Logo | 200x50px | PNG, SVG |
| Favicon | 32x32px | PNG, ICO |
| Login Background | 1920x1080px | JPG, PNG |

## SEO Images

| Image | Size | Format |
|-------|------|--------|
| OG Image | 1200x630px | JPG, PNG |
| Twitter Card | 1200x600px | JPG, PNG |

## Product Images

| Image | Size | Format |
|-------|------|--------|
| Product Thumbnail | 800x800px | JPG, PNG |
| Product Gallery | 1200x1200px | JPG, PNG |

## General Guidelines

### File Size

- Keep under 500KB for web
- Optimize before upload
- Use appropriate compression

### Format Selection

| Format | Best For |
|--------|----------|
| JPG | Photos |
| PNG | Graphics, transparency |
| SVG | Logos, icons |
| WebP | Modern browsers |

## Tips

- Use appropriate dimensions
- Optimize for web
- Maintain aspect ratios
- Test on all devices
