Quick reference for all keyboard shortcuts. Use `Cmd` on Mac or `Ctrl` on Windows/Linux.

## Workspace

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + 1` | Switch to Settings Tab 1 |
| `Cmd/Ctrl + 2` | Switch to Settings Tab 2 |
| `Cmd/Ctrl + 3` | Switch to Settings Tab 3 |
| `Cmd/Ctrl + 4` | Switch to Settings Tab 4 |
| `Cmd/Ctrl + 5` | Switch to Settings Tab 5 |
| `Cmd/Ctrl + 6` | Switch to Settings Tab 6 |

## Website Editor

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + S` | Save changes |
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |
| `Escape` | Close modal or panel |

## Text Editing

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + B` | Bold text |
| `Cmd/Ctrl + I` | Italic text |
| `Cmd/Ctrl + U` | Underline text |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Tab` | Move to next element |
| `Shift + Tab` | Move to previous element |
| `Enter` | Activate or select |
| `Escape` | Close or cancel |

## Tips

- **Learn the basics** — Save (`Cmd/Ctrl + S`), Undo (`Cmd/Ctrl + Z`), and Redo (`Cmd/Ctrl + Shift + Z`) are the most useful
- **Use Tab navigation** — Faster than clicking through elements
- **Escape to close** — Quick way to dismiss modals and panels
