## Credit-Saving Tips

- **Plan before generating** — Clear prompts mean fewer revisions
- **Use templates** — Start from templates instead of scratch
- **Batch edits** — Make multiple changes in one request
- **Review before publishing** — Catch issues early to avoid re-editing

<Note>
Credits are deducted on successful completion. Failed operations don't consume credits.
</Note>
