Reference guide for supported file formats across Lindo.ai features.

## Image Formats

### Supported Formats

| Format | Extension | Best For |
|--------|-----------|----------|
| JPEG | .jpg, .jpeg | Photos, complex images |
| PNG | .png | Graphics with transparency |
| WebP | .webp | Modern web images |
| GIF | .gif | Simple animations |
| SVG | .svg | Logos, icons |

### Recommended Uses

- **Logo**: PNG or SVG (transparent background)
- **Favicon**: PNG (32x32px)
- **Hero images**: JPEG or WebP
- **Product images**: JPEG or PNG
- **Icons**: SVG

## Document Formats

### Digital Products

| Format | Extension | Description |
|--------|-----------|-------------|
| PDF | .pdf | Documents, ebooks |
| ZIP | .zip | Compressed files |

### Import/Export

| Format | Extension | Use |
|--------|-----------|-----|
| CSV | .csv | Batch imports |
| JSON | .json | Data export |
| HTML | .html | Website export |
| ZIP | .zip | Full website export |

## Size Limits

### Images

| Type | Max Size | Recommended |
|------|----------|-------------|
| Logo | 5MB | Under 500KB |
| Favicon | 1MB | Under 50KB |
| Hero | 10MB | Under 500KB |
| Product | 5MB | Under 200KB |

### Documents

| Type | Max Size |
|------|----------|
| Digital product | 100MB |
| CSV import | 10MB |

## Image Dimensions

### Recommended Sizes

| Image Type | Dimensions |
|------------|------------|
| Full logo | 200x50px |
| Favicon | 32x32px |
| Login background | 1920x1080px |
| Hero image | 1920x1080px |
| Product image | 800x800px |
| Blog featured | 1200x630px |
| OG image | 1200x630px |

## Best Practices

### Before Uploading

1. Resize to appropriate dimensions
2. Compress to reduce file size
3. Use correct format for content type
4. Name files descriptively

### Image Optimization

- Use JPEG for photos (80-85% quality)
- Use PNG only when transparency needed
- Consider WebP for modern browsers
- Compress all images before upload

### File Naming

Good naming:
- `company-logo.png`
- `product-blue-widget.jpg`
- `hero-homepage.webp`

Avoid:
- `IMG_12345.jpg`
- `screenshot (1).png`
- `final_final_v2.jpg`
