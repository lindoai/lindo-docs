The AI generates based on your description. Vague prompts produce generic results. Specific prompts produce tailored, professional output.

## Anatomy of a Good Prompt

Include these elements:

1. **Business type** — What kind of website
2. **Target audience** — Who visits the site
3. **Key sections** — What to include
4. **Design direction** — Style, mood, colors
5. **Specific content** — Names, features, details

## Examples

### Bad Prompt
```
Restaurant website
```

### Good Prompt
```
Website for Bella Cucina, an upscale Italian restaurant in Chicago.
Target: couples and business diners, 30-55.
Sections: Hero with interior photo, About our story, Menu
(appetizers/mains/desserts/wine), Private dining, Reservations
(OpenTable link), Location and hours.
Elegant design, warm colors, dark mode.
```

### Bad Prompt
```
SaaS product page
```

### Good Prompt
```
Landing page for CloudSync, a file sync tool for remote teams.
Hero: "Your Files, Everywhere" with app screenshot.
Features: Real-time sync, version history, team sharing,
mobile apps, encryption, integrations.
Pricing: Free (5GB), Pro $10/mo (100GB), Team $8/user (unlimited).
Social proof: Company logos, 3 testimonials.
CTA: "Start Free Trial" throughout.
Modern, clean, blue accent.
```

## Tips

**Be specific about numbers**
```
❌ "some features"
✓ "6 features in a 3-column grid"
```

**Include your brand name** — The AI uses it in headlines, copy, and footer.
```
✓ "Website for TaskFlow, a project management tool"
```

**Specify what NOT to include**
```
Do NOT include: contact form, newsletter signup, blog section
```

**Mention design style**
```
✓ "Modern, minimal design with lots of whitespace"
✓ "Bold, colorful design with playful animations"
```

## Use the Prompt Builder

Click **✨ Prompt Builder** to access:
- Sample prompts by industry
- Section checkboxes
- Design style presets

Select options, click **Apply to Prompt**, then customize further.

## Leverage AI Tools

Click **(+)** to add refinement tools:
- **Language** — Content language
- **Mode** — Dark or light theme
- **Color Palette** — Brand colors
- **Website URL** — Reference an existing design

## Common Mistakes

1. **Too vague** — "Business website" gives generic results
2. **Too long** — Keep under 500 words
3. **Contradictory** — "Minimal with lots of animations" confuses the AI
4. **Missing audience** — Always mention who the site is for
5. **No structure** — List the sections you want

## Iterating on Results

First generation not perfect? Use the Block Editor to refine:

- Be specific: "Make headline larger, add subheadline"
- Reference what works: "Keep layout, change colors to blue"
- Add missing elements: "Add a testimonials section after features"
