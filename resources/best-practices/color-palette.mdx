Select colors that work for your brand.

## Color Basics

### Primary Color

Your main brand color:
- Buttons
- Links
- Highlights

### Secondary Color

Accent color:
- Hover states
- Accents
- Variety

### Background

Page background:
- White/light for most
- Dark for modern look

### Text

Default text color:
- Dark on light backgrounds
- Light on dark backgrounds

## Color Psychology

| Color | Feeling |
|-------|---------|
| Blue | Trust, professional |
| Green | Growth, nature |
| Red | Energy, urgency |
| Purple | Luxury, creative |
| Orange | Friendly, energetic |
| Black | Elegant, modern |

## Contrast

Ensure readability:
- Dark text on light backgrounds
- Light text on dark backgrounds
- Test accessibility

## Tips

- Use brand colors
- Limit to 2-3 colors
- Ensure contrast
- Test on all devices
