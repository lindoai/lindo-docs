Maximize efficiency with workflows by following these best practices for batch website and content creation.

## Planning Your Workflows

### Before Starting

1. Define clear objectives
2. Prepare all content/prompts
3. Check credit availability
4. Set realistic expectations

### Batch Size Recommendations

- **Testing**: 1-3 items
- **Small batch**: 5-10 items
- **Medium batch**: 10-25 items
- **Large batch**: 25-50 items

Start small to validate your approach.

## Writing Effective Prompts

### Be Specific

Good prompt:
```
Create a dental clinic website for "Smile Dental" with:
- Hero with appointment booking CTA
- Services: cleanings, whitening, implants, orthodontics
- Team section with 3 dentists
- Patient testimonials
- Insurance accepted section
- Contact with map
```

Avoid vague prompts:
```
Create a dental website
```

### Use Templates

Create prompt templates for similar projects:

```
Create a [INDUSTRY] website for "[BUSINESS_NAME]" with:
- Hero with [CTA_ACTION]
- Services: [SERVICE_LIST]
- About section
- Testimonials
- Contact form
```

## CSV Import Best Practices

### Prepare Your Data

1. Download sample CSV template
2. Fill in all required fields
3. Validate data before upload
4. Use consistent formatting

### CSV Structure

```csv
name,prompt,language,mode
"Site 1","Create a...",en,dark
"Site 2","Create a...",en,light
```

### Common CSV Errors

- Missing required columns
- Incorrect encoding (use UTF-8)
- Extra commas in text
- Inconsistent formatting

## Monitoring Workflows

### Check Status Regularly

- Running: In progress
- Completed: Review results
- Error: Investigate and retry

### Handle Errors

1. Note which items failed
2. Review error messages
3. Fix prompt issues
4. Retry failed items only

## Post-Workflow Review

### Quality Check

1. Open each generated website
2. Review content accuracy
3. Check design consistency
4. Note needed adjustments

### Batch Editing

For common changes across sites:

1. Identify pattern
2. Use AI agents for bulk edits
3. Or create new workflow

## Credit Management

### Estimate Usage

- Website: 30 credits each
- Blog post: 30 credits each
- Page: 30 credits each

Calculate total before starting.

### Monitor Balance

Check credits before large batches to avoid interruptions.

## Tips for Success

- Test prompts individually first
- Use consistent naming conventions
- Document successful prompts
- Schedule during off-hours
- Enable email notifications
- Review results promptly
