Keep your workspace organized for efficiency.

## Naming Conventions

### Websites

Use clear, descriptive names:
- "Client Name - Project"
- "Company - Landing Page"
- "Brand - Website v2"

### Clients

Consistent client naming:
- Company name
- Contact name
- Project identifier

## Organization Tips

### By Client

Group websites by client.

### By Project Type

- Landing pages
- Full websites
- E-commerce

### By Status

- Active projects
- Completed
- Templates

## Using Search

- Search by name
- Filter quickly
- Find specific sites

## Templates

Save common designs:
- Reuse for similar projects
- Maintain consistency
- Speed up creation

## Tips

- Name clearly
- Use consistent patterns
- Archive completed work
- Review regularly
