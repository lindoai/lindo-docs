---
title: "Designing for Mobile First"
description: "Responsive design tips"
---

Ensure your website works great on all devices.

## Why Mobile First

- 50%+ traffic is mobile
- Google prioritizes mobile
- Better user experience
- Wider reach

## Key Principles

### Simple Navigation

- Hamburger menu
- Clear labels
- Easy to tap

### Readable Text

- Large enough fonts
- Good contrast
- Short paragraphs

### Touch-Friendly

- Large tap targets
- Adequate spacing
- No hover-only features

### Fast Loading

- Optimized images
- Minimal scripts
- Quick load times

## Testing

Always check:
1. Mobile preview
2. Tablet preview
3. Desktop preview

## Common Issues

### Text Too Small

Increase font sizes for mobile.

### Buttons Too Close

Add more spacing between elements.

### Images Too Large

Optimize for mobile screens.

## Tips

- Test on real devices
- Check all pages
- Prioritize mobile experience
