description: "Common questions about search engine optimization"
---

<AccordionGroup>
  <Accordion title="Does Lindo support SEO?">
    Yes, you can configure all essential SEO settings including page titles, meta descriptions, Open Graph tags, and schema markup.
  </Accordion>

  <Accordion title="Is there a sitemap?">
    Yes, Lindo automatically generates a sitemap.xml for your website. Search engines can discover and index your pages.
  </Accordion>

  <Accordion title="Can I edit robots.txt?">
    Yes, enable manual robots.txt editing in website Settings → SEO. Customize which pages search engines can crawl.
  </Accordion>

  <Accordion title="How do I set page titles?">
    Set page titles in:
    - Page settings panel (while editing)
    - Pages tab → click page → Edit SEO
    - Each page can have a unique title
  </Accordion>

  <Accordion title="What's a good meta description?">
    A good meta description is:
    - Under 160 characters
    - Compelling and action-oriented
    - Includes relevant keywords naturally
    - Unique for each page
  </Accordion>

  <Accordion title="Can I set OG images for social sharing?">
    Yes, configure Open Graph images in page SEO settings. These images appear when your pages are shared on social media.
  </Accordion>

  <Accordion title="Are pages mobile-friendly?">
    Yes, all pages are responsive by default. Google prioritizes mobile-friendly sites, and Lindo handles this automatically.
  </Accordion>

  <Accordion title="Is the site fast?">
    Yes, Lindo optimizes for performance:
    - Static HTML generation
    - CDN delivery
    - Optimized images
    - Minimal JavaScript
  </Accordion>

  <Accordion title="Can I add schema markup?">
    Yes, add JSON-LD schema markup in website Settings → SEO. Schema helps search engines understand your content better.
  </Accordion>

  <Accordion title="How do I improve my SEO?">
    Best practices:
    1. Unique, descriptive titles per page
    2. Quality, original content
    3. Descriptive, readable URLs
    4. Internal linking between pages
    5. Mobile-friendly design (automatic)
    6. Fast loading (automatic)
  </Accordion>

  <Accordion title="How long until my site appears in Google?">
    Typically 1-4 weeks after publishing. You can speed this up by:
    - Submitting your sitemap to Google Search Console
    - Building backlinks from other sites
    - Sharing on social media
  </Accordion>

  <Accordion title="Can I connect Google Search Console?">
    Yes, verify your site with Google Search Console using the TXT record or HTML file method. This gives you search performance data.
  </Accordion>
</AccordionGroup>
