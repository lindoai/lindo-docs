---
title: "Website Editor FAQ"
description: "Common questions about the website editor"
---

<AccordionGroup>
  <Accordion title="How do I access the editor?">
    Click any website card in your workspace to open it in the editor. You can also use the "Edit" option from the website actions menu.
  </Accordion>

  <Accordion title="Can I edit without AI?">
    The editor is designed around AI editing. While you can make some manual adjustments, AI makes editing much faster and more powerful.
  </Accordion>

  <Accordion title="Are changes saved automatically?">
    Yes, changes save automatically as drafts. You won't lose work if you close the browser. Publish when you're ready to go live.
  </Accordion>

  <Accordion title="Which AI agent should I use?">
    - **Block Creator**: Add new sections
    - **Block Editor**: Modify existing sections
    - **Element Editor**: Fine-tune individual elements
    - **Tailwind Class**: Quick style changes
  </Accordion>

  <Accordion title="How do I stop AI generation?">
    Click the stop button that appears while the AI is generating. The operation cancels and no credits are consumed.
  </Accordion>
</AccordionGroup>

<AccordionGroup>
  <Accordion title="Can I undo AI changes?">
    Yes, multiple options:
    - Press Cmd/Ctrl + Z to undo
    - Use the Undo button in the toolbar
    - Restore from page history for older versions
  </Accordion>

  <Accordion title="How many pages can I have?">
    Depends on your plan:
    - **Free**: 5 pages
    - **Pro**: 20 pages
    - **Business**: 50 pages
    - **Elite/Reseller**: 100 pages
  </Accordion>

  <Accordion title="Can I duplicate pages?">
    Yes, use the duplicate action in the Pages tab. Creates an exact copy you can modify.
  </Accordion>

  <Accordion title="How do I delete a page?">
    Use the delete action in the Pages tab. Warning: this cannot be undone.
  </Accordion>

  <Accordion title="What's the difference between draft and published?">
    - **Draft**: Only visible to you and your team
    - **Published**: Live and visible to everyone on the internet
  </Accordion>

  <Accordion title="Can I unpublish a page?">
    Yes, unpublishing hides the page from public view while keeping it as a draft you can edit.
  </Accordion>

  <Accordion title="How do I preview on mobile?">
    Use the device preview buttons in the editor toolbar to see desktop, tablet, and mobile views.
  </Accordion>
</AccordionGroup>
