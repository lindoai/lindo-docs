<AccordionGroup>
  <Accordion title="What are AI agents?">
    AI agents are specialized tools that perform different editing tasks. Each agent is optimized for a specific type of work—creating new content, editing existing sections, or fine-tuning individual elements.
  </Accordion>

  <Accordion title="How do I choose the right agent?">
    Match the agent to your task:
    - **Block Creator** — Generate new sections from scratch
    - **Block Editor** — Modify existing sections
    - **Element Editor** — Fine-tune individual elements (text, buttons, images)
    - **Tailwind Class** — Quick style adjustments
    - **Blog Writer** — Create blog posts
    - **Page Builder** — Generate entire pages
  </Accordion>

  <Accordion title="Do all agents cost the same?">
    No, costs vary by agent complexity:

    | Agent | Credits |
    |-------|---------|
    | Website Builder | 10x (10 per page) |
    | Block Creator | 10 |
    | Block Editor | 10 |
    | Element Editor | 10 |
    | Tailwind Class | 10 |
    | Blog Writer | 10 |
    | Page Builder | 10 |
  </Accordion>

  <Accordion title="Why is Website Builder free?">
    The Website Builder agent generates initial business information and theme settings. It's free to help you get started quickly without consuming credits.
  </Accordion>

  <Accordion title="Can I use multiple agents on one page?">
    Yes, you can switch between agents as needed. Use Block Creator to add a section, then Block Editor to refine it, then Element Editor for final touches.
  </Accordion>

  <Accordion title="What if the AI doesn't understand my request?">
    Try these approaches:
    - Be more specific about what you want
    - Break complex requests into smaller steps
    - Reference existing elements ("make the hero headline larger")
    - Use clear, simple language
  </Accordion>

  <Accordion title="Can I undo AI changes?">
    Yes, you have multiple options:
    - Press Cmd/Ctrl + Z to undo
    - Use the Undo button in the toolbar
    - Restore from page history for older versions
  </Accordion>

  <Accordion title="How does the system choose which agent to use?">
    The system automatically selects the appropriate agent based on:
    - What you have selected (block, element, or nothing)
    - The type of request (add, edit, modify)
    - Context from your prompt
  </Accordion>

  <Accordion title="Can I manually select an agent?">
    Yes, use the agent selector in the chat panel to override the automatic selection when needed.
  </Accordion>

  <Accordion title="What's the difference between Block Editor and Element Editor?">
    **Block Editor** modifies entire sections (hero, features, testimonials). **Element Editor** changes individual elements within a section (a specific button, headline, or image).
  </Accordion>
</AccordionGroup>
