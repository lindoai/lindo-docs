<AccordionGroup>
  <Accordion title="What is the AI agent?">
    Lindo has a single, website-aware AI agent. It understands your whole site and handles every kind of task—creating pages and sections, editing content, fine-tuning individual elements, updating the header and footer, writing blog posts, and building or redesigning the entire website—all from one chat.
  </Accordion>

  <Accordion title="Do I need to pick a tool for each task?">
    No. There are no separate agents to choose between anymore. Just describe what you want—mention the section or element when it helps—and the agent figures out the right change, whether that's one element, a section, a page, or the whole site.
  </Accordion>

  <Accordion title="How much does each task cost?">
    There's no fixed per-action price. Credit usage depends on how much work the agent does and how complex the task is. Small edits (a single element, a quick style change) use far fewer credits than generating new sections, full pages, or an entire website. Credits are only deducted when a task completes successfully.
  </Accordion>

  <Accordion title="How much does creating a website cost?">
    Website creation is credit-based and scales with complexity—a larger, multi-page site uses more credits than a small single-page one. Credits are only deducted when the build completes successfully.
  </Accordion>

  <Accordion title="Can I make several changes in one session?">
    Yes. Keep chatting with the agent to add a section, refine it, then tweak a specific element—each follow-up builds on the last.
  </Accordion>

  <Accordion title="What if the AI doesn't understand my request?">
    Try these approaches:
    - Be more specific about what you want
    - Break complex requests into smaller steps
    - Reference existing elements ("make the hero headline larger")
    - Use clear, simple language
  </Accordion>

  <Accordion title="Can I undo AI changes?">
    Yes, you have multiple options:
    - Press Cmd/Ctrl + Z to undo
    - Use the Undo button in the toolbar
    - Restore from page history for older versions
  </Accordion>

  <Accordion title="How do I target a specific element vs a whole section?">
    Select an element first to scope the change to it, or describe the area in your prompt (e.g. "the hero headline" vs "the features section"). The agent applies the change at the level you point it to.
  </Accordion>
</AccordionGroup>
