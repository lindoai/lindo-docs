Definitions of key terms used in Lindo.ai.

---

## A

### Agentic Mode
The AI-powered interface for creating and editing websites. Users interact with AI agents through natural language instead of traditional builders.

### AI Agent
A specialized AI model for specific tasks. Examples: Block Creator (generates sections), Block Editor (modifies sections), Simple Blog (writes posts).

### AI Credits
Currency for AI operations. Each agent consumes credits when performing tasks. Credits reset monthly based on plan.

---

## B

### Block
A self-contained HTML section on a page. Examples: hero, features grid, testimonials, pricing table. Each block is a `<section>` element.

### Block Creator Agent
AI agent that generates new HTML sections. Costs 10 credits.

### Block Editor Agent
AI agent that modifies existing HTML sections. Costs 10 credits.

### Batch Import
Creating multiple workflows from a CSV file for bulk website generation.

---

## C

### Client (Workspace CRM)
A person or company managed in your agency workspace. Clients can have their own login, websites, and credit allocation.

### Color Palette
Predefined color schemes for generated websites. Options include Lime, Blue, Purple, Orange, etc.

### Custom Domain
Your own domain connected to a website (www.yourdomain.com) instead of the default subdomain.

---

## D

### Dark Mode
Color scheme with dark backgrounds and light text. Set using the Mode tool.

### Draft
Saved but unpublished page state. Only visible to editors, not public visitors.

### DNS Records
Domain Name System entries (CNAME, TXT, A) that connect your domain to Lindo.ai.

---

## E

### Element
Individual HTML component within a block. Examples: button, heading, image, paragraph.

### Element Editor Agent
AI agent that modifies individual HTML elements. Costs 10 credits.

---

## L

### Light Mode
Color scheme with light backgrounds and dark text. Set using the Mode tool.

### Layers Panel
Panel showing hierarchical structure of page elements for selection and organization.

---

## M

### Multi-page Website
Website with multiple pages and navigation. Contrast with single-page.

---

## P

### Page Builder Agent
AI agent that creates page structure and section definitions. Costs 10 credits.

### Preview Mode
Editor mode showing the website as visitors see it. Read-only, no editing.

### Prompt Builder
Modal interface for constructing AI prompts using samples, sections, and styles.

### Published
Live page state visible to all visitors.

---

## R

### Reseller
Plan tier for agencies reselling Lindo.ai services with full white label.

---

## S

### Select Mode
Editor mode for selecting elements without full editing. Useful for targeted AI edits.

### Single-page Website
Website with all content on one scrolling page. No navigation between pages.

### Subdomain
Default website address: yoursite.lindo.site

---

## T

### Template
Pre-built website design used as a starting point. Organized by category.

### Theme Panel
Panel for customizing design settings: colors, fonts, buttons, links.

---

## W

### White Label
Branding features letting agencies present Lindo.ai as their own platform. Includes custom domain, logo, colors.

### Workflow
Background process for creating websites. Can be single or batch (CSV import).

### Workspace
Main dashboard for managing websites, clients, and settings. Each account has one workspace.

### Workspace CRM
Client management system for agencies. Manages clients, billing, credit allocation.

### Website CRM
Customer management for a single website. Manages buyers, subscribers, form submissions.
