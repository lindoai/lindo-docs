---
title: "Editor Loading Slowly"
description: "Improve website editor performance"
---

If the website editor is loading slowly, try these solutions.

## Quick Fixes

### Clear Browser Cache

1. Open browser settings
2. Clear cached images and files
3. Reload the editor

### Try Different Browser

Test in another browser:
- Chrome
- Firefox
- Safari
- Edge

### Check Internet Connection

- Test your connection speed
- Try a wired connection
- Restart your router

## Browser Optimization

### Disable Extensions

Extensions can slow down the editor:

1. Open browser extensions
2. Disable all extensions
3. Reload editor
4. Re-enable one by one to find culprit

### Update Browser

Ensure you're using the latest version:

1. Check for browser updates
2. Install available updates
3. Restart browser

### Clear Storage

Free up browser storage:

1. Open browser settings
2. Clear site data for Lindo.ai
3. Reload and log in again

## System Optimization

### Close Other Tabs

Too many tabs consume memory:

- Close unused tabs
- Especially close other heavy apps
- Keep only essential tabs open

### Check System Resources

Monitor your computer:

- CPU usage
- Memory (RAM) usage
- Close resource-heavy applications

### Restart Computer

A fresh start can help:

1. Save your work
2. Restart computer
3. Open only the editor

## Website-Specific Issues

### Large Websites

Websites with many pages may load slower:

- Consider splitting into multiple websites
- Archive unused pages
- Optimize images

### Complex Pages

Pages with many elements:

- Simplify page structure
- Remove unused sections
- Optimize images

## Network Issues

### Slow Connection

If on slow internet:

- Wait for full load before editing
- Avoid multiple simultaneous edits
- Save frequently

### VPN Issues

VPNs can slow connections:

- Try disconnecting VPN
- Use VPN server closer to you
- Check VPN settings

## When to Contact Support

If issues persist after trying all solutions:

1. Note your browser and version
2. Describe the slowness
3. Include any error messages
4. Contact support
