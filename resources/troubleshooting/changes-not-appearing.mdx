Troubleshoot when changes don't show up.

## Common Causes

### Browser Cache

Browser stores old version.

**Solution:**
1. Hard refresh (Cmd/Ctrl + Shift + R)
2. Clear browser cache
3. Try incognito mode

### CDN Cache

Content delivery network caching.

**Solution:**
1. Wait a few minutes
2. Changes propagate automatically

### Not Published

Changes saved but not published.

**Solution:**
1. Click Publish button
2. Verify page is published

### Wrong Page

Viewing different page than edited.

**Solution:**
1. Check page URL
2. Navigate to correct page

## Quick Fixes

1. Hard refresh browser
2. Clear cache
3. Check publish status
4. Wait 5 minutes

## Tips

- Always publish changes
- Use hard refresh
- Check correct page
