Troubleshoot image display problems.

## Common Issues

### Images Not Showing

**Symptoms:**
- Broken image icons
- Empty spaces

**Solutions:**
1. Refresh the page
2. Check image URL
3. Re-upload image

### Slow Loading

**Symptoms:**
- Images take long to load

**Solutions:**
1. Optimize image size
2. Use appropriate format
3. Check connection

### Wrong Size

**Symptoms:**
- Images distorted
- Wrong dimensions

**Solutions:**
1. Use correct aspect ratio
2. Resize before upload
3. Check CSS settings

### Upload Failed

**Symptoms:**
- Can't upload images

**Solutions:**
1. Check file size (max limits)
2. Use supported format
3. Try different browser

## Supported Formats

- JPG/JPEG
- PNG
- GIF
- WebP
- SVG

## Tips

- Optimize before upload
- Use appropriate formats
- Check file sizes
