---
title: "Preview Not Loading"
description: "Troubleshooting preview issues in the editor"
---

The website preview isn't displaying in the editor.

## Symptoms

- Preview area is blank
- Loading spinner won't stop
- Error message in preview
- Preview shows old content

## Common Causes & Solutions

<Steps>
  <Step title="Browser Cache">
    **Problem:** Cached data causing issues.

    **Solution:**
    1. Hard refresh: `Cmd/Ctrl + Shift + R`
    2. Or clear browser cache
    3. Try incognito/private mode
  </Step>
  <Step title="Browser Extensions">
    **Problem:** Ad blockers or extensions interfering.

    **Solution:**
    1. Disable extensions temporarily
    2. Test in incognito mode (extensions disabled)
    3. Whitelist lindo.ai in ad blocker
  </Step>
  <Step title="Network Issues">
    **Problem:** Slow or interrupted connection.

    **Solution:**
    1. Check internet connection
    2. Try refreshing the page
    3. Wait and try again
  </Step>
  <Step title="Large Page Content">
    **Problem:** Page has too much content to load quickly.

    **Solution:**
    1. Wait longer for initial load
    2. Simplify page if possible
    3. Check for very large images
  </Step>
  <Step title="Browser Compatibility">
    **Problem:** Unsupported or outdated browser.

    **Solution:**
    Use a modern browser:
    - Chrome (recommended)
    - Firefox
    - Safari
    - Edge
  </Step>
</Steps>
## Quick Fixes

### Refresh the Page

Press `Cmd/Ctrl + R` or click browser refresh.

### Hard Refresh

Press `Cmd/Ctrl + Shift + R` to bypass cache.

### Try Different Browser

If one browser fails, try another.

### Clear Site Data

1. Open browser settings
2. Find site data/cookies
3. Clear data for lindo.ai
4. Refresh and log in again

## When to Contact Support

Contact support if:
- Issue persists across browsers
- Preview worked before, now doesn't
- Error messages appear
- Other users report same issue

Include:
- Browser and version
- Steps to reproduce
- Screenshots of errors
- When issue started
