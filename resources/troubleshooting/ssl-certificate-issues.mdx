Troubleshoot SSL/HTTPS problems.

## Common Issues

### Certificate Not Valid

**Symptoms:**
- Browser shows "Not Secure"
- HTTPS not working

**Solutions:**
1. Wait for certificate provisioning
2. Verify DNS is correct
3. Check domain ownership

### Mixed Content

**Symptoms:**
- Some resources not loading
- Security warnings

**Solutions:**
1. Ensure all resources use HTTPS
2. Update image URLs
3. Check external scripts

### Certificate Expired

**Symptoms:**
- Security warning
- Site not accessible

**Solutions:**
1. Certificates auto-renew
2. Contact support if persists

## Prevention

- Verify DNS correctly
- Wait for propagation
- Use HTTPS for all resources

## Tips

- SSL is automatic
- Wait 24-48 hours
- Contact support if stuck
