If your website or pages won't publish, follow these troubleshooting steps.

## Common Publishing Issues

### Save Button Not Working

**Symptoms:**
- Click publish but nothing happens
- Button appears disabled

**Solutions:**

1. **Check for unsaved changes**
   - Look for pending edits
   - Save draft first, then publish

2. **Refresh the editor**
   - Press Cmd/Ctrl + R
   - Try publishing again

3. **Check browser console**
   - Open DevTools (F12)
   - Look for error messages

### Publishing Stuck

**Symptoms:**
- Loading indicator won't stop
- Page never completes publishing

**Solutions:**

1. **Wait a moment**
   - Large pages take longer
   - Give it 30-60 seconds

2. **Cancel and retry**
   - Refresh the page
   - Try publishing again

3. **Check internet connection**
   - Ensure stable connection
   - Try different network

### Error Message on Publish

**Symptoms:**
- Error toast appears
- Specific error message shown

**Solutions:**

1. **Note the error message**
   - Screenshot if possible
   - Check for specific guidance

2. **Common errors:**
   - "Unauthorized" - Log out and back in
   - "Network error" - Check connection
   - "Validation error" - Check page content

## Page-Specific Issues

### Page Won't Publish

Check these items:

1. **Page status**
   - Ensure page isn't locked
   - Check page permissions

2. **Content issues**
   - Remove any invalid content
   - Check for broken elements

3. **SEO settings**
   - Verify meta tags are valid
   - Check for required fields

### Changes Not Appearing

After publishing:

1. **Clear cache**
   - Browser cache
   - CDN cache (wait a few minutes)

2. **Check correct URL**
   - Verify you're viewing published URL
   - Not preview or draft URL

3. **Hard refresh**
   - Cmd/Ctrl + Shift + R
   - Or clear browser cache

## Account Issues

### Permission Denied

If you can't publish:

1. Check your role permissions
2. Contact workspace owner
3. Verify website access

### Credit Issues

Publishing doesn't require credits, but:

1. Ensure account is active
2. Check subscription status
3. Verify no billing issues

## Still Having Issues?

If problems persist:

1. Try a different browser
2. Clear all browser data
3. Contact support with:
   - Error messages
   - Steps to reproduce
   - Browser information
