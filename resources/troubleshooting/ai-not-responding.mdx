---
title: "AI Not Responding"
description: "Troubleshooting AI chat issues"
---

The AI chat isn't responding to your messages.

## Symptoms

- Message sent but no response
- Loading indicator spins indefinitely
- Error message appears
- Chat seems frozen

## Common Causes & Solutions

<Steps>
  <Step title="Insufficient Credits">
    **Problem:** No credits available for AI operations.

    **Solution:**
    1. Check credit balance (toolbar or Settings → Credits)
    2. If zero, purchase more credits
    3. Or wait for monthly reset
  </Step>
  <Step title="Network Issues">
    **Problem:** Connection interrupted.

    **Solution:**
    1. Check your internet connection
    2. Refresh the page
    3. Try again
  </Step>
  <Step title="Browser Issues">
    **Problem:** Browser cache or extensions interfering.

    **Solution:**
    1. Clear browser cache
    2. Disable browser extensions temporarily
    3. Try incognito/private mode
    4. Try a different browser
  </Step>
  <Step title="Long Request">
    **Problem:** Complex request taking longer than expected.

    **Solution:**
    1. Wait a bit longer (complex requests take 30-60 seconds)
    2. If over 2 minutes, refresh and try simpler request
    3. Break complex requests into smaller parts
  </Step>
  <Step title="Service Temporarily Unavailable">
    **Problem:** Backend service issue.

    **Solution:**
    1. Wait a few minutes
    2. Refresh the page
    3. Try again
    4. Check status page if available
  </Step>
</Steps>
## Quick Fixes

### Refresh the Page

Most issues resolve with a page refresh:
1. Press `Cmd/Ctrl + R`
2. Or click browser refresh button

### Clear and Retry

1. Refresh the page
2. Type a simple test message: "Hello"
3. If that works, try your original request

### Try Simpler Request

Break down complex requests:

**Instead of:**
```
Create a complete pricing page with 3 tiers, feature
comparison table, FAQ, testimonials, and CTA
```

**Try:**
```
Add a pricing section with 3 tiers
```

Then add more sections separately.

## Checking AI Status

Look for indicators:
- Loading spinner = processing
- Error message = something went wrong
- No response = may be stuck

## When to Contact Support

Contact support if:
- AI consistently doesn't respond
- Error messages persist after refresh
- Credits available but AI won't work
- Issue continues across browsers

Include:
- What you were trying to do
- Error messages (if any)
- Browser and device info
- Screenshots
