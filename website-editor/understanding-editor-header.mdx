The editor header provides navigation and controls.

## Header Components

### Logo/Back Button

- Click to return to workspace
- Shows Lindo.ai or your brand logo

### Website Dropdown

- Current website name
- Credit balance
- Back to workspace option

### Navigation Tabs

- Design
- Pages
- Insights
- CRM
- Settings

## Tab Functions

| Tab | Purpose |
|-----|---------|
| Design | Edit current page |
| Pages | Manage all pages |
| Insights | View analytics |
| CRM | Manage customers |
| Settings | Website settings |

## Tips

- Use tabs to navigate
- Check credits in dropdown
- Return to workspace via logo
