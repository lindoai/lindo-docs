Switch between different sections of the website editor.

## Available Tabs

- **Design** - Edit current page
- **Pages** - Manage all pages
- **Insights** - View analytics
- **CRM** - Manage customers
- **Settings** - Website settings

<Steps>
  <Step title="Find the Tabs">
    Look at the header navigation.
  </Step>
  <Step title="Click a Tab">
    Click the tab name to switch.
  </Step>
  <Step title="View Content">
    Tab content loads below.
  </Step>
</Steps>
## Tab Overview

| Tab | What You Do |
|-----|-------------|
| Design | Edit with AI, preview |
| Pages | Add, edit, delete pages |
| Insights | View traffic data |
| CRM | Manage customers |
| Settings | Configure website |

## Tips

- Design is default tab
- Each tab has sub-sections
- Changes save automatically
