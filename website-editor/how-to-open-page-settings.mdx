---
title: "How to Open Page Settings Panel"
description: "Accessing page settings"
---

Access settings for the current page.

<Steps>
  <Step title="Find Page Settings Button">
    Look for the settings icon in preview header.
  </Step>
  <Step title="Click to Open">
    Click the button.
  </Step>
  <Step title="Panel Opens">
    See page-specific settings.
  </Step>
</Steps>
## What You Can Change

- Page URL
- Page lock
- SEO settings

## Tips

- Set URL before publishing
- Configure SEO for each page
- Use lock for private content
