title: "What is the Theme Panel"
description: "Design customization panel"
---

The Theme panel controls global design settings.

## What It Controls

- Typography (fonts)
- Colors
- Buttons & Links
- Global styles

## Accessing Theme

Click the Theme button in the preview header.

## Settings Available

### Typography

- Heading font
- Body font

### Colors

- Primary color
- Secondary color
- Background
- Text color

### Buttons

- Roundness
- Colors
- Border

### Links

- Link color
- Hover color

## Tips

- Set early in project
- Consistent branding
- Affects entire site
