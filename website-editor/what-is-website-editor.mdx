description: "Edit, customize, and manage individual websites"
---

The Website Editor is where you edit, customize, and manage individual websites. After creating a website in the workspace, open it here to refine content, adjust design, manage pages, and publish changes.

## Layout

**Header** — Contains the logo (click to return to workspace), navigation tabs (Design, Pages, Insights, CRM, Settings), and a website dropdown with credits and info.

**AI Chat Panel (Left)** — Chat input for AI commands, message history, AI tools, and agent selector. This is where you tell the AI what to change.

**Preview Area (Right)** — Live website preview with editor controls, device preview buttons (desktop/tablet/mobile), and the Publish button.

## Editor Tabs

| Tab | Purpose |
|-----|---------|
| Design | AI chat and live preview for editing |
| Pages | Manage all pages, blogs, and products |
| Insights | Website analytics and traffic data |
| CRM | Customer management and invoicing |
| Settings | Website configuration, domain, integrations |

## Editor Modes

The preview toolbar has three modes:

- **Editor** — Full editing capabilities, click elements to select
- **Select** — Element selection only, no direct editing
- **Preview** — Read-only view, see the site as visitors do

## Key Actions

**Edit with AI** — Type in the chat panel to make changes like "Make the hero headline larger" or "Add a testimonials section."

**Switch pages** — Use the address bar dropdown or go to the Pages tab.

**Preview devices** — Click the desktop, tablet, or mobile icons to see responsive layouts.

**Publish** — Click the Publish button to make changes live.
