---

The website editor has three modes that control how you interact with your page.

## Editor Mode

Full editing capabilities:

- AI chat is active
- Can send prompts
- All editing tools available
- Make changes to content

Use Editor mode when actively editing your website.

## Select Mode

Element selection only:

- Click to select elements
- View element properties
- No AI editing
- Useful for inspection

Use Select mode to examine page structure without making changes.

## Preview Mode

Read-only preview:

- See page as visitors do
- No editing capabilities
- Test interactions
- Check responsive design

Use Preview mode to review your work before publishing.

## Switching Modes

1. Look at the preview header
2. Find the mode tabs (Editor, Select, Preview)
3. Click the mode you want
4. Interface updates accordingly

## Tips

- Use Editor for most work
- Switch to Preview before publishing
- Select mode helps understand structure
- Each mode serves a specific purpose
