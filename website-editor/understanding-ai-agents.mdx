AI Agents are specialized models that handle different types of editing tasks.

## Agent Types

### Block Creator Agent

**Purpose:** Generate new sections
**Credits:** 10
**Use for:** Adding hero, features, testimonials, pricing, etc.

```
Add a features section with 6 benefit cards
```

### Block Editor Agent

**Purpose:** Modify existing sections
**Credits:** 10
**Use for:** Changing content, layout, or styling of a section

```
Update the hero section with a new headline
```

### Element Editor Agent

**Purpose:** Edit individual elements
**Credits:** 10
**Use for:** Fine-tuning buttons, images, text, icons

```
Make this button larger with rounded corners
```

### Simple Blog Agent

**Purpose:** Write blog posts
**Credits:** 10
**Use for:** Creating blog content in markdown

```
Write a blog post about productivity tips
```

### Page Builder Agent

**Purpose:** Create entire pages
**Credits:** 10
**Use for:** Building new pages from scratch

```
Create an About page with team and company story
```

## Automatic Agent Selection

The AI automatically chooses the right agent based on:
- What you selected (block, element, nothing)
- What you're asking for (add, edit, create)
- Context of your request

## Manual Agent Selection

You can manually select an agent:
1. Look for the agent selector in the chat panel
2. Click to see available agents
3. Select the one you want

## Agent Credit Costs

| Agent | Credits |
|-------|---------|
| Website Builder | 10x (10 per page) |
| Page Builder | 10 |
| Block Creator | 10 |
| Block Editor | 10 |
| Element Editor | 10 |
| Simple Blog | 10 |

## Tips

- **Let AI choose** - Usually picks the right agent
- **Select first** - Selection helps AI understand context
- **Check credits** - Know the cost before generating
