---
title: "How to Create Blog Workflow"
description: "Start a blog pipeline to generate multiple posts"
---

Create a blog workflow to generate multiple blog posts automatically.

## Accessing Workflows

1. Open your website in the editor
2. Click **Settings** tab
3. Go to **Workflows** section

Or access from the sidebar Workflows icon.

## Creating Blog Pipeline

1. Click **Create Workflow**
2. Select **Blog Pipeline** type
3. Enter your blog topics

### Entering Topics

Add one topic per line:

```
10 Tips for Better Sleep
How to Start a Morning Routine
The Benefits of Meditation
Healthy Meal Prep Ideas
Work-Life Balance Strategies
```

## Configuration Options

### Language

Select content language for the posts.

### Email Notification

Toggle to receive email when workflow completes.

## Starting the Workflow

1. Review your topics
2. Check credit availability
3. Click **Start Workflow**
4. Workflow begins processing

## Monitoring Progress

Track your workflow:

- View status (Running, Completed, Error)
- See progress count
- Check for any failures

Click **Refresh** to update status.

## After Completion

1. Go to Pages tab
2. Find new blog posts in the list
3. Review each post
4. Edit as needed
5. Publish when ready

## Tips

- Start with 3-5 topics to test
- Use specific, descriptive topics
- Review all content before publishing
- Add images and personal touches
- Schedule publishing for best engagement
