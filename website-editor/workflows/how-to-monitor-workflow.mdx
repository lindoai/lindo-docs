---
title: "How to Monitor Website Workflow Progress"
description: "Checking the status of your workflows"
---

Track your workflows to see their current status.

## Viewing Workflow Status

1. Open the Workflows panel
2. See the list of all workflows
3. Each workflow shows its current status

## Status Types

| Status | Meaning |
|--------|---------|
| Running | Workflow is in progress |
| Completed | Finished successfully |
| Error | Something went wrong |

## Workflow Details

Each workflow entry shows:
- Workflow type (blog, page)
- Creation time
- Current status
- Result (when completed)

## Refreshing Status

Click the refresh button to update the workflow list and see the latest status.

## Handling Errors

If a workflow shows an error:
1. Check the error message
2. Verify your prompt was valid
3. Ensure you have sufficient credits
4. Try creating a new workflow
