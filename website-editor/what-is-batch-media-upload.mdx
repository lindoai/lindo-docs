---
title: "What is Batch Media Upload"
description: "Upload multiple media files at once in the Media Library"
---

Batch media upload lets you select and upload multiple files in a single action from the Media Library.

## How It Works

1. Open the Media Library and go to the Upload tab
2. Click "Upload Asset"
3. Select multiple files from the file picker (hold Cmd/Ctrl to multi-select)
4. All files are validated and uploaded in parallel

## Supported File Types

- Images: JPG, PNG, GIF, WebP, SVG, AVIF
- Videos: MP4, WebM, OGG, MOV
- Audio: MP3, WAV, OGG, AAC, FLAC
- Animations: Lottie JSON

## Limits

- 20MB total size per batch (all files combined)
- Up to 20 files per batch
- Individual file errors won't block the rest of the upload

## Tips

- Use batch upload to quickly populate your Media Library after starting a new project
- All uploaded files appear immediately in the library grid
- Single file uploads still work exactly as before
