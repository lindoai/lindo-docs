---
title: "How to Edit Blocks with AI"
description: "Modify sections using AI commands"
---

Modify existing sections (blocks) using AI commands. A block is a section of your page like a hero, features grid, testimonials, pricing table, or FAQ.

<Steps>
  <Step title="Open the Website Editor">
    Click your website from the workspace to open the editor.
  </Step>
  <Step title="Select the block">
    Switch to **Select** mode in the toolbar and click on the section you want to edit. The block highlights to show it's selected. Alternatively, just describe which section in your chat message.
  </Step>
  <Step title="Type your edit request">
    In the AI Chat, describe what you want to change:

    ```
    Change the headline to "Build Faster, Ship Sooner"
    ```

    ```
    Add a fourth testimonial card
    ```

    ```
    Make the background darker and add more padding
    ```
  </Step>
  <Step title="Submit and review">
    Click submit or press Enter. The AI generates the updated block. Review the changes in the preview.
  </Step>
  <Step title="Iterate if needed">
    Not perfect? Send another message to refine:

    ```
    Make the headline even larger
    ```
  </Step>
</Steps>

## Example Edits

**Change text:**
```
Update the hero headline to "Your Success Starts Here"
```

**Add elements:**
```
Add a "Watch Demo" button next to the main CTA
```

**Change layout:**
```
Change the features from 3 columns to 2 columns
```

**Update styling:**
```
Add a gradient background to this section
```

**Replace images:**
```
Change the hero image to show a team meeting
```

## Tips

- **Be specific** — "Make it better" is vague; "Add more whitespace" is clear
- **Reference elements** — "The second button" or "the headline"
- **One change at a time** — Easier to review and iterate

<Note>
Block Editor Agent costs **10 credits** per edit.
</Note>
