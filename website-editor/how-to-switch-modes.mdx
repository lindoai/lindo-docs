Switch between different modes to edit, select, or preview your website.

## Switching Modes

1. Look at the preview header
2. Find the mode tabs (Editor, Select, Preview)
3. Click the mode you want
4. The preview updates to that mode

## Available Modes

### Editor Mode
Full editing capabilities:
- Click to edit text
- Drag elements
- Make direct changes

### Select Mode
Element selection only:
- Click to select elements
- No accidental edits
- Prepare for AI editing

### Preview Mode
Read-only preview:
- See visitor experience
- Test interactions
- No editing UI

## When to Use Each

| Mode | Best For |
|------|----------|
| Editor | Direct text editing, quick changes |
| Select | Selecting elements for AI editing |
| Preview | Final review, testing links |

## Keyboard Shortcuts

Some modes may have keyboard shortcuts for quick switching. Check the keyboard shortcuts reference for details.
