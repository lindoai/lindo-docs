---
title: "How to Manage Inventory"
description: "Track product inventory with Stripe-compatible apps"
---

Lindo does not store inventory counts directly in the website editor. Instead, inventory is managed through Stripe-compatible tools.

## Recommended option

Lindo recommends **Stockify** for Stripe-based inventory management.

1. Open your website in the editor
2. Go to **Settings -> Shop**
3. In the **Inventory** guide, click **Open App**
4. Install and configure Stockify in Stripe

## When inventory tracking is useful

- Physical products with limited stock
- Limited drops and seasonal launches
- Handmade or made-to-order items
- Stores that need low-stock alerts

## Suggested workflow

1. Create products in Lindo
2. Connect Stripe to the website
3. Install Stockify in Stripe
4. Map and manage inventory there
5. Test checkout before publishing campaigns

## Important note

If you need advanced inventory logic like stock syncing, bundles, or warehouse workflows, manage that inside the Stripe app rather than inside Lindo.
