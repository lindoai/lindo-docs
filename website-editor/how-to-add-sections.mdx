---
title: "How to Add New Sections"
description: "Adding content blocks to your pages"
---

Add new sections (blocks) to your website pages using AI.

<Steps>
  <Step title="Open the Website Editor">
    Click your website from the workspace.
  </Step>
  <Step title="Navigate to the Page">
    Make sure you're on the page where you want to add a section.
  </Step>
  <Step title="Describe the Section">
    In the AI Chat, describe what you want:

    ```
    Add a testimonials section with 3 customer reviews
    ```
  </Step>
  <Step title="Submit">
    Click send or press Enter.
  </Step>
  <Step title="Review">
    The new section appears in your page. Review in the preview.
  </Step>
  <Step title="Refine if Needed">
    Make adjustments:

    ```
    Make the testimonial cards larger with photos
    ```
  </Step>
</Steps>
## Example Prompts

### Features Section

```
Add a features section with 6 cards in a 3-column grid.
Each card has an icon, title, and description.
```

### Testimonials

```
Add a testimonials carousel with 3 customer quotes,
including photos, names, and company names.
```

### Pricing Table

```
Add a pricing section with 3 tiers:
- Basic: $9/month
- Pro: $29/month
- Enterprise: Contact us
Include feature lists for each tier.
```

### FAQ

```
Add an FAQ accordion section with 5 questions about our service.
```

### Team Section

```
Add a team section showing 4 team members with photos,
names, titles, and short bios.
```

### CTA Section

```
Add a call-to-action section with headline
"Ready to get started?" and two buttons.
```

### Stats Section

```
Add a statistics section showing 4 metrics:
- 10,000+ customers
- 99.9% uptime
- 24/7 support
- 50+ integrations
```

## Section Placement

By default, new sections are added at the end of the page.

To specify placement:

```
Add a testimonials section after the features section
```

```
Add a CTA section before the footer
```

## Tips

- **Be specific** - Include numbers and details
- **Mention style** - "Modern", "minimal", etc.
- **Reference existing** - "Match the style of the features section"
