Enhance AI prompts with additional tools.

## Available Tools

### Language

Set content language for generated text.

### Mode

Choose dark or light theme.

### Color Palette

Define custom colors.

### Website URL

Use reference design.

## Using Tools

1. Click the tools button
2. Select a tool
3. Configure settings
4. Tool added to prompt

## When to Use

- Language: Localized content
- Mode: Theme preference
- Colors: Brand consistency
- URL: Design inspiration

## Tips

- Combine multiple tools
- Set before generating
- Consistent results
