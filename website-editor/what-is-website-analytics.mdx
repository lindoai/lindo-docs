---
title: "What is Website Analytics"
description: "Overview of the insights dashboard for a single website"
---

Website Analytics (Insights tab) shows traffic and visitor data for a single website.

## Accessing Analytics

1. Open your website in the editor
2. Click the **Insights** tab

## Analytics Components

### Statistics Cards

Quick metrics at the top:
- **Visitors** - Unique visitors
- **Page Views** - Total page loads
- **Avg Response Time** - Page load speed
- **Total Pages** - Number of pages
- **Total Blogs** - Number of blog posts

### Traffic Chart

Line graph showing:
- Page views over time
- Visitors over time
- Customizable date range

### Top Pages

Most visited pages on this website:
- Page path
- View count
- Percentage of total

### Operating Systems

Visitor OS breakdown:
- Windows, macOS, iOS, Android, Linux

### Browsers

Visitor browser breakdown:
- Chrome, Safari, Firefox, Edge

### Devices

Device type breakdown:
- Desktop
- Mobile
- Tablet
- Bot

### Top Countries

Geographic distribution of visitors.

### Referrers

Where traffic comes from:
- Direct
- Search engines
- Social media
- Other websites

### UTM Campaigns

Marketing campaign performance:
- Campaign name
- Source and medium
- Visit count

## Website vs Workspace Analytics

| Website Analytics | Workspace Analytics |
|-------------------|---------------------|
| Single website | All websites combined |
| In editor (Insights tab) | In workspace sidebar |
| Top pages | Top websites |
| Detailed page data | Aggregated data |

## Date Range

Change the time period:
1. Click the date picker
2. Select start and end dates
3. Data updates automatically

## Refreshing Data

Click the refresh button to get the latest data.
