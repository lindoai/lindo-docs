title: "How to Edit Page SEO"
description: "Accessing SEO settings for page"
---

Configure SEO settings for a specific page.

<Steps>
  <Step title="Go to Pages Tab">
    Click "Pages" in the header.
  </Step>
  <Step title="Find the Page">
    Locate the page to edit.
  </Step>
  <Step title="Click Actions">
    Hover and click the actions menu.
  </Step>
  <Step title="Select Edit SEO">
    Click "Edit SEO" option.
  </Step>
  <Step title="Configure Settings">
    Set SEO fields:
    - Title
    - Description
    - OG Image
  </Step>
</Steps>
## SEO Fields

| Field | Purpose |
|-------|---------|
| Title | Browser tab, search results |
| Description | Search result snippet |
| OG Image | Social sharing image |

## Tips

- Unique title per page
- Compelling description
- Relevant OG image
