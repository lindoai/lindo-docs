Save your work without making it live.

## How It Works

Changes save automatically as drafts:
- Work is preserved
- Not visible to public
- Publish when ready

## Draft vs Published

| Draft | Published |
|-------|-----------|
| Only you see | Public sees |
| Work in progress | Live content |
| Auto-saved | Manually published |

## Tips

- Work is auto-saved
- Publish when ready
- Draft is safe to leave
