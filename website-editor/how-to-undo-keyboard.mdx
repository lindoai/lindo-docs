Quickly undo changes using the keyboard shortcut.

## Undo Shortcut

Press **Cmd + Z** (Mac) or **Ctrl + Z** (Windows/Linux) to undo.

## What Gets Undone

The shortcut reverts:

- Recent changes
- Text edits
- Element modifications

## Multiple Undos

Press the shortcut multiple times to:

- Undo multiple changes
- Step back through history
- Revert to earlier state

## Tips

- Undo immediately after mistakes
- Can undo multiple times
- Use redo to restore if needed
- Some actions can't be undone
