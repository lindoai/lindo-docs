---
title: "How to Duplicate a Page"
description: "Copying an existing page"
---

Create a copy of an existing page.

<Steps>
  <Step title="Go to Pages Tab">
    Click "Pages" in the header.
  </Step>
  <Step title="Find the Page">
    Locate the page to duplicate.
  </Step>
  <Step title="Click Actions">
    Hover and click the actions menu.
  </Step>
  <Step title="Select Duplicate">
    Click "Duplicate" option.
  </Step>
  <Step title="Copy Created">
    New page appears with "(Copy)" suffix.
  </Step>
</Steps>
## Use Cases

- Create similar pages
- Backup before changes
- Template from existing

## Tips

- Rename the copy
- Update content
- Change URL
