Save your changes quickly using the keyboard shortcut.

## Save Shortcut

Press **Cmd + S** (Mac) or **Ctrl + S** (Windows/Linux) to save.

## What Gets Saved

The shortcut saves:

- Current page changes
- Draft status
- All modifications

## When to Use

Use the save shortcut:

- After making changes
- Before switching pages
- Periodically while editing
- Before closing editor

## Tips

- Save frequently
- Shortcut is faster than clicking
- Works in editor mode
- Confirmation toast appears
