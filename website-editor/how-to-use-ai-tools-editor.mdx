Use AI tools in the editor to enhance your prompts with specific settings.

## Available Tools

### Language Tool

Set the language for generated content:

1. Click **(+)** in chat input
2. Select **Language**
3. Choose your language
4. Include in your prompt

### Mode Tool

Set light or dark theme:

1. Click **(+)** in chat input
2. Select **Mode**
3. Choose Light or Dark
4. Include in your prompt

### Color Palette Tool

Define custom colors:

1. Click **(+)** in chat input
2. Select **Color Palette**
3. Set your colors
4. Include in your prompt

### Website URL Tool

Reference another website:

1. Click **(+)** in chat input
2. Select **Website URL**
3. Enter the reference URL
4. Include in your prompt

## Combining Tools

You can use multiple tools together:

```
[Language: Spanish]
[Mode: Dark]
[Colors: #3B82F6, #10B981]

Create a hero section with a bold headline and CTA button
```

## Tips

- Tools appear as tags in your message
- They guide the AI's output
- Combine with clear instructions
- Preview results and adjust
