---
title: "How to Change Page URL"
description: "Editing the page path"
---

Change the URL path for a page.

<Steps>
  <Step title="Open Page Settings">
    Click the Page Settings button.
  </Step>
  <Step title="Find URL Field">
    Locate the URL/path input.
  </Step>
  <Step title="Edit URL">
    Enter the new path:
    - /about
    - /services
    - /contact
  </Step>
  <Step title="Save">
    Changes save automatically.
  </Step>
</Steps>
## URL Guidelines

- Use lowercase
- Use hyphens for spaces
- Keep it short
- Be descriptive

## Examples

| Good | Bad |
|------|-----|
| /about-us | /About Us |
| /services | /our-services-page |
| /contact | /contact_us |

## Tips

- Set before publishing
- SEO-friendly URLs
- Avoid changing after launch
