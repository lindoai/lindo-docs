The AI Chat is your interface for editing websites using natural language. Describe what you want to change, and the AI makes it happen.

## Location

Left panel of the Website Editor.

## Components

**Chat Input** — Text area at the bottom where you type commands. Describe changes, ask for new sections, or request modifications.

**Message History** — Above the input, see your messages, AI responses, and generation previews.

**Toolbar** — Below the input with (+) Add Tools, Enhance button, and Submit button.

**Agent Selector** — Choose which AI agent to use (Block agents for sections, Element agents for details, Page agents for full pages).

## How It Works

<Steps>
  <Step title="Type your request">
    Describe the change you want in plain language.
  </Step>
  <Step title="AI processes">
    The system analyzes your request and selects the appropriate agent.
  </Step>
  <Step title="Preview appears">
    See the generated result in the preview area.
  </Step>
  <Step title="Applied to page">
    Changes are applied to your page automatically.
  </Step>
</Steps>

## Example Commands

**Add content:**
```
Add a testimonials section with 3 customer reviews
```

**Modify existing:**
```
Make the hero headline larger and add a subheadline
```

**Change styling:**
```
Change the button colors to green
```

**Replace content:**
```
Update the features section with these 4 benefits: ...
```

## AI Tools

Click **(+)** to add context to your request:
- **Language** — Generate in a specific language
- **Mode** — Dark or light theme
- **Color Palette** — Your brand colors
- **Website URL** — Reference an existing design

## Tips

- **Be specific** — Clear requests get better results
- **One change at a time** — Easier to review
- **Use selection** — Select elements before typing for targeted edits
