Use keyboard shortcuts to work faster in the website editor.

## Available Shortcuts

### Editing

| Shortcut | Action |
|----------|--------|
| Cmd/Ctrl + S | Save changes |
| Cmd/Ctrl + Z | Undo |
| Cmd/Ctrl + Shift + Z | Redo |

### Navigation

| Shortcut | Action |
|----------|--------|
| Cmd/Ctrl + 1-6 | Switch settings tabs |
| Escape | Close modal |

## Using Shortcuts

- Cmd on Mac, Ctrl on Windows/Linux
- Works when editor is focused
- Combine with mouse for efficiency

## Tips

- Save frequently with Cmd/Ctrl + S
- Undo mistakes quickly
- Learn common shortcuts
- Faster than clicking
