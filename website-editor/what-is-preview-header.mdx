The preview header contains controls for editing and viewing.

## Components

### Mode Tabs

- Editor
- Select
- Preview

### Control Buttons

- Layers
- Theme
- Page Settings

### Address Bar

- Current page URL
- Navigation controls

### Action Buttons

- Undo/Redo
- Device preview
- Fullscreen
- Publish

## Using the Header

- Switch modes with tabs
- Access panels with buttons
- Navigate with address bar
- Publish when ready

## Tips

- Learn each control
- Use keyboard shortcuts
- Quick access to features
