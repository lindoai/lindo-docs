---
title: "How to View Page History"
description: "Accessing version history for pages"
---

View and restore previous versions of your pages.

## Accessing Page History

1. Open your website in the editor
2. Navigate to the page you want
3. Click the **History** icon in the preview toolbar (clock icon)

## History Panel

The panel shows:
- List of saved versions
- Timestamp for each version
- Preview of each version

## Viewing a Version

1. Click any version in the list
2. Preview updates to show that version
3. Compare with current version

## Restoring a Version

To revert to a previous version:

1. Find the version you want
2. Click **Restore** button
3. Confirm the action
4. Page reverts to that version

**Note:** Restoring creates a new version, so you can always undo.

## What Gets Saved

Each version captures:
- All page content
- Section structure
- Styling and layout
- Images and media

## When Versions Are Created

New versions are saved when you:
- Click Save
- Publish the page
- Make significant AI edits

## Tips

### Before Major Changes

1. Save current version
2. Make your changes
3. If unhappy, restore previous

### Finding the Right Version

- Look at timestamps
- Preview each to compare
- Restore the one you want

### Version Limits

- Recent versions are kept
- Very old versions may be pruned
- Important versions: save/export

## Use Cases

**Undo mistakes:**
Made a bad edit? Restore previous version.

**Compare designs:**
Preview different versions to compare.

**Client requests:**
"Go back to how it was" - easy with history.

**Experimentation:**
Try changes knowing you can revert.
