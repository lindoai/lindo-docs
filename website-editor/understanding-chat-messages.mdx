The AI chat shows your conversation with AI agents.

## Message Types

### User Messages

Your prompts and requests:
- Appear on right side
- Your input text
- Can include images

### Agent Responses

AI-generated responses:
- Appear on left side
- Include explanations
- Show what was changed

## Message Flow

1. You send a message
2. AI processes request
3. AI responds with changes
4. Preview updates

## Tips

- Be specific in requests
- Review AI responses
- Check preview for changes
