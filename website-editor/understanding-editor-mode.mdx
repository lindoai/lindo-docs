description: "Full editing capabilities in the editor"

Editor mode gives you full control to make direct changes to your website. Click text to edit it, select elements to modify them, and use AI to make updates.

## Capabilities

**Text Editing** — Click any text to edit it directly. Change content and format text inline.

**Element Selection** — Click elements to select them. See selection highlights and access element options.

**AI Editing** — Select elements for AI changes. Send prompts about your selection and watch the AI make updates.

## When to Use

Editor mode is best for:
- Quick text changes
- Direct content updates
- Active editing sessions
- Working with AI

## Visual Indicators

In Editor mode, editing controls are visible, selection highlights appear when you click elements, and the toolbar is fully active.
