title: "How to Upload Digital File"
description: "Adding downloadable file"
---

Add a downloadable file to your product.

<Steps>
  <Step title="Open Product Settings">
    Click the product to open settings.
  </Step>
  <Step title="Find File Upload">
    Locate the digital file upload area.
  </Step>
  <Step title="Click Upload">
    Click to select a file.
  </Step>
  <Step title="Select File">
    Choose the file from your device.
  </Step>
  <Step title="Save">
    Click save to apply.
  </Step>
</Steps>
## Supported Files

- PDF documents
- ZIP archives
- Images
- Audio files
- Video files

## How It Works

1. Customer purchases
2. Payment confirmed
3. Download link provided
4. Customer downloads file

## Tips

- Test download link
- Clear file names
- Reasonable file sizes
