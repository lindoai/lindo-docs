Preview your website on different devices.

## Available Views

### Desktop

- Full-width view
- Standard computer display
- Default view

### Tablet

- Medium width
- iPad-like display
- Portrait orientation

### Mobile

- Narrow width
- Phone display
- Portrait orientation

## Why Preview Devices

- Check responsiveness
- Ensure mobile-friendly
- Test all layouts
- Catch issues early

## Tips

- Check all three views
- Mobile is most important
- Fix issues before publishing
