title: "Understanding the Address Bar"
description: "Domain, URL, and controls"
---

The address bar shows the current page URL and provides navigation.

## Components

### Domain Display

Shows your website domain or subdomain.

### Page Path

Current page URL path (e.g., /about).

### Controls

- Open in new tab
- Copy URL
- Navigation

## Uses

- See current page
- Navigate to pages
- Share URLs
- Open live site

## Tips

- Check URL before sharing
- Use for navigation
- Open in new tab to test
