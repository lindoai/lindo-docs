---
title: "How to Check Credit Balance in Editor"
description: "View your available credits while editing"
---

Check your credit balance without leaving the website editor.

## Finding Your Balance

1. Look at the header in the editor
2. Click the website dropdown (shows website name)
3. Your credit balance is displayed

## What You See

The dropdown shows:

- Current credit balance
- Website name
- Back to workspace option

## Why Check Credits

Monitor credits to:

- Know before starting AI tasks
- Avoid running out mid-edit
- Plan your editing session
- Decide which agent to use

## Credit Usage

Different agents use different credits:

| Agent | Credits |
|-------|---------|
| Block Creator | 10 |
| Block Editor | 10 |
| Element Editor | 10 |
| Tailwind Class | 5 |
| Page Builder | 30 |
| Blog Writer | 30 |

## Running Low?

If credits are low:

- Complete essential edits first
- Use lower-cost agents
- Purchase more credits
- Return to workspace to buy
