---
title: "How to Open Page in New Tab"
description: "View your live page in a separate browser tab"
---

Open your published page in a new browser tab to see exactly how visitors will experience it.

## Opening in New Tab

1. Look at the address bar in the preview header
2. Click the **Open in New Tab** icon (external link icon)
3. Page opens in a new browser tab

## What You See

The new tab shows:

- Your live published page
- Actual domain URL
- Real visitor experience

## When to Use

Open in new tab to:

- Test actual page behavior
- Check forms and interactions
- Share URL with others
- Verify published changes

## Requirements

- Page must be published
- Or use preview URL for drafts

## Tips

- Test on different devices
- Check all interactive elements
- Verify links work correctly
- Test forms submit properly
