---
title: "How to Switch AI Agents"
description: "Changing between different agents"
---

Change which AI agent handles your request.

## Available Agents

- Block Creator
- Block Editor
- Element Editor
- Blog Writer
- And more

<Steps>
  <Step title="Find Agent Selector">
    Look for the agent dropdown in chat.
  </Step>
  <Step title="Click to Open">
    See available agents.
  </Step>
  <Step title="Select Agent">
    Choose the appropriate agent.
  </Step>
  <Step title="Agent Active">
    New agent handles your requests.
  </Step>
</Steps>
## Choosing Agents

| Task | Agent |
|------|-------|
| Add new section | Block Creator |
| Edit existing section | Block Editor |
| Fine-tune element | Element Editor |
| Write blog post | Blog Writer |

## Tips

- Match agent to task
- Check credit costs
- Right agent = better results
