description: "Real-time generation preview"
---

Stream preview shows AI generation in real-time.

## What It Does

As AI generates content:
- See changes appear live
- Watch sections build
- Real-time feedback

## How It Works

1. Send AI request
2. AI starts generating
3. Preview updates live
4. See progress in real-time

## Benefits

- No waiting blindly
- See what's being created
- Can stop if needed
- Immediate feedback

## Tips

- Watch the preview
- Stop if going wrong
- Provides transparency
