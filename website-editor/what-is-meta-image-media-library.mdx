---

The page meta image field (used for Open Graph and social sharing previews) now integrates with the Media Library.

## How It Works

1. Open Page Settings and go to the SEO section
2. Find the meta image field
3. Click "Select Image" to open the Media Library
4. Browse or upload an image, then select it
5. The image URL is automatically set as your page's meta image

## What Changed

Previously, you had to manually paste an image URL into the meta image field. Now you can browse your uploaded media and select an image visually — the same way you pick images for page content.

## Why It Matters

- The meta image appears when your page is shared on social media, messaging apps, and search results
- Using the Media Library ensures the image is hosted on your CDN
- No more copy-pasting URLs or uploading images elsewhere first

## Tips

- Use a 1200×630px image for best results on social platforms
- The selected image applies to both Open Graph and Twitter Card previews
- You can still paste a URL manually if you prefer
