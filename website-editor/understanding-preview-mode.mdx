---
title: "Understanding Preview Mode"
description: "Read-only preview mode in the editor"
---

Preview mode shows your website exactly as visitors will see it.

## Overview

In Preview mode:
- No editing controls visible
- Interactions work (links, buttons)
- See the true visitor experience
- Test functionality

## When to Use

### Final Review
Before publishing, check:
- How the page looks
- If links work
- If buttons function
- Overall appearance

### Testing Interactions
- Click navigation links
- Test form submissions
- Check hover effects
- Verify animations

### Client Presentations
- Show clean preview
- No editing UI visible
- Professional appearance

## How to Access

1. Look at the mode tabs in the preview header
2. Click **Preview**
3. You're now in Preview mode

## Preview vs Editor Mode

| Preview Mode | Editor Mode |
|--------------|-------------|
| Read-only | Full editing |
| No UI overlays | Editing controls visible |
| True visitor view | Working view |

## Fullscreen Preview

For an even cleaner view:
1. Enter Preview mode
2. Click the fullscreen icon
3. See full-screen preview
