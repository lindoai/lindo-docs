The AI chat panel is where you interact with AI agents to edit your website.

## What's Included

### Agent Selector

Choose which AI agent to use.

### Chat History

Previous messages and responses.

### Input Area

Type prompts and attach images.

### AI Tools

Add language, mode, colors to prompts.

## How to Use

1. Select an agent
2. Type your prompt
3. Add tools if needed
4. Send message
5. Watch changes in preview

## Tips

- Be specific in prompts
- Use appropriate agent
- Attach reference images
- Review changes before saving
