title: "What is the Theme Panel"
description: "Design customization panel"
---

The Theme panel provides quick access to design settings.

## What's Included

- Typography settings
- Color options
- Button styling
- Link colors

## How to Access

1. Click Theme icon in preview header
2. Panel opens with options
3. Make changes
4. Preview updates live

## Benefits

- Quick design changes
- Live preview
- Consistent styling
- Easy access

## Tips

- Use for quick adjustments
- Full settings in Settings tab
- Preview changes before saving
- Maintain brand consistency
