---
title: "Understanding Device Preview"
description: "Desktop, tablet, and mobile views"
---

Device preview lets you see how your website looks on different devices.

## Available Views

### Desktop

Full-width desktop view.

### Tablet

Medium-width tablet view.

### Mobile

Narrow mobile phone view.

## Switching Views

1. Find device icons in preview header
2. Click desired device
3. Preview adjusts to that size

## Tips

- Check all devices before publishing
- Mobile is increasingly important
- Test interactions on each size
- Ensure readability on all devices
