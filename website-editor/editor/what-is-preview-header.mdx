The preview header contains controls for the editor and preview area.

## What's Included

### Mode Tabs

- Editor mode
- Select mode
- Preview mode

### Panel Toggles

- Layers panel
- Theme panel
- Page settings

### Device Preview

- Desktop
- Tablet
- Mobile

### Actions

- Undo/Redo
- History
- Fullscreen
- Open in new tab

### Save/Publish

- Save draft
- Publish page

## Tips

- Use mode tabs for different tasks
- Toggle panels as needed
- Preview on all devices
- Save frequently
