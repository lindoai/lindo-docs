---
title: "What is Page History"
description: "Version history for pages"
---

Page History tracks changes to your pages over time.

## What's Included

- Previous versions
- Timestamps
- Version comparison
- Restore option

## How to Access

1. Click History icon in preview header
2. View version list
3. Select version to preview
4. Restore if needed

## Benefits

- Recover from mistakes
- Track changes
- Compare versions
- Undo major changes

## Tips

- Check history before major edits
- Restore if something breaks
- History saves automatically
- Limited history retention
