The chat panel shows your conversation with AI agents.

## Message Types

### User Messages

Your prompts and requests:

- Appear on right side
- Show your input
- Include attached images
- Include AI tools

### Agent Responses

AI agent replies:

- Appear on left side
- Show what was done
- Include status updates
- May show errors

## Tips

- Review agent responses
- Check for errors
- Scroll to see history
- Clear context if needed
