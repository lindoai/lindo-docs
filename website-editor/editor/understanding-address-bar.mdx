description: "Domain, URL, and controls"
---

The address bar shows the current page URL and provides controls.

## What You See

- Domain name
- Page path
- Open in new tab button
- Page controls

## Features

### URL Display

Shows the full URL of the current page.

### Open in New Tab

Click to view the live page in a new browser tab.

### Page Controls

Access page settings and actions.

## Tips

- Check URL before sharing
- Use new tab for testing
- Verify domain is correct
- Access controls quickly
