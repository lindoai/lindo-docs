AI tools enhance your prompts with specific settings.

## Available Tools

### Language

Set content language for generation.

### Mode

Choose light or dark theme.

### Color Palette

Define custom colors.

### Website URL

Reference another website for inspiration.

## Using Tools

1. Click (+) in chat input
2. Select a tool
3. Configure settings
4. Tool appears in your message

## Tips

- Combine multiple tools
- Set language for localization
- Use colors for brand consistency
- Reference sites for style guidance
