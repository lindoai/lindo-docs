---
title: "What is the Page Settings Panel"
description: "Page-specific settings"
---

The Page Settings panel controls settings for the current page.

## What's Included

- Page URL/path
- Page lock status
- SEO settings
- Page-specific options

## How to Access

1. Click Settings icon in preview header
2. Panel opens with options
3. Make changes
4. Save

## Benefits

- Quick page configuration
- URL management
- Lock/unlock pages
- SEO optimization

## Tips

- Set meaningful URLs
- Lock sensitive pages
- Optimize SEO per page
- Review before publishing
