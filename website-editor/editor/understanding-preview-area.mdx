description: "Right panel with live website preview"
---

The preview area shows your website as visitors will see it.

## What You See

- Live website preview
- Current page content
- Real-time updates
- Device-specific views

## Features

### Device Preview

Switch between:

- Desktop view
- Tablet view
- Mobile view

### Fullscreen

Expand preview for better viewing.

### Address Bar

Shows current page URL and controls.

## Tips

- Check all device sizes
- Preview before publishing
- Use fullscreen for detail work
- Test interactions
