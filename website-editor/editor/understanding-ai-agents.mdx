The editor provides access to various AI agents for different editing tasks.

## Available Agents

### Page Builder

Create complete pages with AI.

### Block Creator

Add new content sections.

### Block Editor

Modify existing sections.

### Element Editor

Fine-tune individual elements.

### Blog Writer

Create blog posts.

### Tailwind Class

Quick CSS adjustments.

## Switching Agents

1. Click agent selector in chat
2. Choose appropriate agent
3. Send your prompt

## Tips

- Use right agent for task
- Block Creator for new sections
- Element Editor for details
- Check credit costs
