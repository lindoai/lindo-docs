---
title: "What is the Layers Panel"
description: "Layer management for page elements"
---

The Layers panel shows the structure of your page elements.

## What You See

- Hierarchical element list
- Element names
- Nesting structure
- Selection indicators

## How to Use

1. Click Layers icon in preview header
2. Panel opens showing structure
3. Click elements to select
4. View page hierarchy

## Benefits

- Understand page structure
- Select nested elements
- Navigate complex layouts
- Debug layout issues

## Tips

- Use for complex pages
- Select hard-to-click elements
- Understand nesting
- Close when not needed
