---

The editor header provides navigation and key controls.

## Header Elements

### Logo

Click to return to workspace.

### Tabs

Navigate between:

- Design
- Pages
- Insights
- CRM
- Settings

### Website Dropdown

Shows:

- Current website name
- Credit balance
- Back to workspace option

## Tips

- Use tabs to navigate sections
- Check credits in dropdown
- Click logo to exit editor
