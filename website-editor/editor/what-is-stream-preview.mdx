---
title: "What is Stream Preview"
description: "Real-time generation preview"
---

Stream Preview shows AI-generated content as it's being created.

## How It Works

1. Send a prompt to AI
2. Generation begins
3. Content appears in real-time
4. Preview updates as it generates

## Benefits

- See progress immediately
- Stop if going wrong direction
- Faster feedback loop
- Interactive experience

## Tips

- Watch for early issues
- Stop and adjust if needed
- Wait for completion
- Review final result
