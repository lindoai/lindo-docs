---
title: "How to Lock/Unlock Page"
description: "Setting page authentication"
---

Require authentication to view a page.

<Steps>
  <Step title="Open Page Settings">
    Click the Page Settings button.
  </Step>
  <Step title="Find Lock Toggle">
    Locate the page lock option.
  </Step>
  <Step title="Enable Lock">
    Turn on to require authentication.
  </Step>
  <Step title="Save">
    Changes save automatically.
  </Step>
</Steps>
## What Locking Does

- Requires login to view
- Protects private content
- Members-only access

## Use Cases

- Member content
- Client previews
- Private pages
- Gated content

## Tips

- Use for private content
- Communicate to users
- Test the lock
