description: "Page-specific settings"
---

Page settings control individual page configuration.

## What It Controls

- Page URL/path
- Page lock (authentication)
- SEO settings
- Page-specific options

## Accessing

Click the Page Settings button in preview header.

## Settings Available

### URL

Change the page path.

### Lock

Require authentication to view.

### SEO

- Title
- Description
- OG image

## Tips

- Set URL before publishing
- Use lock for private pages
- Optimize SEO for each page
