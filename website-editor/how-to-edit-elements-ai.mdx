Make precise edits to individual elements like buttons, images, and text.

## What is an Element?

Elements are individual components within a section:
- Buttons
- Headings
- Paragraphs
- Images
- Icons
- Links

<Steps>
  <Step title="Switch to Select Mode">
    Click **Select** in the preview toolbar.
  </Step>
  <Step title="Click the Element">
    Click directly on the element you want to edit. It highlights.
  </Step>
  <Step title="Describe Your Edit">
    In the AI Chat:

    ```
    Make this button larger with rounded corners
    ```
  </Step>
  <Step title="Submit">
    The Element Editor Agent makes the change.
  </Step>
  <Step title="Review">
    Check the result in the preview.
  </Step>
</Steps>

## Without Selection

You can also describe the element:

```
Change the hero button to say "Start Free Trial"
```

```
Make the logo in the header larger
```

But selecting first is more precise.

## Element Editor vs Block Editor

| Element Editor | Block Editor |
|----------------|--------------|
| Single element | Entire section |
| Precise changes | Broader changes |
| Buttons, images, text | Layout, structure |
| 10 credits | 10 credits |

## Tips

- **Select first** - More accurate results
- **Be specific** - "The blue button" not "a button"
- **One element at a time** - Easier to review

## Credit Cost

Element Editor Agent: **10 credits** per edit
