description: "Testing desktop, tablet, and mobile views"
---

Test how your website looks on different screen sizes.

## Device Preview Options

### Desktop

Full-width view for desktop computers.
- Width: ~1200px+
- Shows full navigation
- All columns visible

### Tablet

Medium-width view for tablets.
- Width: ~768px
- May show hamburger menu
- Columns may stack

### Mobile

Narrow view for phones.
- Width: ~375px
- Mobile navigation
- Single column layout

## How to Switch Devices

### In Preview Toolbar

1. Look for device icons in the preview header
2. Click the device you want:
   - 🖥️ Desktop
   - 📱 Tablet
   - 📱 Mobile (smaller)
3. Preview updates immediately

## What to Check

### Desktop

- Full navigation visible
- Multi-column layouts correct
- Images properly sized
- Spacing looks good

### Tablet

- Navigation adapts (hamburger menu?)
- Columns reduce appropriately
- Touch targets large enough
- Content readable

### Mobile

- Single column layout
- Text readable without zooming
- Buttons easy to tap
- Images scale properly
- No horizontal scrolling

## Common Issues

### Text Too Small

```
Make the body text larger on mobile
```

### Buttons Too Close

```
Add more spacing between buttons on mobile
```

### Images Overflow

```
Make images responsive and fit within the container
```

### Navigation Broken

```
Fix the mobile navigation menu
```

## Fullscreen Preview

For a better view:
1. Click the fullscreen icon
2. Preview expands
3. Click again to exit

## Tips

- **Always check mobile** - Most traffic is mobile
- **Test interactions** - Click buttons, open menus
- **Check all pages** - Not just homepage
- **Preview before publishing** - Catch issues early
