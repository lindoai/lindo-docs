---
title: "What is the Layers Panel"
description: "Layer management for page elements"
---

The Layers panel shows the structure of your page.

## What It Shows

- Page hierarchy
- All sections/blocks
- Nested elements
- Selection state

## Uses

- Navigate page structure
- Select specific elements
- Understand layout
- Reorder sections

## Opening Layers

Click the Layers button in the preview header.

## Panel Features

- Expandable tree view
- Click to select
- Visual hierarchy

## Tips

- Use for complex pages
- Find nested elements
- Understand structure
