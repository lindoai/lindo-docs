Restore changes you've undone using the keyboard shortcut.

## Redo Shortcut

Press **Cmd + Shift + Z** (Mac) or **Ctrl + Shift + Z** (Windows/Linux) to redo.

## What Gets Restored

The shortcut restores:

- Previously undone changes
- Reverted modifications
- Undone edits

## Multiple Redos

Press the shortcut multiple times to:

- Redo multiple undos
- Step forward through history
- Restore to later state

## Tips

- Use after accidental undo
- Can redo multiple times
- Only works after undo
- New changes clear redo history
