---
title: "What is the Asset Library Panel"
description: "Browse and manage all media assets used across your pages and blocks"
---

The Asset Library panel scans your page for all media assets and displays them in one place.

## What It Shows

- All images used in your page sections
- Background images applied via CSS classes or inline styles
- Videos and other media elements
- Assets organized by block/section

## How It Works

1. Click the Asset Library button in the editor toolbar
2. The panel scans all sections on your page
3. Assets are displayed as a visual grid
4. Click any asset to select it on the page
5. Use the Media Library to swap it with a different file

## Panel Features

- Floating resizable panel
- Works for both page-level and block-level assets
- Click-to-select highlights the asset on the page
- Opens the Media Library for quick replacements

## Tips

- Use it to quickly audit all images on a page
- Swap placeholder images after generating a page with AI
- Find background images that aren't visible in the regular editor view
