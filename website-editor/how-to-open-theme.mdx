---

Access global design settings.

<Steps>
  <Step title="Find Theme Button">
    Look for the Theme icon in preview header.
  </Step>
  <Step title="Click to Open">
    Click the button.
  </Step>
  <Step title="Panel Opens">
    See theme settings.
  </Step>
</Steps>
## What You Can Change

- Fonts
- Colors
- Buttons
- Links

## Tips

- Set early in project
- Changes affect all pages
- Preview changes live
