Switch between settings tabs quickly using keyboard shortcuts.

## Tab Shortcuts

| Shortcut | Tab |
|----------|-----|
| Cmd/Ctrl + 1 | General |
| Cmd/Ctrl + 2 | Domains |
| Cmd/Ctrl + 3 | Theme |
| Cmd/Ctrl + 4 | Integrations |
| Cmd/Ctrl + 5 | SEO |
| Cmd/Ctrl + 6 | Team |

## Using Shortcuts

1. Open Settings modal
2. Press the shortcut
3. Tab switches instantly

## Tips

- Faster than clicking tabs
- Works when modal is open
- Memorize common tabs
- Combine with other shortcuts
