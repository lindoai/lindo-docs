Actions available for each page.

## Available Actions

### Edit

Open page in editor.

### Duplicate

Create a copy of the page.

### Delete

Remove page permanently.

### Change Status

Toggle published/draft.

### Edit SEO

Configure page SEO settings.

## Accessing Actions

- Hover over page card
- Click actions menu
- Select action

## Tips

- Duplicate before major changes
- Delete with caution
- Set SEO before publishing
