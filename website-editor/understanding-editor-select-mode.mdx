---
title: "Understanding Select Mode"
description: "Element selection only mode in the editor"
---

Select mode lets you click on elements to select them without making changes.

## Overview

In Select mode:
- Click elements to select them
- View element properties
- No accidental edits
- Prepare for AI editing

## When to Use

### Before AI Editing
1. Switch to Select mode
2. Click the element you want to change
3. The element is highlighted
4. Send your AI prompt about that element

### Exploring the Page
- Click around to understand structure
- See how elements are nested
- Identify what to edit

## How to Access

1. Look at the mode tabs in the preview header
2. Click **Select**
3. You're now in Select mode

## Select vs Editor Mode

| Select Mode | Editor Mode |
|-------------|-------------|
| Click to select | Click to edit |
| No changes made | Direct editing |
| Safe exploration | Active editing |

## Visual Indicators

When you select an element:
- Highlight appears around it
- Element info may display
- Ready for AI commands
