---
title: "What is Page History"
description: "Version history for pages"
---

Page history tracks all versions of your page.

## What It Does

- Saves page versions
- Allows restoration
- Tracks changes over time
- Safety net for edits

## How It Works

1. You make changes
2. Versions saved automatically
3. Access history anytime
4. Restore previous versions

## Uses

- Undo major changes
- Compare versions
- Recover from mistakes
- Track evolution

## Tips

- Check before major edits
- Restore if needed
- Versions saved automatically
