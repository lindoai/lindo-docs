The preview area shows your website in real-time.

## What It Shows

- Live website preview
- Current page content
- Real-time updates
- Interactive elements

## Preview Features

### Device Views

Switch between:
- Desktop
- Tablet
- Mobile

### Fullscreen

Expand preview to full screen.

### Address Bar

Shows current page URL.

### Controls

- Undo/Redo
- Publish
- Page settings

## Interacting with Preview

- Click elements to select
- Scroll to navigate
- See changes instantly

## Tips

- Check all device views
- Use fullscreen for detail
- Changes appear in real-time
