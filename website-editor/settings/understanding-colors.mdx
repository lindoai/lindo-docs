Color settings define the visual palette of your website.

## Color Types

### Primary Color
Your main brand color used for:
- Buttons
- Links
- Accents
- Highlights

### Secondary Color
Supporting color for:
- Secondary buttons
- Hover states
- Complementary elements

### Background Color
The main page background:
- Usually white or dark
- Sets the overall tone
- Affects readability

### Text Color
Default text color:
- Should contrast with background
- Usually dark on light, light on dark

## Choosing Colors

### Brand Alignment
- Use your existing brand colors
- Maintain consistency with other materials
- Consider color psychology

### Contrast
- Ensure text is readable
- Test accessibility
- Check on different devices

### Harmony
- Use complementary colors
- Limit to 2-3 main colors
- Create visual hierarchy

## Color Formats

Enter colors as:
- Hex codes: `#FF5733`
- RGB: `rgb(255, 87, 51)`
