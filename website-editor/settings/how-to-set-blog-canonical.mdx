Configure the canonical URL for your blog listing page.

## What is Canonical URL

A canonical URL tells search engines which version of a page is the "official" one. This prevents duplicate content issues.

## Setting Canonical URL

1. Go to **Settings** → **SEO**
2. Find the **Blog Config** section
3. Locate the **Canonical URL** field
4. Enter the canonical URL
5. Save changes

## When to Use

### Custom Domain
If your blog is accessible at multiple URLs:
- `yoursite.com/blog`
- `www.yoursite.com/blog`

Set the canonical to your preferred version.

### Syndicated Content
If your blog content appears elsewhere, the canonical points back to the original.

## Format

Enter the full URL:
```
https://yoursite.com/blog
```

## Best Practices

- Use HTTPS
- Include or exclude www consistently
- Match your primary domain
- Keep it simple
