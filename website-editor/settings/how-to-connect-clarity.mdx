Connect Microsoft Clarity to see how visitors interact with your website through heatmaps and recordings.

## What is Clarity?

Microsoft Clarity provides:

- Heatmaps showing click patterns
- Session recordings
- Scroll depth analysis
- Free to use

## Connecting Clarity

1. Open your website in the editor
2. Click **Settings** tab
3. Go to **Integrations** section
4. Find **Clarity**
5. Enter your Clarity Project ID
6. Save

## Finding Your Project ID

1. Go to clarity.microsoft.com
2. Create or select a project
3. Go to Settings → Setup
4. Copy the Project ID

## After Connecting

Clarity will:

- Record visitor sessions
- Generate heatmaps
- Track user behavior
- Provide insights

## Tips

- Review recordings regularly
- Use heatmaps to optimize CTAs
- Identify usability issues
- Clarity is free and privacy-friendly
