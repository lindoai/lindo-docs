---

Intercom is a customer messaging platform for sales, marketing, and support.

## Overview

Intercom provides:
- Live chat
- Automated messaging
- Help center
- Product tours

## Features

### Messenger
- Chat widget on your website
- Customizable appearance
- Mobile-friendly

### Automation
- Targeted messages
- Chatbots
- Auto-responses

### Help Center
- Self-service articles
- Searchable knowledge base
- In-app help

### Product Tours
- Onboarding flows
- Feature announcements
- User guidance

## Use Cases

- Customer support
- Sales conversations
- User onboarding
- Product feedback

## Setup

1. Create an Intercom account
2. Get your App ID
3. Add the ID to Lindo.ai
4. Messenger appears on your site
