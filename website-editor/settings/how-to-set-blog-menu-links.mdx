Add navigation links to your blog listing page.

## Setting Menu Links

1. Go to **Settings** → **SEO**
2. Find the **Blog Config** section
3. Locate the **Menu Links** field
4. Add your links
5. Save changes

## Link Format

Add links as:
- Category pages
- Tag pages
- Archive links
- External resources

## Example Links

```
All Posts: /blog
Technology: /blog/category/technology
Design: /blog/category/design
Archive: /blog/archive
```

## Best Practices

### Keep It Simple
- Limit to 5-7 links
- Use clear labels
- Prioritize popular categories

### User Experience
- Help visitors find content
- Group related topics
- Include "All Posts" option

### SEO
- Use descriptive link text
- Link to important categories
- Create logical structure
