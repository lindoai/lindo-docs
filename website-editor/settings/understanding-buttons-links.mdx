Customize how buttons and links appear on your website.

## Button Properties

### Button Color
The background color of buttons:
- Primary buttons use primary color
- Can be customized per button type

### Button Hover Color
Color when hovering over buttons:
- Usually darker or lighter than default
- Provides visual feedback

### Button Text Color
The text inside buttons:
- Should contrast with button background
- Usually white or dark

### Button Roundness
Corner radius options:
- None (square corners)
- Small (slightly rounded)
- Medium (moderately rounded)
- Full (pill-shaped)

### Border Thickness
Button border options:
- None (no border)
- Thin
- Medium
- Thick

### Border Color
Color of button borders:
- Often matches button or text color
- Can create outline button style

## Link Properties

### Link Color
Default link text color:
- Usually matches primary color
- Should be distinguishable from regular text

### Link Hover Color
Color when hovering over links:
- Provides visual feedback
- Often darker or underlined
