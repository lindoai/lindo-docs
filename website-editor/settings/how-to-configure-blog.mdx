title: "How to Configure Blog Settings"
description: "Set up your blog list page configuration"
---

Configure how your blog list page appears and functions.

## Accessing Blog Config

1. Open your website in the editor
2. Click **Settings** tab
3. Go to **SEO** section
4. Find **Blog Config** area

## Blog Settings

### Blog Path

The URL path for your blog list page:

```
/blog
```

This creates: `yoursite.com/blog`

### Canonical URL

Set the canonical URL for SEO:

```
https://yoursite.com/blog
```

Helps prevent duplicate content issues.

### Blog Heading

The main title on your blog list page:

```
Our Blog
```

Or:

```
Latest Articles
```

### Blog Sub Heading

Description text below the heading:

```
Insights, tips, and updates from our team
```

### Menu Links

Add navigation links to your blog page:

- Link text
- Link URL
- Opens in new tab (optional)

## Saving Changes

Click **Save** after making changes. The blog list page updates automatically.

## Blog Post URLs

Individual blog posts have URLs like:

```
yoursite.com/blog/post-title
```

The path is based on the blog post's URL slug.

## Tips

- Use descriptive headings
- Keep sub headings concise
- Set canonical URL for SEO
- Add relevant menu links
- Preview after changes
