Find your sitemap URL to submit to search engines for better indexing.

## What is a Sitemap?

A sitemap:

- Lists all your pages
- Helps search engines crawl
- Auto-generated by Lindo.ai
- Updates when you publish

## Finding Sitemap URL

1. Open your website in the editor
2. Click **Settings** tab
3. Go to **SEO** section
4. Find **Sitemap** area
5. View the sitemap URL

## Sitemap Location

Your sitemap is at:

```
https://yoursite.com/sitemap.xml
```

## Submitting to Search Engines

### Google Search Console

1. Go to search.google.com/search-console
2. Select your property
3. Go to Sitemaps
4. Enter sitemap URL
5. Submit

### Bing Webmaster Tools

1. Go to bing.com/webmasters
2. Select your site
3. Go to Sitemaps
4. Submit sitemap URL

## Tips

- Submit after major changes
- Check for errors in Search Console
- Sitemap updates automatically
- Include in robots.txt
