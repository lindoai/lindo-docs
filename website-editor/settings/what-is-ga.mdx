Google Analytics (GA) is the industry-standard tool for website traffic analysis.

## Overview

GA provides detailed insights into:
- Who visits your website
- How they found you
- What they do on your site
- Conversion tracking

## Key Metrics

### Audience
- Users and sessions
- Demographics
- Devices and browsers
- Geographic location

### Acquisition
- Traffic sources
- Campaigns
- Search queries
- Referrals

### Behavior
- Page views
- Time on site
- Bounce rate
- User flow

### Conversions
- Goal completions
- E-commerce tracking
- Conversion paths

## Setup

1. Create a Google Analytics account
2. Get your Measurement ID (G-XXXXXXXXXX)
3. Add the ID to Lindo.ai
4. Data starts collecting

## GA4 vs Universal Analytics

Lindo.ai supports GA4 (Google Analytics 4), the current version of Google Analytics.
