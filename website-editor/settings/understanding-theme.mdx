The Theme section in Website Settings controls your website's visual design.

## What's Included

### Typography

- Heading font
- Body font

### Colors

- Primary color
- Secondary color
- Background color
- Text color

### Buttons & Links

- Button roundness
- Button colors
- Button hover colors
- Border settings
- Link colors

## Tips

- Match brand colors
- Test font combinations
- Ensure contrast
- Preview changes
