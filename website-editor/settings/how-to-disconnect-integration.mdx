---
title: "How to Disconnect Website Integration"
description: "Removing a third-party integration from your website"
---

Remove an integration that's no longer needed.

## Disconnecting an Integration

1. Go to **Settings** → **Integrations**
2. Find the integration you want to remove
3. Clear the ID/key field
4. Save changes

Or:
1. Click the disconnect/remove button if available
2. Confirm the disconnection

## Available Integrations

- Google Tag Manager
- Microsoft Clarity
- Google Analytics
- Crisp
- Intercom
- HubSpot Chat

## What Happens

When you disconnect an integration:
- The tracking/chat code is removed from your website
- Data collection stops immediately
- Historical data remains in the third-party service
- You can reconnect anytime

## Verifying Removal

After disconnecting:
1. Visit your live website
2. Check browser developer tools
3. Verify the integration script is no longer present
