Create custom TXT files for domain verification and other purposes.

## What are TXT Files?

TXT files are plain text files accessible at your domain:

```
yoursite.com/filename.txt
```

Common uses:
- Domain verification (Google, Bing)
- Security verification
- Custom configuration files

## Accessing TXT Files

1. Open your website in the editor
2. Click **Settings** tab
3. Go to **SEO** section
4. Find **TXT Files** area

## Creating a TXT File

1. Click **Add TXT File**
2. Enter the filename (without .txt extension)
3. Enter the file content
4. Click **Save**

### Example: Google Verification

Filename:
```
google1234567890abcdef
```

Content:
```
google-site-verification: google1234567890abcdef.html
```

## Editing a TXT File

1. Find the file in the list
2. Click to edit
3. Modify the content
4. Click **Save**

## Deleting a TXT File

1. Find the file in the list
2. Click the delete icon
3. Confirm deletion

## Common TXT Files

| Purpose | Filename Example |
|---------|------------------|
| Google verification | googleXXXXXXXX |
| Bing verification | BingSiteAuth |
| Security.txt | security |
| Custom | your-filename |

## Tips

- Use exact filenames required by services
- Copy content exactly as provided
- Test by visiting the URL
- Remove files when no longer needed

## Verification

After creating, verify the file is accessible:

```
https://yoursite.com/filename.txt
```

The content should display in your browser.
