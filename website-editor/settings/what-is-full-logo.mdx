---
title: "What is Full Logo"
description: "Your main website logo"
---

The full logo is your main brand logo displayed on your website.

## Overview

The full logo appears in:
- Website header/navigation
- Footer (depending on design)
- Other branded areas

## Recommended Specifications

| Property | Recommendation |
|----------|----------------|
| Size | 200x50 pixels |
| Format | PNG (for transparency) or JPG |
| Orientation | Horizontal |
| Background | Transparent (PNG) |

## Best Practices

### Design
- Keep it simple and readable
- Ensure it works on both light and dark backgrounds
- Include your brand name if possible

### File Quality
- Use high-resolution source files
- Optimize file size for web
- Test on different screen sizes

### Consistency
- Use the same logo across all websites
- Match your other brand materials

## Logo vs Favicon

| Logo | Favicon |
|------|---------|
| Full brand logo | Small icon |
| 200x50px | 32x32px |
| Header/footer | Browser tab |
| Horizontal | Square |
