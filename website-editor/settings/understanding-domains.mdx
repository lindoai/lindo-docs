The Domains section in Website Settings manages your website's URLs.

## What's Included

### Subdomain

Your default website address:

```
yoursite.lindo.site
```

### Custom Domain

Connect your own domain:

```
www.yourdomain.com
```

### DNS Configuration

- Connection options
- Required DNS records
- Status verification

## Tips

- Start with subdomain
- Add custom domain for professional presence
- Follow DNS setup carefully
- Wait for propagation
