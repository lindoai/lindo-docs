Adjust your logo image when uploading to ensure it displays correctly.

## Using the Crop Tool

1. Go to **Settings** → **General**
2. Click to upload a logo
3. Select your image file
4. The crop tool appears
5. Adjust the crop area
6. Click **Save** or **Apply**

## Crop Guidelines

### Full Logo
- Recommended size: 200x50 pixels
- Horizontal orientation works best
- Leave some padding around the logo

### Favicon
- Recommended size: 32x32 pixels
- Square format required
- Simple designs work best at small sizes

## Tips

- Use PNG for logos with transparency
- Keep important elements within the crop area
- Preview the result before saving
- Use high-resolution source images

## If Crop Tool Doesn't Appear

Some image formats may skip the crop tool:
- Pre-sized images matching requirements
- Certain file types

You can always re-upload to access the crop tool again.
