title: "What is Crisp"
description: "Live chat integration for your website"
---

Crisp is a live chat platform that lets you communicate with website visitors in real-time.

## Overview

Add a chat widget to your website so visitors can:
- Ask questions
- Get support
- Request information
- Start conversations

## Features

### Live Chat
- Real-time messaging
- Typing indicators
- Read receipts
- File sharing

### Chatbot
- Automated responses
- FAQ handling
- Lead qualification

### Team Inbox
- Shared conversations
- Assignment rules
- Response templates

## Benefits

- **Increase conversions** - Answer questions instantly
- **Improve support** - Help visitors in real-time
- **Capture leads** - Collect contact information
- **Build relationships** - Personal communication

## Setup

1. Create a Crisp account
2. Get your Website ID
3. Add the ID to Lindo.ai
4. Chat widget appears on your site
