Add a description below your blog heading.

## Setting the Sub Heading

1. Go to **Settings** → **SEO**
2. Find the **Blog Config** section
3. Locate the **Blog Sub Heading** field
4. Enter your description
5. Save changes

## Examples

Good sub headings:
- "Insights, tips, and updates from our team"
- "Stay up to date with the latest industry news"
- "Expert advice for growing your business"
- "Thoughts and ideas from [Company]"

## Best Practices

### Be Descriptive
- Explain what readers will find
- Set expectations
- Highlight value

### Keep It Brief
- 1-2 sentences maximum
- Easy to scan
- Get to the point

### SEO Benefits
- Include relevant keywords
- Helps with search visibility
- Appears in meta description
