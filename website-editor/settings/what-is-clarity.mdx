Microsoft Clarity is a free analytics tool that provides heatmaps and session recordings.

## Overview

Clarity helps you understand how visitors interact with your website through visual analytics.

## Features

### Heatmaps
See where visitors:
- Click most often
- Scroll to
- Move their mouse

### Session Recordings
Watch actual visitor sessions:
- See their journey
- Identify confusion points
- Understand behavior

### Insights
Automatic detection of:
- Rage clicks (frustration)
- Dead clicks (broken elements)
- Excessive scrolling

## Benefits

- **Free** - No cost for any traffic level
- **Privacy-focused** - Masks sensitive data
- **Easy setup** - Just add your Project ID

## How It Works

1. Create a Clarity account
2. Get your Project ID
3. Add the ID to Lindo.ai
4. Data starts collecting

## Use Cases

- Identify UX problems
- Optimize conversion paths
- Understand user behavior
- Test design changes
