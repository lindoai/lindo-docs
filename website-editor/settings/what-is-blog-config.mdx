Blog config controls how your blog listing page appears and functions.

## Overview

When you have blog posts, Lindo.ai creates a blog listing page that shows all your posts. Blog config lets you customize this page.

## Configuration Options

### Blog Path
The URL path for your blog:
- Default: `/blog`
- Custom: `/articles`, `/news`, `/posts`

### Canonical URL
The authoritative URL for your blog:
- Helps with SEO
- Prevents duplicate content issues

### Blog Heading
The main title on your blog page:
- Example: "Our Blog"
- Example: "Latest News"

### Blog Sub Heading
Description text below the heading:
- Explains what the blog is about
- Helps with SEO

### Menu Links
Navigation links on the blog page:
- Categories
- Tags
- Archive links

## Accessing Blog Config

1. Go to **Settings** → **SEO**
2. Find the **Blog Config** section
3. Configure your options
4. Save changes
