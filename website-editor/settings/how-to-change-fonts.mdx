Change your website's fonts to match your brand identity.

## Accessing Font Settings

1. Open your website in the editor
2. Click **Settings** tab
3. Go to **Theme** section
4. Find **Typography** area

## Font Types

### Heading Font

Used for:
- Page titles (H1)
- Section headings (H2, H3)
- Important text

### Body Font

Used for:
- Paragraph text
- Lists
- General content

## Changing Fonts

### Change Heading Font

1. Find "Heading Font" dropdown
2. Click to open font list
3. Select your preferred font
4. Preview updates automatically

### Change Body Font

1. Find "Body Font" dropdown
2. Click to open font list
3. Select your preferred font
4. Preview updates automatically

## Available Fonts

Lindo.ai includes popular web fonts:

- Inter
- Roboto
- Open Sans
- Lato
- Montserrat
- Poppins
- Playfair Display
- Merriweather
- And many more

## Font Pairing Tips

Good combinations:

| Heading | Body |
|---------|------|
| Playfair Display | Source Sans Pro |
| Montserrat | Open Sans |
| Poppins | Inter |
| Roboto Slab | Roboto |

## Best Practices

- Limit to 2-3 fonts maximum
- Ensure readability on all devices
- Test with actual content
- Consider brand consistency

## Saving Changes

Font changes save automatically. Preview your website to see the full effect across all pages.
