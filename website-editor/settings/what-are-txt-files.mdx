---
title: "What are TXT Files"
description: "Custom text files for your website"
---

TXT files are plain text files you can add to your website root.

## Common Uses

### Domain Verification
Services often require TXT files for verification:
- Google Search Console
- Email services
- Third-party tools

### Security Files
Some security standards require:
- security.txt
- humans.txt

### Custom Files
Any plain text file you need at your domain root.

## How They Work

TXT files are accessible at:
```
https://yoursite.com/filename.txt
```

## Creating TXT Files

1. Go to **Settings** → **SEO**
2. Find the TXT Files section
3. Click **Add TXT File**
4. Enter filename and content
5. Save

## File Naming

- Use lowercase letters
- Include .txt extension
- No spaces (use hyphens)
- Keep names short

## Examples

### security.txt
```
Contact: security@yoursite.com
Expires: 2025-12-31T23:59:59.000Z
```

### Verification File
```
google-site-verification: abc123xyz
```
