The Integrations section in Website Settings connects external services.

## Available Integrations

### Analytics

- Google Tag Manager
- Microsoft Clarity
- Google Analytics

### Chat & Support

- Crisp
- Intercom
- HubSpot Chat

## How to Connect

1. Get ID from service dashboard
2. Enter in Lindo.ai
3. Save changes
4. Verify on published site

## Tips

- Connect analytics early
- Choose one chat widget
- Test after connecting
- Monitor integration status
