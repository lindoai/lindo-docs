---
title: "How to Add Schema Markup"
description: "Add structured data for better search results"
---

Add schema markup to help search engines understand your content and potentially show rich results.

## Adding Schema

1. Open your website in the editor
2. Click **Settings** tab
3. Go to **SEO** section
4. Find **Schema Markup** area
5. Enter your JSON-LD code
6. Save

## Schema Format

Use JSON-LD format:

```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Your Company",
  "url": "https://yoursite.com"
}
```

## Common Schema Types

- Organization
- LocalBusiness
- Product
- Article
- FAQPage

## Testing Schema

1. Go to search.google.com/test/rich-results
2. Enter your URL
3. Check for errors
4. Fix any issues

## Tips

- Use valid JSON-LD
- Test before publishing
- Don't duplicate schema
- Keep data accurate
