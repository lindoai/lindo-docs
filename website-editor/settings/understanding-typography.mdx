Typography settings control the fonts used throughout your website.

## Font Types

### Heading Font
Used for:
- Page titles (H1)
- Section headings (H2, H3)
- Card titles
- Navigation items

### Body Font
Used for:
- Paragraph text
- List items
- Form labels
- General content

## Available Fonts

Lindo.ai provides a curated selection of web-safe fonts:
- Sans-serif options (modern, clean)
- Serif options (traditional, elegant)
- Display fonts (decorative, unique)

## Font Pairing

Good font combinations:
- Sans-serif heading + Sans-serif body (modern)
- Serif heading + Sans-serif body (classic)
- Display heading + Simple body (creative)

## Best Practices

### Readability
- Body text should be highly readable
- Avoid decorative fonts for body text
- Test on different screen sizes

### Consistency
- Use the same fonts across all pages
- Limit to 2 fonts maximum
- Maintain visual hierarchy

### Performance
- Web fonts are optimized for loading
- Fewer fonts = faster load times
