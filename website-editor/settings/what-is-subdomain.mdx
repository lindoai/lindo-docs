Every website gets a free subdomain on Lindo.ai's domain.

## Default Subdomain

When you create a website, it's automatically assigned a subdomain:
- Format: `yoursite.lindo.ai`
- Or: `yoursite.yourworkspace.com` (if using white label)

## Subdomain Features

### Always Available
Your subdomain works immediately, no DNS setup required.

### Free
Included with all plans, no additional cost.

### SSL Included
HTTPS is automatically enabled.

## Customizing Your Subdomain

1. Go to **Settings** → **Domains**
2. Find the **Subdomain** field
3. Enter your preferred subdomain
4. Save changes

### Subdomain Rules
- Lowercase letters and numbers only
- Hyphens allowed (not at start/end)
- Must be unique within the workspace
- 3-63 characters

## Subdomain vs Custom Domain

| Feature | Subdomain | Custom Domain |
|---------|-----------|---------------|
| Cost | Free | Requires Pro+ plan |
| Setup | Instant | DNS configuration needed |
| Branding | Shows lindo.ai | Your own domain |
| SSL | Automatic | Automatic |
