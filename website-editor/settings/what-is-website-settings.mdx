Website Settings is a comprehensive configuration panel for customizing your website's appearance, functionality, and integrations.

## Accessing Settings

1. Open your website in the editor
2. Click the **Settings** tab in the header
3. The settings modal opens

## Settings Sections

### General

Configure basic website properties:

- **Logo**: Upload your website logo
- **Favicon**: Browser tab icon
- **Social Links**: Social media URLs
- **Widgets**: Cookie consent, age gate
- **Danger Zone**: Delete website

### Domains

Manage website URLs:

- **Subdomain**: Default website address
- **Custom Domain**: Your own domain name
- **DNS Configuration**: Connection settings

### Theme

Customize design elements:

- **Typography**: Heading and body fonts
- **Colors**: Primary, secondary, background, text
- **Buttons & Links**: Button and link styling

### Integrations

Connect third-party services:

- Google Tag Manager
- Microsoft Clarity
- Google Analytics
- Crisp
- Intercom
- HubSpot Chat

### SEO

Search engine optimization:

- Sitemap configuration
- Robots.txt settings
- Schema markup
- TXT files
- Blog configuration

### Team

Collaboration settings:

- Invite team members
- Manage permissions
- Revoke access

### Workflows

Background automation:

- Blog pipeline
- Page workflows
- Email notifications

## Saving Changes

Changes are saved automatically when you:

- Click Save button
- Close the modal
- Switch sections

## Tips

- Configure General settings first
- Set up domain before going live
- Connect analytics early
- Review SEO settings before publishing
