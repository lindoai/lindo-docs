description: "Button border options"
---

Border thickness controls the outline around buttons.

## Thickness Options

| Option | Description |
|--------|-------------|
| None | No border (solid button) |
| Thin | Subtle outline |
| Medium | Visible border |
| Thick | Bold outline |

## Button Styles

### Solid Buttons (No Border)
- Filled background color
- Most common style
- Strong visual presence

### Outline Buttons (With Border)
- Transparent or light background
- Border defines the button
- Secondary action style

### Ghost Buttons
- Transparent background
- Thin border
- Minimal visual weight

## When to Use Borders

### Primary Actions
Usually solid buttons without visible borders.

### Secondary Actions
Often outline buttons with borders.

### Tertiary Actions
Ghost buttons with thin borders.

## Design Tips

- Use borders consistently
- Match border color to design
- Consider hover states
- Test visibility on different backgrounds
