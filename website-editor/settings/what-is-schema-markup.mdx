title: "What is Schema Markup"
description: "JSON-LD structured data for search engines"
---

Schema markup is structured data that helps search engines understand your content.

## Overview

Schema markup uses JSON-LD format to provide explicit information about your website, business, products, and content.

## Benefits

### Rich Results
Schema can enable:
- Star ratings in search results
- Business information cards
- Product details
- FAQ accordions
- Event listings

### Better Understanding
Search engines can:
- Categorize your content accurately
- Display relevant information
- Connect related entities

## Common Schema Types

| Type | Use Case |
|------|----------|
| Organization | Company information |
| LocalBusiness | Physical business location |
| Product | E-commerce products |
| Article | Blog posts and news |
| FAQPage | FAQ sections |
| Event | Events and webinars |

## Example Schema

```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Your Company",
  "url": "https://yoursite.com",
  "logo": "https://yoursite.com/logo.png"
}
```

## Adding Schema

1. Go to **Settings** → **SEO**
2. Find the Schema Markup section
3. Add your JSON-LD code
4. Save changes
