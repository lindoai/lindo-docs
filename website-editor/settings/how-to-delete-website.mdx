---
title: "How to Delete Website"
description: "Permanently remove a website"
---

Delete a website when you no longer need it. This action is permanent.

## Deleting a Website

1. Open your website in the editor
2. Click **Settings** tab
3. Go to **General** section
4. Scroll to **Danger Zone**
5. Click **Delete Website**
6. Confirm deletion

## What Gets Deleted

When you delete:

- All pages are removed
- All products are removed
- All analytics data is lost
- Domain configuration is removed
- Cannot be undone

## Before Deleting

Consider:

- Export the website first
- Save any important content
- Inform stakeholders
- Check for active customers

## Alternatives

Instead of deleting:

- Unpublish pages
- Remove from active use
- Archive for later

## Tips

- Double-check before confirming
- Export as backup first
- Consider implications
- Deletion is permanent
