---
title: "How to Set Social Links"
description: "Configure social media links for your website"
---

Add social media links to your website so visitors can find and follow you on various platforms.

## Accessing Social Links

1. Open your website in the editor
2. Click **Settings** tab
3. Go to **General** section
4. Find **Social Links** area

## Available Platforms

You can add links for:

- Facebook
- Instagram
- X (Twitter)
- LinkedIn
- YouTube
- TikTok
- Discord
- WhatsApp

## Adding a Social Link

1. Find the platform you want to add
2. Enter your profile URL in the input field
3. Changes save automatically

### URL Format Examples

| Platform | URL Format |
|----------|------------|
| Facebook | https://facebook.com/yourpage |
| Instagram | https://instagram.com/yourusername |
| X (Twitter) | https://x.com/yourusername |
| LinkedIn | https://linkedin.com/company/yourcompany |
| YouTube | https://youtube.com/@yourchannel |
| TikTok | https://tiktok.com/@yourusername |
| Discord | https://discord.gg/yourinvite |
| WhatsApp | https://wa.me/yourphonenumber |

## Additional Contact Info

You can also set:

- **Support Email**: Contact email address
- **Business Address**: Physical location

## Where Links Appear

Social links typically appear in:

- Website footer
- Contact sections
- Header (if designed that way)

The AI uses these links when generating social sections.

## Tips

- Use complete URLs including https://
- Verify links work before publishing
- Keep profiles consistent with your brand
- Update if you change usernames
