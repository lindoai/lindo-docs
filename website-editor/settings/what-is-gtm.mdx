description: "Tag management integration for your website"
---

Google Tag Manager (GTM) lets you manage tracking codes and scripts without editing your website code.

## Overview

GTM acts as a container for:
- Analytics tracking
- Conversion pixels
- Marketing tags
- Custom scripts

## Benefits

### Centralized Management
- All tags in one place
- No code changes needed
- Easy to add/remove tags

### Flexibility
- Deploy tags without developers
- Test before publishing
- Version control

### Performance
- Asynchronous loading
- Reduced page load impact
- Efficient tag firing

## How It Works

1. Create a GTM account
2. Get your Container ID (GTM-XXXXXX)
3. Add the ID to Lindo.ai
4. Manage tags in GTM dashboard

## Common Tags

- Google Analytics
- Facebook Pixel

- Google Ads conversion
