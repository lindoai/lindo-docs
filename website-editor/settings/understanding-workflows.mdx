The Workflows section in Website Settings manages background automation.

## Available Workflows

### Blog Pipeline

Generate multiple blog posts automatically.

### Page Workflow

Create pages in the background.

## Features

- Background processing
- Email notifications
- Progress tracking
- Error handling

## Tips

- Use for bulk content
- Enable notifications
- Monitor progress
- Review results
