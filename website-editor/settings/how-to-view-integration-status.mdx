---
title: "How to View Integration Status"
description: "Checking which integrations are connected"
---

See which third-party integrations are active on your website.

## Viewing Status

1. Go to **Settings** → **Integrations**
2. Review each integration section
3. Connected integrations show their ID/key
4. Empty fields indicate no connection

## Integration Indicators

### Connected
- Field contains an ID or key
- Integration is active on your website
- Tracking/chat is enabled

### Not Connected
- Field is empty
- Integration is not active
- No tracking/chat from this service

## Available Integrations

| Integration | What It Does |
|-------------|--------------|
| Google Tag Manager | Tag management |
| Microsoft Clarity | Heatmaps and recordings |
| Google Analytics | Traffic analytics |
| Crisp | Live chat |
| Intercom | Customer messaging |
| HubSpot Chat | CRM chat widget |

## Testing Integrations

To verify an integration is working:
1. Visit your live website
2. Open browser developer tools
3. Check for the integration's script
4. Verify data appears in the third-party dashboard
