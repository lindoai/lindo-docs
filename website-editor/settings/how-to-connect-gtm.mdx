title: "How to Connect Google Tag Manager"
description: "Add GTM integration to your website"
---

Connect Google Tag Manager to manage all your tracking tags in one place.

## What is GTM?

Google Tag Manager lets you:

- Manage multiple tracking codes
- Add tags without editing code
- Control when tags fire
- Test before publishing

## Connecting GTM

1. Open your website in the editor
2. Click **Settings** tab
3. Go to **Integrations** section
4. Find **Google Tag Manager**
5. Enter your GTM Container ID
6. Save

## Finding Your Container ID

1. Log into tagmanager.google.com
2. Select your container
3. Find ID in format: GTM-XXXXXXX
4. Copy the ID

## After Connecting

GTM is now active on your website. You can:

- Add tags in GTM dashboard
- Configure triggers
- Test with Preview mode
- Publish changes

## Tips

- Test tags before publishing
- Use GTM for all tracking
- Keep container organized
- Document your tags
