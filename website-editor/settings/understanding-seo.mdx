The SEO section in Website Settings helps optimize your site for search engines.

## What's Included

### Sitemap

- Auto-generated sitemap
- URL for submission

### Robots.txt

- Default or manual control
- Crawl rules

### Schema Markup

- JSON-LD structured data
- Rich results support

### TXT Files

- Custom text files
- Domain verification

### Blog Config

- Blog path
- Heading and description
- Menu links

## Tips

- Submit sitemap to search engines
- Add relevant schema
- Configure blog settings
- Monitor search performance
