title: "How to Duplicate a Product"
description: "Copy an existing product to create a similar one"
---

Duplicate a product to quickly create variations or similar offerings.

## Duplicating a Product

1. Open your website in the editor
2. Go to **Pages** tab
3. Find the product to duplicate
4. Click the three-dot menu
5. Select **Duplicate**
6. Edit the copy as needed

## What Gets Copied

The duplicate includes:

- Product name (with "Copy" suffix)
- Description
- Price and frequency
- Checkout options
- Product image

## After Duplicating

1. Update the product name
2. Adjust price if different
3. Change description
4. Update image if needed
5. Save changes

## Use Cases

Duplicate products for:

- Size variations
- Color options
- Pricing tiers
- Seasonal versions

## Tips

- Rename immediately to avoid confusion
- Update all relevant details
- Test the new product
- Keep products organized
