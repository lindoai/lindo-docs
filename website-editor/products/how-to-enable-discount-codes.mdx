---
title: "How to Enable Discount Codes"
description: "Allow customers to use promo codes at checkout"
---

Enable discount codes so customers can apply promotional codes when purchasing your products.

## Enabling Discount Codes

1. Open your website in the editor
2. Go to **Pages** tab
3. Click on a product
4. Open product settings
5. Find **Checkout Options**
6. Toggle **Discount Codes** on
7. Save changes

## How It Works

When enabled:

- Checkout shows a promo code field
- Customers enter their code
- Discount is applied to total
- You manage codes in Stripe

## Managing Codes

Discount codes are managed in Stripe:

1. Log into Stripe dashboard
2. Go to Products → Coupons
3. Create new coupon codes
4. Set discount amount/percentage
5. Set expiration if needed

## Types of Discounts

In Stripe, you can create:

- Percentage off (e.g., 20% off)
- Fixed amount off (e.g., $10 off)
- Limited use codes
- Time-limited codes

## Tips

- Create codes for promotions
- Track code usage in Stripe
- Set reasonable limits
- Communicate codes to customers
