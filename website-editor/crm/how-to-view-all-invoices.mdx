---
title: "How to View All Website Invoices"
description: "Using the Billing tab to see all invoices"
---

See all invoices across all customers in one place.

## Accessing All Invoices

1. Go to the **CRM** tab in the website editor
2. Click on the **Billing** tab
3. View the complete invoice list

## Invoice List

Each invoice shows:
- Customer name
- Service name
- Amount and currency
- Status (open, paid, refunded)
- Date

## Filtering Invoices

You can filter by:
- Status (all, open, paid, refunded)
- Search by customer name

## Invoice Actions

From the list, you can:
- View invoice details
- Copy payment link
- Download invoice PDF
- Process refunds (for paid invoices)

## Invoice Status

| Status | Meaning |
|--------|---------|
| Open | Awaiting payment |
| Paid | Payment received |
| Refunded | Payment returned |
