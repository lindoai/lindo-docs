---
title: "How to Set Invoice Service Name"
description: "Configuring the service description on invoices"
---

Give your invoices clear, descriptive service names.

## Setting Service Name

When creating an invoice:

1. Open customer details
2. Go to the Billing tab
3. Click **Create Invoice**
4. Enter the **Service Name** field
5. Complete other invoice details
6. Send the invoice

## Best Practices

### Be Descriptive
Good examples:
- "Website Design - Homepage"
- "Monthly Hosting - March 2024"
- "E-commerce Setup"

Avoid:
- "Service"
- "Payment"
- "Invoice"

### Be Consistent
Use consistent naming for recurring services so customers recognize charges.

### Include Details
For project work, include:
- Project name
- Phase or milestone
- Date range if applicable

## Examples by Service Type

| Service Type | Example Name |
|--------------|--------------|
| Website design | "Website Design - Company Name" |
| Monthly hosting | "Monthly Hosting - April 2024" |
| Maintenance | "Website Maintenance - Q1" |
| Add-on feature | "Blog Setup & Configuration" |
