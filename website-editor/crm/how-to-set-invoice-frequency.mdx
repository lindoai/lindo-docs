description: "Choosing one-time vs recurring billing"
---

Configure whether an invoice is one-time or recurring.

## Setting Frequency

When creating an invoice:

1. Open customer details
2. Go to the Billing tab
3. Click **Create Invoice**
4. Find the **Frequency** dropdown
5. Select the billing frequency
6. Complete other details
7. Send the invoice

## Frequency Options

| Option | Description |
|--------|-------------|
| One-time | Single payment, no recurrence |
| Monthly | Billed every month |
| Yearly | Billed once per year |
| Weekly | Billed every week |

## When to Use Each

### One-time
- Project-based work
- Setup fees
- One-off services

### Monthly
- Hosting services
- Maintenance plans
- Ongoing support

### Yearly
- Annual subscriptions
- Yearly maintenance contracts
- Discounted annual plans

### Weekly
- Short-term intensive projects
- Weekly retainers

## Recurring Billing

When you select monthly, yearly, or weekly:
- A subscription is created
- Customer is billed automatically
- View in the Subscriptions tab
- Can be cancelled anytime
