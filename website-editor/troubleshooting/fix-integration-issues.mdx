If your website integrations aren't working, try these troubleshooting steps.

## Common Issues

### Integration Not Working

**Symptoms:**
- Chat widget not appearing
- Analytics not tracking
- No data in third-party dashboard

**Solutions:**

1. **Verify ID/Key**
   - Check the integration ID is correct
   - Copy directly from service dashboard
   - Remove extra spaces

2. **Check Page is Published**
   - Integrations only work on published pages
   - Preview may not show integrations

3. **Clear Cache**
   - Clear browser cache
   - Try incognito mode
   - Wait for CDN propagation

### Specific Integration Issues

#### Google Analytics

- Verify Measurement ID format (G-XXXXXXXX)
- Check GA4 property is active
- Test with Real-time reports

#### Chat Widgets

- Verify widget ID
- Check for ad blockers
- Test in different browser

## Tips

- Double-check all IDs
- Test after publishing
- Use browser DevTools to check for errors
- Contact integration support if needed
