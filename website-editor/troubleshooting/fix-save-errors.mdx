title: "How to Fix Save/Publish Errors"
description: "Troubleshoot saving and publishing issues"
---

If you're having trouble saving or publishing your pages, try these solutions.

## Common Save Errors

### Network Error

**Symptoms:**
- Save fails
- "Network error" message

**Solutions:**
1. Check internet connection
2. Refresh the page
3. Try again

### Unauthorized Error

**Symptoms:**
- "Unauthorized" message
- Session expired

**Solutions:**
1. Log out
2. Log back in
3. Try saving again

### Validation Error

**Symptoms:**
- "Validation error" message
- Specific field issues

**Solutions:**
1. Check for invalid content
2. Review error details
3. Fix and retry

## Publish Errors

### Page Won't Publish

1. Check page status
2. Ensure no validation errors
3. Try saving first, then publish

### Changes Not Appearing

1. Clear browser cache
2. Wait a few minutes
3. Hard refresh (Cmd/Ctrl + Shift + R)

## Tips

- Save frequently
- Check for error messages
- Note specific errors
- Contact support if persistent
