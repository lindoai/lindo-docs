---

Revert mistakes and restore changes in the editor.

## Undo

### Using Toolbar

Click the **Undo** arrow in the preview toolbar.

### Using Keyboard

Press `Cmd/Ctrl + Z`

### What It Does

Reverts the last change you made.

## Redo

### Using Toolbar

Click the **Redo** arrow in the preview toolbar.

### Using Keyboard

Press `Cmd/Ctrl + Shift + Z`

### What It Does

Restores a change you just undid.

## Undo/Redo Scope

### What Can Be Undone

- Text edits
- AI-generated changes
- Element modifications
- Style changes

### What Cannot Be Undone

- Published changes (use page history)
- Deleted pages
- Settings changes

## Multiple Undos

You can undo multiple times:
1. Press `Cmd/Ctrl + Z` repeatedly
2. Each press undoes one more change
3. Continue until you reach desired state

## Tips

### Before Major Changes

1. Save current state
2. Make your changes
3. Undo if unhappy

### When Undo Isn't Enough

Use Page History for older versions:
1. Click History icon
2. Find the version you want
3. Restore it

### Keyboard is Faster

- `Cmd/Ctrl + Z` - Undo
- `Cmd/Ctrl + Shift + Z` - Redo
- Much faster than clicking

## Common Scenarios

**AI made unwanted change:**
Press `Cmd/Ctrl + Z` immediately.

**Went too far back:**
Press `Cmd/Ctrl + Shift + Z` to redo.

**Need older version:**
Use Page History instead.
