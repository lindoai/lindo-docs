---
title: "What is Page Workflow"
description: "Background page creation automation"
---

Page workflows let you create pages in the background while you continue working.

## Overview

Instead of waiting for AI to generate a page, you can:
1. Start a page workflow
2. Continue with other tasks
3. Get notified when complete
4. Review the finished page

## Benefits

### Time Efficiency
- No waiting for generation
- Work on other tasks
- Batch multiple pages

### Background Processing
- Pages generate asynchronously
- No browser tab needed
- Email notifications available

## Creating a Page Workflow

1. Access the Workflows section
2. Select Page workflow type
3. Enter your page prompt
4. Configure AI tools (optional)
5. Enable notifications (optional)
6. Start the workflow

## Workflow Status

| Status | Meaning |
|--------|---------|
| Running | Page is being generated |
| Completed | Page is ready |
| Error | Something went wrong |

## After Completion

When the workflow finishes:
- Page appears in your Pages list
- Review the generated content
- Make any needed adjustments
- Publish when ready
