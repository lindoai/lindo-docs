---
title: "How to Return to Workspace"
description: "Using the back button in website dropdown"
---

Go back to the main workspace from the editor.

## Methods

### Method 1: Logo Click

Click the logo in the top-left corner.

### Method 2: Website Dropdown

1. Click the website name dropdown
2. Select "Back to Workspace"

## What Happens

- Editor closes
- Workspace loads
- Your work is saved

## Tips

- Changes auto-save
- Use either method
- Quick navigation
