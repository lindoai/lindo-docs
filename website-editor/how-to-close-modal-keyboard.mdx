Quickly close modals and dialogs using the Escape key.

## Close Shortcut

Press **Escape** to close the current modal.

## What It Closes

The Escape key closes:

- Settings modal
- Confirmation dialogs
- Dropdown menus
- Popovers

## When to Use

Use Escape to:

- Quickly dismiss modals
- Cancel actions
- Close without saving (be careful)
- Navigate faster

## Tips

- Faster than clicking X
- Works on most modals
- May not save changes
- Check for unsaved work first
