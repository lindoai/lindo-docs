The Insights tab provides analytics for your individual website.

## What's Included

### Statistics Cards

- Visitors
- Page views
- Response time
- Page count
- Blog count

### Traffic Chart

Visual representation of traffic over time.

### Breakdowns

- Top pages
- Operating systems
- Browsers
- Devices
- Countries
- Referrers
- UTM campaigns

## Tips

- Check analytics regularly
- Monitor trends
- Track campaign performance
- Use data for decisions
