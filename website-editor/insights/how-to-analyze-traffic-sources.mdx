Understand where your website visitors are coming from.

## Viewing Traffic Sources

1. Go to the **Insights** tab
2. Find the **Referrers** section
3. See which sites send you traffic

## Understanding Referrers

Referrers show the source of your traffic:

| Referrer | Meaning |
|----------|---------|
| google.com | Organic search traffic |
| facebook.com | Social media traffic |
| Direct | Typed URL or bookmarks |
| Your other sites | Internal referrals |

## UTM Campaigns

For marketing campaigns, use UTM parameters:

### UTM Parameters
- `utm_source` - Traffic source (e.g., facebook)
- `utm_medium` - Marketing medium (e.g., cpc, email)
- `utm_campaign` - Campaign name

### Example URL
```
yoursite.com?utm_source=facebook&utm_medium=cpc&utm_campaign=summer_sale
```

### Viewing UTM Data
1. Go to **Insights** tab
2. Find the **UTM Campaigns** section
3. See traffic by campaign

## Using This Data

### Identify Best Channels
- Which sources bring the most traffic?
- Which sources have best engagement?
- Where should you invest more?

### Measure Campaigns
- Track specific marketing efforts
- Compare campaign performance
- Calculate ROI

### Optimize Strategy
- Double down on working channels
- Improve or abandon poor performers
- Test new traffic sources
