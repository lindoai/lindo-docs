title: "How to View Website Analytics"
description: "Access and navigate the Insights tab"
---

View detailed analytics for your website to understand traffic patterns and visitor behavior.

## Accessing Analytics

1. Open your website in the editor
2. Click **Insights** tab in the header
3. Analytics dashboard loads

## Navigating the Dashboard

### Statistics Overview

Top section shows key metrics:

- Total visitors
- Total page views
- Average response time
- Page count
- Blog count

### Traffic Chart

Main visualization showing:

- Page views (line graph)
- Visitors (line graph)
- Time period on x-axis

Hover over points to see exact values.

### Detailed Breakdowns

Scroll down to see:

- Top pages
- Operating systems
- Browsers
- Devices
- Countries
- Referrers
- UTM campaigns

## Setting Date Range

1. Find the date picker
2. Select preset range or custom dates
3. Analytics update automatically

### Preset Ranges

- Last 7 days
- Last 30 days
- Last 90 days

### Custom Range

1. Click date picker
2. Select start date
3. Select end date
4. Apply

## Refreshing Data

Click the **Refresh** button to:

- Load latest data
- Update all metrics
- Refresh charts

## Exporting Data

Currently, analytics are viewed in-dashboard. For external analysis, note key metrics manually or use connected analytics tools (Google Analytics, etc.).

## Tips

- Check analytics weekly
- Compare time periods
- Note traffic spikes
- Track after campaigns
- Monitor page performance
