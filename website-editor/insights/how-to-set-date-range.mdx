title: "How to Set Website Date Range"
description: "Filter analytics by specific time periods"
---

Set a date range to view analytics for specific time periods.

## Setting Date Range

1. Open your website in the editor
2. Click **Insights** tab
3. Find the date picker
4. Select your range
5. Analytics update automatically

## Preset Ranges

Quick options:

- Last 7 days
- Last 30 days
- Last 90 days

## Custom Range

For specific dates:

1. Click the date picker
2. Select start date
3. Select end date
4. Apply the range

## Tips

- Compare similar periods
- Track campaign performance
- Monitor seasonal trends
- Export data for reports
