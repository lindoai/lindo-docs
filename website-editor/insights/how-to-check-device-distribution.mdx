---
title: "How to Check Device Distribution"
description: "Understanding what devices your visitors use"
---

See the breakdown of desktop, mobile, and other devices visiting your website.

## Viewing Device Data

1. Go to the **Insights** tab
2. Find the **Devices** section
3. See the breakdown by device type

## Device Categories

| Category | Description |
|----------|-------------|
| Desktop | Computers and laptops |
| Mobile | Smartphones |
| Tablet | iPads and tablets |
| Bot | Search engine crawlers |

## Using This Data

### Mobile Optimization
If mobile traffic is high:
- Ensure mobile experience is excellent
- Test all pages on mobile devices
- Check load times on mobile

### Desktop Focus
If desktop dominates:
- May indicate B2B audience
- Optimize for larger screens
- Consider complex layouts

### Bot Traffic
Some bot traffic is normal:
- Search engine crawlers
- Social media preview bots
- Monitoring services

High bot traffic may indicate:
- Good SEO (search engines crawling)
- Potential spam (if excessive)

## Industry Benchmarks

Typical distribution varies by industry:
- E-commerce: 60-70% mobile
- B2B: 50-60% desktop
- News/Media: 70-80% mobile
