Statistics cards provide quick overview of your website's key metrics.

## Available Metrics

### Visitors

Total unique visitors to your website in the selected period.

### Page Views

Total number of page loads across all pages.

### Avg Response Time

Average server response time in milliseconds.

### Pages

Total number of pages on your website.

### Blogs

Total number of blog posts.

## Reading the Cards

Each card shows:

- Metric name
- Current value
- Comparison to previous period (if available)

## Tips

- Check metrics regularly
- Compare to previous periods
- Monitor for anomalies
- Use for reporting
