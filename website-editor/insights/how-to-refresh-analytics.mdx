---
title: "How to Refresh Website Analytics"
description: "Update analytics with the latest data"
---

Refresh your analytics to see the most recent visitor data.

## Refreshing Analytics

1. Open your website in the editor
2. Click **Insights** tab
3. Click the **Refresh** button
4. Data updates with latest information

## When to Refresh

Refresh analytics when:

- Checking real-time traffic
- After publishing changes
- Monitoring campaigns
- Verifying tracking works

## Data Freshness

Analytics data:

- Updates periodically
- May have slight delay
- Refresh for latest numbers

## Tips

- Don't refresh excessively
- Allow time for data processing
- Check after major changes
- Monitor trends over time
