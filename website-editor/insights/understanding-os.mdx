The Operating Systems section shows which OS your visitors use.

## Common Operating Systems

- Windows
- macOS
- iOS
- Android
- Linux
- Others

## What You See

For each OS:

- OS name
- Visitor count
- Percentage of total

## Why It Matters

OS data helps you:

- Understand device preferences
- Prioritize testing
- Plan mobile strategy
- Debug OS-specific issues

## Tips

- High iOS/Android = mobile focus
- High Windows/macOS = desktop focus
- Test on popular platforms
- Consider OS-specific features
