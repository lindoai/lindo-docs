---
title: "How to Restore Previous Version"
description: "Rolling back to earlier version"
---

Restore a page to a previous state.

<Steps>
  <Step title="Open Page History">
    Access the history panel.
  </Step>
  <Step title="Browse Versions">
    See list of saved versions.
  </Step>
  <Step title="Select Version">
    Click the version to preview.
  </Step>
  <Step title="Click Restore">
    Confirm restoration.
  </Step>
  <Step title="Page Restored">
    Page reverts to selected version.
  </Step>
</Steps>
## When to Restore

- Made unwanted changes
- Prefer earlier design
- Recover from mistakes
- Compare versions

## Important Notes

- Current version saved first
- Can restore again if needed
- Review before confirming

## Tips

- Preview before restoring
- Note what changed
- Use as safety net
