---
title: "How to Access the Website Editor"
description: "Opening a website from the workspace to edit it"
---

Open any website in the editor to make changes and updates.

## From the Workspace

1. Go to your workspace dashboard
2. Find the website you want to edit in the grid
3. Click on the website card
4. The website opens in the editor

## From Website Actions

1. Hover over a website card
2. Click the **Edit** option from the actions menu
3. The website opens in the editor

## Editor Interface

When you open a website, you'll see:

- **Left Panel** - AI chat for making changes
- **Right Panel** - Live preview of your website
- **Header** - Navigation tabs (Design, Pages, Insights, CRM, Settings)
- **Website Dropdown** - Switch websites, return to workspace

## First Time in Editor

If this is your first time:
1. The Design tab is selected by default
2. The AI chat panel is open
3. Your homepage is displayed in the preview
