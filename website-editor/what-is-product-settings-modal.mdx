Configure all product settings in one place.

## Settings Available

### Basic Info

- Product name
- Description
- Price
- Currency

### Pricing

- Frequency (one-time, monthly, etc.)
- Discount codes

### Media

- Product image
- Digital file (for downloads)

### Checkout

- Phone collection
- Address collection
- Quantity adjustment

### Post-Purchase

- Thank you message
- Redirect URL

## Accessing

Click a product to open settings modal.

## Tips

- Complete all fields
- Test checkout flow
- Clear thank you message
