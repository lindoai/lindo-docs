---
title: "Understanding Page Status"
description: "Published vs Draft indicators"
---

Page status indicates whether a page is live or hidden.

## Status Types

### Published

- Page is live
- Visible to visitors
- Indexed by search engines
- Shows green badge

### Draft

- Page is hidden
- Not visible to visitors
- Not indexed
- Shows gray badge

## Changing Status

1. Find the page
2. Click action menu
3. Select "Change Status"
4. Status toggles

## Tips

- Use draft for work in progress
- Publish when ready
- Unpublish to hide temporarily
- Status affects SEO
