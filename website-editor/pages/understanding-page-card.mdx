title: "What is the Page Card"
description: "Page thumbnail and info display"
---

Page cards display your pages in the Pages tab.

## Card Elements

- Page thumbnail
- Page name
- Page URL
- Status badge (Published/Draft)
- Action menu

## Actions Available

Click the menu to:

- Edit page
- Duplicate page
- Change status
- Edit SEO
- Delete page

## Tips

- Thumbnails update automatically
- Status shows at a glance
- Use actions for quick management
- Click card to edit
