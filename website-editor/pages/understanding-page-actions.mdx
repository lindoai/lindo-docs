description: "Available actions for pages"

Page actions let you manage your pages quickly.

## Available Actions

### Edit

Open the page in the editor.

### Duplicate

Create a copy of the page.

### Change Status

Switch between Published and Draft.

### Edit SEO

Open SEO settings for the page.

### Delete

Remove the page permanently.

## Accessing Actions

1. Find the page in Pages tab
2. Click the three-dot menu
3. Select an action

## Tips

- Use duplicate for similar pages
- Change status to hide pages
- Edit SEO for search optimization
- Delete carefully (permanent)
