The Pages tab organizes content into different types.

## Page Types

### Pages

Regular website pages:

- Home page
- About page
- Contact page
- Service pages

### Blogs

Blog posts and articles:

- News updates
- Articles
- Tutorials
- Announcements

### Products

E-commerce products:

- Physical products
- Digital downloads
- Services
- Subscriptions

## Switching Types

Use the tabs in Pages section to filter by type.

## Tips

- Organize by type
- Use appropriate type for content
- Blogs have special features
- Products have checkout
