---
title: "Understanding Website Status"
description: "Published vs Draft indicators"
---

Website status indicates whether your site is live or still in progress.

## Status Types

### Published

- Website is live
- Accessible via URL
- Visitors can see it
- Green badge indicator

### Draft

- Website is not live
- Only you can see it
- Work in progress
- Gray badge indicator

## Where Status Appears

- Website cards in grid
- Website dropdown in editor
- Pages tab (per-page status)

## Publishing Flow

1. **Create** - Website starts as draft
2. **Edit** - Make changes
3. **Publish** - Go live
4. **Update** - Changes auto-publish (or save as draft)

## Page vs Website Status

| Level | Meaning |
|-------|---------|
| Website | Overall site status |
| Page | Individual page status |

A published website can have draft pages that aren't visible yet.

## Checking Status

### In Workspace

Look at the badge on website cards.

### In Editor

Check the website dropdown or Pages tab.

## Changing Status

### Publish a Website

1. Open website in editor
2. Make your changes
3. Click "Publish"

### Unpublish a Page

1. Go to Pages tab
2. Click page actions
3. Select "Unpublish"

## Tips

- Draft lets you work without visitors seeing
- Publish when ready
- Individual pages can be draft while site is published
