---
title: "How to Embed Your Workspace"
description: "Using iframe to embed in other sites"
---

Add your workspace to another website using an iframe.

## Prerequisites

- Embed enabled
- Embed code copied

<Steps>
  <Step title="Get Embed Code">
    From Settings → Reseller, copy the embed code.
  </Step>
  <Step title="Open Your Website">
    Access your website's HTML or CMS.
  </Step>
  <Step title="Add the Code">
    Paste the iframe code where you want the workspace:

    ```html
    <iframe
      src="https://app.yourdomain.com/embed"
      width="100%"
      height="800px"
      frameborder="0"
      allow="clipboard-write">
    </iframe>
    ```
  </Step>
  <Step title="Adjust Dimensions">
    Set appropriate width and height:
    - Width: Usually 100%
    - Height: 600-1000px recommended
  </Step>
  <Step title="Save and Test">
    Save your changes and test the embed.
  </Step>
</Steps>
## Styling Tips

### Full Width

```html
<iframe
  src="..."
  style="width: 100%; height: 100vh; border: none;">
</iframe>
```

### Fixed Size

```html
<iframe
  src="..."
  width="1200"
  height="800"
  style="border: none;">
</iframe>
```

## Common Issues

### Height Too Short

Increase height value or use viewport height.

### Scrollbars

Set `overflow: hidden` on container.

### Authentication

Ensure cookies are allowed for iframe.

## Tips

- Test on different devices
- Handle mobile responsively
- Consider loading states
- Match your site's design
