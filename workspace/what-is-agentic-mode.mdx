Agentic Mode is the modern AI-powered interface for building and editing websites.

## What It Is

Agentic Mode provides:
- AI chat-based editing
- Natural language commands
- Intelligent suggestions
- Streamlined workflow

## Agentic vs Legacy

| Agentic Mode | Legacy Builder |
|--------------|----------------|
| AI chat interface | Traditional editor |
| Natural language | Manual controls |
| Modern UI | Classic UI |
| Faster editing | More manual work |

## Features

### AI Chat

Describe what you want in plain language:
- "Add a testimonials section"
- "Change the hero background to blue"
- "Make the buttons rounded"

### Smart Suggestions

AI understands context and suggests:
- Relevant changes
- Design improvements
- Content enhancements

### Streamlined Workflow

- Fewer clicks
- Faster results
- More intuitive

## Enabling Agentic Mode

### For Your Workspace

1. Go to Settings → White Label
2. Find "Agentic Mode" toggle
3. Turn it on

### For All Clients

1. Go to Settings → White Label
2. Click "Enable for All Clients"
3. Confirm migration

## Who Should Use It

Agentic Mode is recommended for:
- New users
- Anyone comfortable with AI
- Those wanting faster workflows

## Tips

- Try agentic for new projects
- Use natural language
- Be specific in requests
