Build a complete website with multiple interconnected pages.

## What is Multi-Page

A multi-page website has:
- Separate pages for different content
- Navigation menu linking pages
- Consistent header/footer
- Dedicated page for each topic

## When to Use

Multi-page works best for:
- Business websites
- Agency sites
- E-commerce
- Content-heavy sites
- Sites needing SEO for multiple topics

<Steps>
  <Step title="Write Your Prompt">
    Describe your website and mention the pages you want:

    > "Create a multi-page website for my marketing agency. Include Home, About, Services, Portfolio, and Contact pages."
  </Step>
  <Step title="Add Tools (Optional)">
    - Set language
    - Choose dark/light mode
    - Define colors
    - Add reference URL
  </Step>
  <Step title="Generate">
    Click send to create your website.
  </Step>
  <Step title="Review">
    Your multi-page site is ready with navigation.
  </Step>
</Steps>
## Typical Pages

A good multi-page site includes:
- **Home** - Main landing page
- **About** - Company/personal story
- **Services/Products** - What you offer
- **Portfolio/Work** - Examples
- **Blog** - Articles (optional)
- **Contact** - Get in touch

## Navigation

Multi-page sites have:
- Header navigation menu
- Links to all pages
- Consistent across pages
- Mobile-responsive menu

## Tips

- Plan your pages before generating
- Keep navigation simple (5-7 items)
- Consistent design across pages
- Clear page purposes

## Single vs Multi-Page

| Multi-Page | Single-Page |
|------------|-------------|
| Multiple pages | One scrolling page |
| Page navigation | Anchor navigation |
| More content | Focused message |
| Better for SEO | Faster to build |

## Adding More Pages Later

After generation, add pages:
1. Open website in editor
2. Go to Pages tab
3. Click "Add Page"
4. Use AI to generate content
