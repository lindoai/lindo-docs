The AI Prompt Builder is the main interface for creating websites with AI. It's the large input area in the center of your workspace.

## Components

### Text Input

The large textarea where you describe your website:
- Placeholder: "Describe your website idea..."
- Supports multi-line input
- No character limit (but concise prompts work best)

### Toolbar

Below the input, a toolbar with:
- **(+) Add Tools** - Open tools menu
- **Page Structure** - Single-page or multi-page dropdown
- **Submit Button** - Arrow icon to generate

### AI Tools Menu

Click **(+)** to access:
- **Language** - Set content language
- **Mode** - Dark or Light theme
- **Color Palette** - Brand colors
- **Website URL** - Reference site for inspiration

### Prompt Builder Button

**✨ Prompt Builder** opens a full-screen modal with:
- Sample prompts by industry
- Sample sections to include
- Design style options

## How It Works

1. **Enter description** - Type what you want to build
2. **Add tools** (optional) - Set language, mode, colors
3. **Choose structure** - Single or multi-page
4. **Submit** - Click the arrow button
5. **Wait** - AI generates your website (30-60 seconds)
6. **Result** - Website appears in "Your Websites" grid

## Example Prompts

**Simple:**
```
Coffee shop website with menu and location
```

**Detailed:**
```
Modern SaaS landing page for ProjectHub, a project management tool.
Hero with headline "Manage Projects Smarter", features grid with
6 benefits, pricing table with 3 tiers, testimonials from 3 customers,
FAQ section, and final CTA. Dark mode with purple accent.
```

## Tips for Better Results

1. **Include business name** - AI uses it throughout
2. **Specify sections** - List what you want included
3. **Mention design style** - Modern, minimal, bold, etc.
4. **Add context** - Industry, audience, location
5. **Use tools** - Set colors and mode for consistency
