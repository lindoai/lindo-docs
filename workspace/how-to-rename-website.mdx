---

Change the name of your website.

<Steps>
  <Step title="Find the Website">
    Locate the website in your Websites Grid.
  </Step>
  <Step title="Open Actions Menu">
    Hover over the website card to reveal actions.
  </Step>
  <Step title="Click Rename">
    Click **Rename** from the actions menu.
  </Step>
  <Step title="Enter New Name">
    Type the new name for your website.
  </Step>
  <Step title="Save">
    Press Enter or click Save.
  </Step>
</Steps>
## What Changes

- Website name in your grid
- Name in website dropdown
- Internal reference

## What Doesn't Change

- Website URL/subdomain
- Page content
- Custom domain
- Published pages

## Changing the URL

To change the actual website address:

### Subdomain

1. Open website in editor
2. Go to Settings → Domains
3. Edit the subdomain field

### Custom Domain

1. Open website in editor
2. Go to Settings → Domains
3. Add or change custom domain

## Naming Best Practices

Good names help organization:

**By client:**
```
Acme Corp - Main Website
Acme Corp - Landing Page
```

**By project:**
```
2024 Q1 Campaign
Product Launch - Beta
```

**By type:**
```
Restaurant - Italian Bistro
Portfolio - Photography
```

## Tips

- **Be descriptive** - Easy to identify later
- **Include client name** - For agency work
- **Use consistent format** - Easier to search
