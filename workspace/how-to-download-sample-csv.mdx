Get the template CSV for batch importing workflows.

<Steps>
  <Step title="Open Workflows">
    Click **Workflows** in the sidebar.
  </Step>
  <Step title="Find Batch Import">
    Look for the batch import section.
  </Step>
  <Step title="Click Download Sample">
    Click to download the sample CSV.
  </Step>
  <Step title="Open the File">
    Open in Excel, Google Sheets, or text editor.
  </Step>
</Steps>
## Sample CSV Format

```csv
name,prompt,language,mode
My Website,Create a landing page for...,en,light
Client Site,Build a portfolio for...,es,dark
```

## Columns

| Column | Required | Description |
|--------|----------|-------------|
| name | Yes | Website name |
| prompt | Yes | AI prompt |
| language | No | Content language |
| mode | No | light or dark |

## Tips

- Follow format exactly
- Don't change column headers
- Test with one row first
- Save as CSV format
