Add Microsoft Clarity for heatmaps and recordings.

## What is Clarity

Microsoft Clarity provides:
- Heatmaps showing clicks
- Session recordings
- User behavior insights
- Free analytics tool

## Prerequisites

- Microsoft Clarity account
- Clarity project ID

<Steps>
  <Step title="Get Your Project ID">
    1. Log into clarity.microsoft.com
    2. Create or select project
    3. Go to Settings → Setup
    4. Copy the project ID
  </Step>
  <Step title="Open Integrations">
    Go to Settings → White Label → Integrations tab.
  </Step>
  <Step title="Find Clarity">
    Locate Microsoft Clarity card.
  </Step>
  <Step title="Enter Project ID">
    Paste your Clarity project ID.
  </Step>
  <Step title="Save">
    Click Save or Connect.
  </Step>
</Steps>
## What You Get

After connecting:
- Heatmaps of user clicks
- Session recordings
- Scroll depth data
- Rage click detection

## Where It Applies

Clarity tracks:
- Workspace login page
- Client dashboard
- Workspace pages

## Tips

- Review recordings regularly
- Look for UX issues
- Check rage clicks
- Use insights to improve

## Privacy Note

Clarity automatically:
- Masks sensitive data
- Respects privacy settings
- GDPR compliant
