---
title: "How to Enable Agentic Mode for Client"
description: "Switch a specific client to the AI-powered interface"
---

Enable Agentic Mode for individual clients to give them access to the AI-powered website builder.

## What is Agentic Mode?

Agentic Mode provides:

- AI-powered website creation
- Chat-based editing interface
- Modern builder experience

## Enabling for a Client

1. Open **CRM** from sidebar
2. Select the client
3. Go to **Info** tab
4. Find **Agentic Mode** toggle
5. Turn it on
6. Click **Save**

## What Changes for the Client

When enabled, the client sees:

- AI prompt builder
- Chat-based editor
- AI agents for editing
- Modern interface

## When to Enable

Enable Agentic Mode when:

- Client wants AI features
- They're comfortable with AI tools
- You want them to self-serve
- They need advanced editing

## Tips

- Explain the new interface to clients
- Provide brief training if needed
- Monitor their credit usage
- Offer support during transition
