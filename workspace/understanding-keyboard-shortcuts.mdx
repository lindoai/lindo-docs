Keyboard shortcuts help you navigate and work faster in Lindo.ai.

## Available Shortcuts

### Settings Navigation

Switch between settings tabs quickly:

| Shortcut | Action |
|----------|--------|
| Cmd/Ctrl + 1 | Go to Workspace tab |
| Cmd/Ctrl + 2 | Go to Appearance tab |
| Cmd/Ctrl + 3 | Go to Domain tab |
| Cmd/Ctrl + 4 | Go to Integrations tab |
| Cmd/Ctrl + 5 | Go to Custom Code tab |
| Cmd/Ctrl + 6 | Go to Members tab |

### General

| Shortcut | Action |
|----------|--------|
| Escape | Close modal |

## Using Shortcuts

1. Open the Settings modal
2. Press the keyboard shortcut
3. The corresponding tab activates

## Tips

- Shortcuts work when Settings modal is open
- Use Cmd on Mac, Ctrl on Windows/Linux
- Combine with mouse for fastest workflow
