---
title: "How to Create Reseller Website"
description: "Setting up your reseller landing page"
---

Create a landing page to promote your white-labeled platform.

## Prerequisites

- Reseller plan
- White label configured

<Steps>
  <Step title="Open Reseller Settings">
    Go to Settings → Reseller tab.
  </Step>
  <Step title="Find Reseller Website">
    Locate the reseller website section.
  </Step>
  <Step title="Enable Website">
    Turn on the reseller website feature.
  </Step>
  <Step title="Customize Content">
    Edit your landing page:
    - Headline
    - Description
    - Features
    - Pricing
    - CTA
  </Step>
  <Step title="Save and Publish">
    Save your changes to go live.
  </Step>
</Steps>
## Customization Options

### Headlines

- Main headline
- Subheadline
- Section titles

### Features

- Feature list
- Descriptions
- Icons

### Pricing

- Plan names
- Prices
- Features per plan

### Call to Action

- Button text
- Sign-up flow
- Contact options

## Tips

- Write compelling copy
- Highlight unique value
- Include social proof
- Clear pricing
