---
title: "What is the Color Palette Tool"
description: "Defining custom color schemes for websites"
---

The Color Palette Tool sets the accent colors for your generated website.

## Available Palettes

Choose from predefined color options:
- **Lime** - Fresh green tones
- **Blue** - Professional blue tones
- **Purple** - Creative purple tones
- **Orange** - Energetic orange tones
- **Pink** - Playful pink tones
- **Red** - Bold red tones
- **Teal** - Calm teal tones
- **Yellow** - Bright yellow tones
- **Gray** - Neutral gray tones

## How to Use

1. Click **(+)** in the AI Prompt Builder toolbar
2. Select **Color Palette**
3. Choose a color from the dropdown
4. Generate your website

## What It Affects

The color palette influences:
- **Primary buttons** - CTAs and main actions
- **Accent elements** - Highlights, badges, icons
- **Links** - Text link colors
- **Hover states** - Interactive element colors
- **Gradients** - If used in design

## Default Behavior

If you don't set a palette:
- AI chooses based on your prompt
- Industry keywords influence color choice
- "Finance" might get blue, "Health" might get green

## Combining with Mode

Color palette works with the Mode tool:
- Light mode + Blue = Blue accents on white
- Dark mode + Blue = Blue accents on dark

## Changing Colors After Generation

To adjust colors on an existing website:

**Via AI Chat:**
```
Change the color scheme to purple
```

**Via Theme Settings:**
1. Open website in editor
2. Go to Settings → Theme
3. Adjust Primary Color, Secondary Color, etc.

## Custom Colors

For exact brand colors, use Theme Settings after generation:
1. Generate with closest palette
2. Open Settings → Theme
3. Enter exact hex codes for each color

## Tips

- **Brand consistency** - Match your existing brand colors
- **Contrast** - Ensure buttons stand out from backgrounds
- **Accessibility** - High contrast improves readability
