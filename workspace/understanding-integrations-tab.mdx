The Integrations tab connects third-party services to your workspace.

## Accessing

1. Click **Settings** in sidebar
2. Select **White Label**
3. Click **Integrations** tab

## Available Integrations

### Stripe

Payment processing for client billing:
- Accept payments
- Send invoices
- Manage subscriptions
- Required for CRM billing

### Google Tag Manager

Tag management for analytics:
- Add tracking codes
- Manage marketing tags
- No code changes needed

### Microsoft Clarity

Heatmaps and session recordings:
- See how users interact
- Identify UX issues
- Free analytics tool

### Google Analytics

Website traffic analytics:
- Track visitors
- Monitor behavior
- Measure conversions

### Crisp

Live chat support:
- Real-time chat
- Customer support
- Chat widget on workspace

### Intercom

Customer messaging platform:
- In-app messaging
- Customer support
- User engagement

### HubSpot Chat

CRM-integrated chat:
- Live chat
- CRM integration
- Lead capture

## Integration Status

Each integration shows:
- **Connected** - Active and working
- **Not Connected** - Needs setup
- **Configure** button - Set up integration

## Setup Process

1. Click integration card
2. Enter required credentials
3. Save configuration
4. Status updates to connected

## Tips

- Connect Stripe first for billing
- Add analytics for insights
- Choose one chat solution
- Test after connecting
