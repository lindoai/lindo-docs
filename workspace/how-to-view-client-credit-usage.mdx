---
title: "How to View Client Credit Usage"
description: "Monitor how clients are using their allocated credits"
---

Track how much of their allocated credits each client has used.

## Accessing Credit Usage

1. Open **CRM** from sidebar
2. Select the client
3. Go to **Credits** tab
4. View usage information

## Credit Information

The credits tab shows:

- **Monthly Allocation**: Credits given per month
- **Used This Month**: Credits consumed
- **Remaining**: Available credits
- **Allocation History**: Past allocations

## Understanding Usage

Monitor for:

- High usage clients
- Unused allocations
- Usage patterns
- Need for adjustments

## Adjusting Allocations

If a client needs more credits:

1. Stay on Credits tab
2. Update monthly allocation
3. Or add one-time credits
4. Save changes

## Tips

- Review usage monthly
- Adjust allocations as needed
- Communicate with high-usage clients
- Consider upgrading heavy users
