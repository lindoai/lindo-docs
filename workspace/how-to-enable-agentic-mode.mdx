---
title: "How to Enable Agentic Mode"
description: "Switching to AI-powered interface"
---

Activate the modern AI-powered interface.

<Steps>
  <Step title="Open Settings">
    Click **Settings** in the sidebar.
  </Step>
  <Step title="Go to White Label">
    Select **White Label** section.
  </Step>
  <Step title="Find Agentic Mode">
    In the Workspace tab, locate the toggle.
  </Step>
  <Step title="Enable Agentic Mode">
    Turn the toggle on.
  </Step>
  <Step title="Save Changes">
    Click **Save** or **Update**.
  </Step>
</Steps>
## What Changes

With Agentic Mode:
- AI chat interface enabled
- Natural language editing
- Modern UI experience
- Streamlined workflows

## For New vs Existing Clients

### New Clients

Automatically get agentic mode.

### Existing Clients

Need separate migration (see below).

## Migrating All Clients

To enable for existing clients:

1. Go to Settings → White Label
2. Find "Enable for All Clients"
3. Click the button
4. Confirm migration

This updates all existing client accounts.

## Tips

- Test agentic yourself first
- Communicate changes to clients
- Provide guidance on new interface
