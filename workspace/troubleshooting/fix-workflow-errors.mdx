title: "How to Fix Workflow Errors"
description: "Debug failed workflows"
---

If your workflows are failing, try these troubleshooting steps.

## Common Issues

### Insufficient Credits

**Solution:**
1. Check credit balance
2. Purchase more credits
3. Retry workflow

### Invalid Prompt

**Solution:**
1. Review prompt content
2. Remove special characters
3. Simplify if too complex
4. Retry with clearer prompt

### Rate Limiting

**Solution:**
1. Wait before retrying
2. Reduce batch size
3. Space out workflows

### Network Issues

**Solution:**
1. Check internet connection
2. Retry workflow
3. Contact support if persistent

## Checking Errors

1. Open Workflows
2. Find errored workflow
3. Click to view details
4. Note error message

## Tips

- Start with small batches
- Test prompts individually first
- Monitor workflow progress
- Document successful prompts
