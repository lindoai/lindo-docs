If you're having issues with invoices, try these solutions.

## Common Issues

### Invoice Not Sending

**Solution:**
1. Verify client email is correct
2. Check Stripe is connected
3. Resend or copy payment link
4. Send link manually

### Payment Not Processing

**Solution:**
1. Verify Stripe verification complete
2. Check client's payment method
3. Review Stripe dashboard for errors

### Wrong Amount

**Solution:**
1. Create new invoice with correct amount
2. Refund incorrect invoice if paid
3. Communicate with client

### Invoice Not Appearing

**Solution:**
1. Refresh the page
2. Check correct client selected
3. Verify invoice was created

## Tips

- Double-check details before sending
- Keep Stripe verification current
- Communicate with clients
- Monitor invoice status
