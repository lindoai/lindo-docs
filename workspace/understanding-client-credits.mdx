Manage AI credits for your agency clients.

## How It Works

You allocate credits to clients from your pool:
- Set monthly allocation
- Add one-time credits
- Track usage
- Monitor consumption

## Credit Types

### Monthly Allocation

- Recurring credits each month
- Resets on billing date
- Automatic allocation

### One-Time Credits

- Manual addition
- Doesn't reset
- For special projects

## Viewing Client Credits

1. Open CRM
2. Select client
3. Go to Credits tab
4. See usage and allocation

## Credit Information

| Field | Description |
|-------|-------------|
| Monthly | Recurring allocation |
| Used | Credits consumed |
| Remaining | Available credits |
| Reset Date | When credits refresh |

## Allocation History

Track all credit changes:
- Date of allocation
- Amount added
- Type (monthly/one-time)

## Tips

- Monitor usage patterns
- Adjust allocations as needed
- Use for pricing tiers
- Track for billing
