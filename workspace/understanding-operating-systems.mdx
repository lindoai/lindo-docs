See what operating systems your visitors use.

## What It Shows

Breakdown of visitors by OS:
- Windows
- macOS
- iOS
- Android
- Linux
- Other

## Reading the Data

- Percentage of visitors
- Count per OS
- Visual chart

## Common Distributions

| OS | Typical % |
|----|-----------|
| Windows | 40-50% |
| iOS | 20-30% |
| Android | 15-25% |
| macOS | 10-20% |

## Uses

- Understand audience
- Prioritize testing
- Optimize for popular OS
- Device strategy

## Tips

- Test on top OS
- Consider mobile vs desktop
- Track changes over time
