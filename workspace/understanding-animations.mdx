Lindo.ai uses subtle animations to enhance the user experience.

## Animation Types

### Fade Animations

- **fadeIn**: Elements appearing smoothly
- **fadeOut**: Elements disappearing smoothly

### Slide Animations

- **slideIn**: Panels sliding into view
- **slideOut**: Panels sliding away

### Loading Animations

- **shimmer**: Skeleton loading effect
- **shine**: Highlight sweep effect
- **pulse**: Gentle pulsing effect

## Where Animations Appear

- Modal transitions
- Panel toggles
- Loading states
- Button interactions
- Notifications

## Performance

Animations are:

- Hardware accelerated
- Optimized for performance
- Subtle and non-distracting
- Consistent across the app

## Tips

- Animations indicate state changes
- Wait for animations to complete
- Smooth transitions mean things are working
- Jerky animations may indicate issues
