---
title: "How to Use the Add Tools Menu"
description: "Opening and selecting tools from the (+) menu"
---

The Add Tools menu lets you enhance your AI prompts with specific settings.

<Steps>
  <Step title="Find the Add Button">
    Look for the (+) button in the toolbar below the prompt input.
  </Step>
  <Step title="Click the Button">
    A dropdown menu appears with available tools.
  </Step>
  <Step title="Select a Tool">
    Click the tool you want to add:
    - **Language** - Set content language
    - **Mode** - Choose dark or light theme
    - **Color Palette** - Define custom colors
    - **Website URL** - Use reference design
  </Step>
  <Step title="Configure the Tool">
    Each tool has its own settings:
    - Language: Select from dropdown
    - Mode: Toggle dark/light
    - Colors: Pick colors
    - URL: Enter website address
  </Step>
  <Step title="Tool Added">
    The tool appears as a tag in your prompt area.
  </Step>
</Steps>
## Adding Multiple Tools

You can combine tools:
1. Add Language tool
2. Add Mode tool
3. Add Color Palette
4. All apply to generation

## Removing Tools

Click the X on any tool tag to remove it.

## Tool Combinations

| Combination | Result |
|-------------|--------|
| Language + Mode | Localized dark/light site |
| Colors + URL | Custom colors with reference layout |
| All tools | Fully customized generation |

## Tips

- Add tools before writing your prompt
- Combine for precise results
- Remove unused tools to simplify
