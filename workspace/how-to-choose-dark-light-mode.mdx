---
title: "How to Choose Dark or Light Mode"
description: "Setting theme mode for generated websites"
---

Set the theme for your AI-generated website.

## Light Mode

- White/light backgrounds
- Dark text
- Clean, bright appearance
- Traditional look

## Dark Mode

- Dark/black backgrounds
- Light text
- Modern, sleek appearance
- Reduced eye strain

<Steps>
  <Step title="Open Add Tools Menu">
    Click the (+) button in the toolbar.
  </Step>
  <Step title="Select Mode Tool">
    Click "Mode" from the menu.
  </Step>
  <Step title="Choose Mode">
    Toggle between:
    - **Light** - Light backgrounds
    - **Dark** - Dark backgrounds
  </Step>
  <Step title="Mode Tag Added">
    The mode appears as a tag in your prompt.
  </Step>
  <Step title="Generate">
    Your website will use the selected theme.
  </Step>
</Steps>
## When to Use Each

| Light Mode | Dark Mode |
|------------|-----------|
| Corporate sites | Tech/SaaS products |
| Traditional businesses | Creative portfolios |
| E-commerce | Gaming/entertainment |
| Healthcare | Photography |

## Tips

- Consider your audience
- Match your brand
- Dark mode is trending for tech
- Light mode is safer for broad audiences

## Changing Mode Later

You can change the theme after generation:
1. Open website in editor
2. Go to Settings → Theme
3. Adjust colors
