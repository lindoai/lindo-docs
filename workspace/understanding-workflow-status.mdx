---
title: "Understanding Workflow Status"
description: "Running, completed, error states"
---

Each workflow has a status indicating its progress.

## Status Types

### Running

- Currently processing
- AI generating content
- Wait for completion
- May take minutes

### Completed

- Successfully finished
- Website created
- Ready to use
- No action needed

### Error

- Something went wrong
- Check error message
- May need to retry
- Contact support if persistent

## Status Indicators

| Status | Color | Meaning |
|--------|-------|---------|
| Running | Blue/Yellow | In progress |
| Completed | Green | Success |
| Error | Red | Failed |

## Handling Errors

If a workflow errors:
1. Check the error message
2. Verify your prompt
3. Try again
4. Contact support if repeated

## Tips

- Be patient with running
- Check completed results
- Address errors promptly
- Refresh to update status
