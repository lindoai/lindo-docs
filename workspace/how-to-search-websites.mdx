title: "How to Search Websites"
description: "Using the search input to filter websites"
---

Quickly find websites in your workspace using search.

<Steps>
  <Step title="Locate the Search Input">
    Find the search input above the Websites Grid in your dashboard.
  </Step>
  <Step title="Type Your Search">
    Enter part of the website name:
    - Search is case-insensitive
    - Partial matches work
    - Results update as you type
  </Step>
  <Step title="View Results">
    The grid filters to show only matching websites.
  </Step>
  <Step title="Clear Search">
    Delete your search text or click the clear button to show all websites again.
  </Step>
</Steps>
## Search Examples

| Search | Finds |
|--------|-------|
| `coffee` | "Coffee Shop", "Bean & Coffee Co" |
| `land` | "Landing Page", "Greenland Tours" |
| `2024` | "Project 2024", "Campaign 2024" |

## Tips

- **Use unique words** - More specific = fewer results
- **Search by project** - If you name sites by project
- **Search by client** - If you include client names

## What Search Doesn't Do

- Search page content (only website names)
- Search by date
- Search by status
- Advanced filters

## Managing Many Websites

If you have lots of websites:
1. Use consistent naming conventions
2. Include client names in website names
3. Use search frequently
4. Consider organizing by project

## Naming Conventions

Good naming helps searching:
- `ClientName - ProjectType`
- `2024-01 - Campaign Name`
- `Industry - Business Name`
