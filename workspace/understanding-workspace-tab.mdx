The Workspace tab in White Label settings controls core workspace configuration.

## Accessing

1. Click **Settings** in sidebar
2. Select **White Label**
3. You're on the Workspace tab

## Settings Available

### Workspace Name

Your workspace's display name:
- Shown in header
- Used in client communications
- Customizable anytime

### WebApp Domain

Subdomain for your workspace:
- Format: yourname.lindo.ai
- Or custom domain if configured
- Where clients access the platform

### Client Registration

Allow clients to self-register:
- Toggle on/off
- Clients can create accounts
- Useful for self-service model

### Agentic Mode

Enable AI-powered interface:
- New modern interface
- AI chat-based editing
- Toggle for workspace

### Enable for All Clients

Migrate existing clients to agentic:
- One-click migration
- Updates all client accounts
- Can't be undone

## Configuration Tips

- Set workspace name to your brand
- Configure domain before inviting clients
- Enable agentic for modern experience
