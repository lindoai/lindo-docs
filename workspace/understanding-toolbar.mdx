The toolbar in the AI prompt builder provides quick access to generation tools.

## Toolbar Location

The toolbar appears below the main prompt input area, showing available tools and your credit balance.

## Available Tools

### Add Tools Button (+)

Opens the tools menu with:
- Language tool
- Mode tool (dark/light)
- Color palette
- Website URL reference

### Language Tool

Set the content language for generated websites.
- Auto-detect from prompt
- Manual selection from 11 languages

### Mode Tool

Choose the theme for generated websites:
- Light mode
- Dark mode

### Color Palette Tool

Define custom colors:
- Primary color
- Secondary color
- Background color
- Text color

### Website URL Tool

Use an existing website as design reference:
- Enter any URL
- AI analyzes the design
- Applies similar style

## Credit Display

The toolbar shows your current credit balance:
- Available credits
- Click to view details

## Using Tools

1. Click the (+) button
2. Select a tool
3. Configure settings
4. Tool appears as a tag in your prompt
5. Generate with tools applied

## Tool Tags

When you add a tool, it appears as a removable tag:
- Click X to remove
- Multiple tools can be combined

## Tips

- Combine tools for precise results
- Use reference URLs for specific styles
- Set language before generating
