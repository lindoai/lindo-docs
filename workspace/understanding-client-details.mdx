---

When you select a client in CRM, you see detailed information across five tabs.

## Info Tab

Basic client information:
- Name
- Email
- Company
- Notes (internal)
- Website limit
- Agentic Mode toggle
- Delete/Update buttons

## Websites Tab

All websites owned by this client:
- Website list
- Status indicators
- Quick edit access

## Credits Tab

Credit management:
- Monthly allocation
- Current usage
- Reset date
- Allocation history
- Add/Update credits

## Billing Tab

Invoice management:
- Billing summary
- Create Invoice button
- Invoice history
- Payment status

## Subscriptions Tab

Recurring billing:
- Active subscriptions
- Create Subscription
- Cancel options

## Navigation

Click tabs to switch between views:
- Info → Websites → Credits → Billing → Subscriptions

## Tips

- Keep info updated
- Monitor credit usage
- Track billing status
- Review subscriptions
