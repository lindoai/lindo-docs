---
title: "How to Set Default Language"
description: "Configure the default interface language for your workspace"
---

Set the default language for your workspace interface, affecting how clients see the dashboard.

## What This Controls

The default language affects:

- Workspace interface text
- Client dashboard language
- Default for new users

## Setting Default Language

1. Open **Settings** in sidebar
2. Go to **Workspace** tab
3. Find **Default Language** dropdown
4. Select your preferred language
5. Click **Save**

## Available Languages

Choose from 11 supported languages:

- English
- Spanish
- French
- German
- Italian
- Portuguese
- Dutch
- Polish
- Swedish
- Norwegian
- Danish

## Who This Affects

- New clients see this language by default
- Existing users keep their preference
- You can still change your own language

## Tips

- Match your primary client base
- Consider your team's language
- Clients can change their own preference
- Test the interface in selected language
