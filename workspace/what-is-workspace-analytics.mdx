---
title: "What is Workspace Analytics"
description: "Aggregated data from all websites"
---

Workspace Analytics shows aggregated data across all websites in your workspace.

## Purpose

Get a bird's-eye view of:
- Total traffic across all sites
- Top performing websites
- Visitor demographics
- Traffic sources

## Accessing Analytics

Click **Analytics** in the sidebar to open the Analytics modal.

## Analytics Components

### Statistics Cards

Quick metrics at the top:
- **Visitors** - Unique visitors
- **Page Views** - Total page loads
- **Avg Response Time** - Server performance
- **Total Pages** - Pages across all sites
- **Total Blogs** - Blog posts across all sites

### Traffic Chart

Line graph showing:
- Page views over time
- Visitors over time
- Selectable date range

### Top Websites

Ranking of your most visited websites:
- Website name
- Visit count
- Percentage of total

### Operating Systems

Visitor OS breakdown:
- Windows
- macOS
- iOS
- Android
- Linux

### Browsers

Visitor browser breakdown:
- Chrome
- Safari
- Firefox
- Edge
- Others

### Devices

Device type breakdown:
- Desktop
- Mobile
- Tablet
- Bot

### Top Countries

Geographic distribution:
- Country name
- Visitor count
- Percentage

### Referrers

Traffic sources:
- Direct
- Google
- Social media
- Other websites

### UTM Campaigns

Marketing campaign tracking:
- Campaign name
- Source
- Medium
- Visits

## Workspace vs Website Analytics

| Workspace Analytics | Website Analytics |
|--------------------|-------------------|
| All websites combined | Single website |
| In workspace sidebar | In website editor |
| Aggregated metrics | Detailed page data |
| Top websites ranking | Top pages ranking |
