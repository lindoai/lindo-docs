Access comprehensive information about any client in your workspace.

## Viewing Client Details

1. Open the CRM from the sidebar
2. Find the client in the list (use search if needed)
3. Click on the client row
4. The client details panel opens

## Client Details Tabs

### Info Tab
- Client name, email, company
- Notes field for internal reference
- Website limit setting
- Monthly credit limit
- Agentic mode toggle

### Websites Tab
View all websites owned by this client. Click any website to open it in the editor.

### Credits Tab
- Current credit balance
- Monthly allocation
- Allocation history
- One-time credit additions

### Billing Tab
- Billing summary (total billed, outstanding, paid)
- Invoice history
- Create new invoices

### Subscriptions Tab
- Active subscriptions
- Subscription details
- Cancel subscriptions

## Actions Available

- Update client information
- Resend invitation email
- Allocate credits
- Create invoices
- Delete client
