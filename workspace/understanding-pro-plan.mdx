title: "Understanding Pro Plan"
description: "Features, limits, pricing for Pro"
---

The Pro plan is designed for individuals and small projects.

## Features

- 5 active websites
- 10 pages per website
- 10,000 monthly visits
- 500 AI credits
- Custom domain support
- Basic analytics
- Email support

## Limits

| Resource | Limit |
|----------|-------|
| Active Websites | 5 |
| Pages per Website | 10 |
| Monthly Visits | 10,000 |
| AI Credits | 500 |
| Team Members | 1 |
| Clients | - |

## What's Included

### Website Building

- AI website generation
- Template access
- All design styles
- Mobile responsive

### Domains

- Free subdomain
- Custom domain support
- SSL included

### Analytics

- Basic traffic stats
- Page views
- Visitor counts

## What's Not Included

- White label
- Client management (CRM)
- Team collaboration
- API access
- Priority support

## Best For

- Personal websites
- Small portfolios
- Side projects
- Testing the platform

## Upgrading

Need more? Consider:
- **Business** - For agencies
- **Elite** - For scale
- **Reseller** - For white label
