---
title: "What is Batch Import"
description: "Creating multiple workflows from CSV"
---

Batch import creates multiple websites at once from a CSV file.

## What It Does

- Upload a CSV file
- Each row becomes a workflow
- Multiple websites generated
- Background processing

## Use Cases

- Creating many client sites
- Bulk website generation
- Agency scaling
- Template-based creation

## CSV Format

Your CSV should include:
- Website name
- Prompt/description
- Optional: language, mode, colors

## How It Works

1. Prepare your CSV
2. Upload to Workflows
3. Workflows created for each row
4. Websites generated in background
5. Check progress in Workflows

## Tips

- Download sample CSV first
- Follow the format exactly
- Test with small batch first
- Monitor progress
