---
title: "How to Change Language"
description: "Selecting interface language from the 11 available options"
---

Change the Lindo.ai interface language to your preferred language.

## Available Languages

Lindo.ai supports 11 interface languages:

- English
- Spanish (Español)
- French (Français)
- German (Deutsch)
- Italian (Italiano)
- Portuguese (Português)
- Dutch (Nederlands)
- Polish (Polski)
- Turkish (Türkçe)
- Russian (Русский)
- Japanese (日本語)

<Steps>
  <Step title="Find the Language Selector">
    In the workspace header, look for the language icon (globe or flag).
  </Step>
  <Step title="Click the Language Icon">
    A dropdown menu appears with all available languages.
  </Step>
  <Step title="Select Your Language">
    Click your preferred language.
  </Step>
  <Step title="Interface Updates">
    The entire interface immediately switches to the selected language.
  </Step>
</Steps>
## What Changes

When you change language:
- All menu items
- Button labels
- Form labels
- Help text
- Error messages

## What Doesn't Change

- Your website content (that's separate)
- Client-facing content
- Custom text you've entered

## Language vs Content Language

| Interface Language | Content Language |
|-------------------|------------------|
| Lindo.ai menus | Your website text |
| Changed in header | Set in AI tools |
| Affects dashboard | Affects generated content |

## Tips

- Interface language is per-user
- Each team member can use their preferred language
- Content language is set per-website
