The Websites Grid displays all websites in your workspace as clickable cards.

## Location

Below the AI Prompt Builder in your workspace dashboard.

## Grid Components

### Search Input

Filter websites by name:
- Type to search
- Results update instantly
- Searches website names

### Website Cards

Each website appears as a card showing:
- **Thumbnail** - Screenshot preview
- **Name** - Website name
- **Status Badge** - Published or Draft

### Load More

If you have many websites, scroll or click to load more.

## Card Layout

Cards are arranged in a responsive grid:
- Desktop: 3-4 cards per row
- Tablet: 2-3 cards per row
- Mobile: 1-2 cards per row

## Card Interactions

### Click

Opens the website in the editor.

### Hover

Reveals the actions menu with options:
- Edit
- Rename
- Clone
- Export
- Delete
- Save as Template
- Upgrade/Downgrade

## Status Indicators

### Published (Green)

Website is live and accessible to visitors.

### Draft (Gray)

Website exists but isn't published yet.

## Empty State

If you have no websites:
- Grid shows a message
- Prompts you to create your first website

## Sorting

Websites are sorted by:
- Most recently edited first
- Newest at the top
