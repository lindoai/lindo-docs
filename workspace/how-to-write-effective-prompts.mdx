Write better prompts to get higher quality AI-generated websites.

## Prompt Structure

A good prompt includes:

1. **What**: Type of website
2. **Who**: Business/person it's for
3. **Sections**: What to include
4. **Style**: Design preferences

## Example Prompts

### Basic Prompt

```
Create a website for a coffee shop
```

### Better Prompt

```
Create a modern coffee shop website for "Bean & Brew" with:
- Hero section with shop interior photo
- Menu with categories: coffee, tea, pastries
- About section with our story
- Location with hours and map
- Contact form
- Warm, cozy color scheme
```

## Tips for Better Results

### Be Specific

- Name the business
- List specific sections
- Describe desired style

### Include Details

- Mention colors
- Specify content types
- Note special features

### Use Examples

- Reference design styles
- Mention similar sites
- Describe the vibe

## Common Mistakes

- Too vague
- Missing key sections
- No style guidance
- Conflicting requirements

## Tips

- Start detailed, refine later
- Use sample prompts as guides
- Iterate and improve
- Combine with AI tools
