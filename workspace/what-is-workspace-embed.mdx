title: "What is Workspace Embed"
description: "Embedding workspace in external websites"
---

Workspace Embed lets you embed your Lindo.ai workspace in other websites.

## What It Does

Embedding allows:
- Workspace inside your website
- Seamless user experience
- No separate login page
- Integrated platform

## How It Works

1. Enable embed feature
2. Get embed code
3. Add to your website
4. Users access workspace inline

## Use Cases

### SaaS Integration

Embed in your SaaS dashboard:
- Users stay in your app
- Seamless experience
- No context switching

### Client Portals

Embed in client portal:
- Branded experience
- Single login
- Integrated workflow

### Website Builders

Offer website building:
- Part of your platform
- Value-add feature
- Differentiation

## Embed Code

Basic iframe embed:
```html
<iframe
  src="https://app.yourdomain.com/embed"
  width="100%"
  height="800px"
  frameborder="0">
</iframe>
```

## Requirements

- Reseller plan
- White label configured
- Custom domain (recommended)

## Tips

- Match styling
- Set appropriate height
- Handle authentication
- Test thoroughly
