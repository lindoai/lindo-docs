title: "How to Enable Agentic Mode for All Clients"
description: "Migrate all existing clients to the AI-powered interface"
---

Enable Agentic Mode for all clients at once to migrate everyone to the AI-powered experience.

## What This Does

Enabling for all clients:

- Switches all existing clients to Agentic Mode
- New clients also get Agentic Mode
- Provides consistent experience

## Enabling for All

1. Open **Settings** in sidebar
2. Go to **Workspace** tab
3. Find **Agentic Mode** section
4. Click **Enable for All Clients**
5. Confirm the action

## Considerations

Before enabling for all:

- Some clients may need training
- Interface changes significantly
- Consider gradual rollout
- Prepare support resources

## What Clients Experience

After migration:

- New AI prompt builder
- Chat-based editing
- AI agents available
- Modern interface design

## Tips

- Communicate changes in advance
- Provide documentation or guides
- Offer support during transition
- Monitor for issues
