description: "Setting up recurring billing"
---

Set up automatic recurring billing for a client.

<Steps>
  <Step title="Open Client Billing">
    Go to CRM → Select client → Billing tab.
  </Step>
  <Step title="Click Create Subscription">
    Or create invoice with recurring frequency.
  </Step>
  <Step title="Configure Subscription">
    Set the details:
    - Service name
    - Price
    - Currency
    - Frequency (monthly/yearly/weekly)
  </Step>
  <Step title="Send">
    Client receives first invoice.
  </Step>
  <Step title="Automatic Billing">
    Future invoices sent automatically.
  </Step>
</Steps>
## Subscription Flow

1. Client pays first invoice
2. Subscription activates
3. Auto-bills on schedule
4. Continues until cancelled

## Tips

- Clear service description
- Appropriate frequency
- Communicate terms
- Monitor for failures
