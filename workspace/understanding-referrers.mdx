See where your traffic comes from.

## What It Shows

Sources that send visitors:
- Search engines
- Social media
- Other websites
- Direct traffic

## Common Referrers

| Source | Type |
|--------|------|
| google.com | Search |
| facebook.com | Social |
| twitter.com | Social |
| linkedin.com | Social |
| (direct) | Direct |

## Reading the Data

Each entry shows:
- Referrer URL/domain
- Visitor count
- Percentage

## Uses

- Understand traffic sources
- Marketing effectiveness
- Partnership opportunities
- SEO performance

## Tips

- Invest in top sources
- Diversify traffic
- Track campaigns
- Build referral relationships
