title: "Understanding Toast Notifications"
description: "Success and error messages in Lindo.ai"
---

Toast notifications are brief messages that appear to confirm actions or report errors.

## Types of Toasts

### Success Toasts

Green notifications indicating:

- Action completed successfully
- Changes saved
- Operation finished

### Error Toasts

Red notifications indicating:

- Something went wrong
- Action failed
- Requires attention

### Info Toasts

Blue notifications providing:

- Helpful information
- Tips or guidance
- Status updates

## Toast Behavior

Toasts:

- Appear briefly (few seconds)
- Auto-dismiss
- Can be clicked to dismiss
- Stack if multiple

## Common Messages

| Message | Meaning |
|---------|---------|
| "Saved successfully" | Changes saved |
| "Website created" | Generation complete |
| "Error occurred" | Something failed |
| "Copied to clipboard" | Copy action worked |

## Tips

- Read error messages carefully
- Note any error codes
- Retry if action failed
- Contact support for persistent errors
