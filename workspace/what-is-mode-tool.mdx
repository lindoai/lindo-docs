---
title: "What is the Mode Tool"
description: "Dark/Light theme selection for generated websites"
---

The Mode Tool sets the color theme (dark or light) for your generated website.

## Options

### Light Mode

- White/light backgrounds
- Dark text
- Clean, bright appearance
- Best for: Professional services, healthcare, education

### Dark Mode

- Dark/black backgrounds
- Light text
- Modern, sleek appearance
- Best for: Tech, gaming, creative, nightlife

## How to Use

1. Click **(+)** in the AI Prompt Builder toolbar
2. Select **Mode**
3. Choose **Light** or **Dark**
4. Generate your website

## What It Affects

The mode setting influences:
- Background colors (page, sections, cards)
- Text colors (headings, body, labels)
- Border colors
- Shadow styles
- Overall contrast

## Default Behavior

If you don't set a mode:
- AI chooses based on your prompt
- Tech/modern prompts often get dark mode
- Traditional business prompts often get light mode

## Changing Mode After Generation

To switch modes on an existing website:

```
Change the entire website to dark mode
```

Or in Theme settings:
1. Open website in editor
2. Go to Settings → Theme
3. Adjust background and text colors

## Both Modes Supported

All generated websites include both light and dark mode CSS:
- Tailwind's `dark:` classes are used
- Users can toggle if you add a theme switcher
- Responsive to system preferences (if configured)

## Tips

- **Match your brand** - Use the mode that fits your brand identity
- **Consider audience** - Tech-savvy users often prefer dark mode
- **Test readability** - Ensure text is readable in your chosen mode
