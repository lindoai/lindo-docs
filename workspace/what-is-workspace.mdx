The workspace is your central dashboard in Lindo.ai where you create websites, manage clients, and configure settings. Everything starts here.

## What You Can Do

- **Create websites** — Use the AI Prompt Builder to generate sites from text descriptions
- **Manage websites** — View, edit, clone, export, and delete your sites
- **Handle clients** — Add clients, allocate credits, send invoices (Business+ plans)
- **View analytics** — See aggregated traffic data across all websites
- **Configure settings** — Set up white label branding, team members, integrations

## Layout

**Header** — Contains your logo (or Lindo.ai default), workspace name, language selector, and user menu with account options.

**Sidebar** — Navigation to CRM, Workflows, Analytics, and Settings. What you see depends on your plan and role.

**Main Area** — The AI Prompt Builder for creating websites, your websites grid below, and a templates panel on the right.

## Workspace vs Website Editor

| Workspace | Website Editor |
|-----------|----------------|
| Create new websites | Edit existing websites |
| Manage all sites | Focus on one site |
| Configure workspace settings | Configure site settings |
| View aggregated analytics | View site-specific analytics |
| Manage agency clients | Manage site customers |

## Who Sees What

| Role | Access |
|------|--------|
| Owner | Full access to everything |
| Admin | Everything except billing and plan changes |
| Editor | Create and edit websites, no settings access |
| Viewer | Read-only access to websites |
| Client | Only their assigned websites and limited features |
