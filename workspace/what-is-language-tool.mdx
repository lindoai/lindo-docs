---
title: "What is the Language Tool"
description: "Setting content language and localization preferences"
---

The Language Tool sets the language for AI-generated website content.

## Purpose

When you create a website, the AI generates all text content:
- Headlines and copy
- Button text
- Navigation labels
- Meta descriptions

The Language Tool tells the AI which language to use.

## Default Behavior

By default, **Auto-detect** is enabled:
- AI detects the language of your prompt
- Generates content in that same language
- Write in Spanish → Get Spanish content

## Manual Selection

To force a specific language:
1. Click **(+)** in the toolbar
2. Select **Language**
3. Choose from the dropdown
4. Or type a language name

## Supported Languages

The AI can generate content in virtually any language, including:
- English, Spanish, French, German
- Italian, Portuguese, Dutch, Polish
- Turkish, Arabic, Japanese
- Chinese, Korean, Russian
- And many more

## Use Cases

### Prompt in English, Content in Spanish

1. Add Language tool
2. Select "Spanish"
3. Write your prompt in English
4. AI generates Spanish content

### Multi-Language Websites

Create versions in different languages:
1. Generate English version
2. Duplicate the website
3. Use AI to translate: "Translate all content to French"

## Language vs Interface Language

| Setting | What It Affects |
|---------|-----------------|
| Language Tool | Generated website content |
| Interface Language | Dashboard buttons and labels |

These are independent settings.

## Tips

- **Be consistent** - Use the same language throughout a site
- **Review translations** - AI translations are good but not perfect
- **Cultural context** - AI adapts content for the target language
