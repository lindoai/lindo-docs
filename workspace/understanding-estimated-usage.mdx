description: "Monthly consumption tracking"

Track your credit consumption throughout the month.

## What It Shows

Estimated usage displays:
- Credits used this period
- Percentage of allocation
- Usage trend
- Projected usage

## Viewing Usage

1. Go to Settings → Credits
2. See usage breakdown
3. View progress bar

## Usage Breakdown

| Metric | Description |
|--------|-------------|
| Used | Credits consumed |
| Remaining | Credits available |
| Total | Monthly allocation |
| Reset Date | When credits refresh |

## Usage Patterns

### Low Usage

- Plenty of credits remaining
- No action needed

### Moderate Usage

- On track for the month
- Monitor as needed

### High Usage

- May run out before reset
- Consider purchasing more

## Tips

- Check usage weekly
- Plan large projects
- Purchase extra if needed
- Upgrade for more credits
