title: "Understanding Website Cards"
description: "Thumbnail, name, status badge components"
---

Website cards display your websites in the workspace grid.

## Card Components

### Thumbnail

- Preview image of your website
- Auto-generated screenshot
- Updates when you publish

### Website Name

- Display name you set
- Editable via actions menu
- Shown below thumbnail

### Status Badge

Indicates website state:
- **Published** - Live and accessible
- **Draft** - Not yet published

### Actions Menu

Appears on hover:
- Edit
- Rename
- Clone
- Export
- Delete
- More options

## Card States

### Normal State

- Thumbnail visible
- Name displayed
- Status badge shown

### Hover State

- Actions menu appears
- Quick access to options
- Click to open editor

### Loading State

- Skeleton animation
- While thumbnail loads
- Brief transition

## Interacting with Cards

### Click Card

Opens the website in the editor.

### Hover Card

Shows the actions menu.

### Right-Click Card

May show context menu (browser dependent).

## Card Grid

Cards are displayed in a responsive grid:
- Multiple columns on desktop
- Fewer columns on tablet
- Single column on mobile

## Tips

- Hover to see actions
- Click to edit
- Use search to filter
- Status shows publish state
