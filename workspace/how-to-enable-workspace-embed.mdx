---
title: "How to Enable Workspace Embed"
description: "Turning on embed functionality"
---

Enable embedding your workspace in external websites.

## Prerequisites

- Reseller plan
- White label configured

<Steps>
  <Step title="Open Reseller Settings">
    Go to Settings → Reseller tab.
  </Step>
  <Step title="Find Embed Section">
    Locate the workspace embed area.
  </Step>
  <Step title="Enable Embed">
    Turn on the embed feature.
  </Step>
  <Step title="Get Embed Code">
    Copy the provided embed code.
  </Step>
  <Step title="Save">
    Save your settings.
  </Step>
</Steps>
## Embed Code

After enabling, you get code like:
```html
<iframe
  src="https://app.yourdomain.com/embed"
  width="100%"
  height="800px"
  frameborder="0">
</iframe>
```

## Configuration Options

- Embed URL
- Allowed domains
- Authentication settings

## Tips

- Test in staging first
- Set appropriate dimensions
- Handle authentication
- Match your site's style
