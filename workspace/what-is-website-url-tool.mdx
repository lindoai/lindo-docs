---
title: "What is the Website URL Tool"
description: "Using existing websites as design references"
---

The Website URL Tool lets you reference an existing website as design inspiration for your AI-generated site.

## Purpose

Instead of describing a design style in words, show the AI an example:
- "Make it look like this site"
- AI analyzes the reference
- Generates similar design patterns

## How to Use

1. Click **(+)** in the AI Prompt Builder toolbar
2. Select **Website URL**
3. Enter the full URL (e.g., `https://example.com`)
4. Write your prompt describing your business
5. Generate

## What the AI Analyzes

From the reference URL, the AI considers:
- Layout structure
- Color scheme
- Typography style
- Section arrangement
- Visual density
- Design patterns

## What It Doesn't Copy

The AI creates original content:
- Your business name and description
- Unique copy and headlines
- Different images
- Your specified colors (if set)

## Example Usage

**Prompt:**
```
Website for my photography studio, similar style to the reference.
Include portfolio gallery, about section, and contact.
```

**With URL Tool:**
```
https://awwwards.com/sites/some-beautiful-site
```

**Result:**
A photography website with similar design aesthetics but original content.

## Best Practices

### Good Reference Sites

- Clean, modern designs
- Clear section structure
- Professional layouts
- Publicly accessible URLs

### Less Effective References

- Sites behind login walls
- Highly complex/animated sites
- Sites with unusual frameworks
- Very minimal single-element pages

## Combining with Other Tools

Use Website URL with:
- **Color Palette** - Override reference colors with your brand
- **Mode** - Force dark/light regardless of reference
- **Language** - Generate in your preferred language

## Tips

- **Multiple references** - Mention specific elements: "Hero like site A, features like site B"
- **Be specific** - "I like the typography and spacing of this site"
- **Iterate** - Use AI chat to refine after initial generation
