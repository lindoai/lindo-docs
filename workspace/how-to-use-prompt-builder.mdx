---
title: "How to Use the Prompt Builder"
description: "Building prompts with sample prompts, sections, and styles"
---

Use the Prompt Builder modal to construct detailed prompts quickly.

<Steps>
  <Step title="Open the Prompt Builder">
    Click **✨ Prompt Builder** below the main prompt input.
  </Step>
  <Step title="Browse Sample Prompts">
    Scroll through pre-written prompts:
    - Agency
    - SaaS Product
    - Restaurant
    - Portfolio
    - E-commerce
    - And more...

    Click any sample to select it.
  </Step>
  <Step title="Select Sections">
    Check the sections you want included:
    - ☑️ Hero
    - ☑️ Features
    - ☑️ About
    - ☑️ Testimonials
    - ☑️ Pricing
    - ☑️ FAQ
    - ☑️ Contact
  </Step>
  <Step title="Choose Design Style">
    Select a visual direction:
    - Modern
    - Minimal
    - Brutalist
    - Glass
    - Playful
    - Elegant
  </Step>
  <Step title="Apply to Prompt">
    Click **Apply to Prompt**. Your selections are combined into a prompt in the main input.
  </Step>
  <Step title="Customize">
    Edit the generated prompt:
    - Replace placeholder business name
    - Add specific details
    - Adjust sections as needed
  </Step>
  <Step title="Generate">
    Click the submit button to create your website.
  </Step>
</Steps>
## Example

**Selections:**
- Sample: SaaS Product
- Sections: Hero, Features, Pricing, FAQ
- Style: Modern

**Generated Prompt:**
```
Modern SaaS landing page with hero section showcasing the product,
features grid highlighting key benefits, pricing table with multiple
tiers, and FAQ section addressing common questions.
```

**Your Customization:**
```
Modern SaaS landing page for TaskFlow, a project management tool.
Hero section with headline "Manage Projects Smarter" and product
screenshot. Features grid with 6 key benefits. Pricing table with
Free, Pro, and Enterprise tiers. FAQ section with 5 common questions.
Dark mode with blue accent.
```

## Tips

- **Start with a sample** - Faster than writing from scratch
- **Select relevant sections** - Don't include everything
- **Always customize** - Add your specific details
- **Combine with tools** - Add Language, Mode, Color Palette
