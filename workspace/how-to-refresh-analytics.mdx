---
title: "How to Refresh Analytics"
description: "Update workspace analytics with latest data"
---

Refresh your workspace analytics to see the most recent traffic data.

## Refreshing Analytics

1. Open **Analytics** from sidebar
2. Click the **Refresh** button
3. Data updates with latest information

## When to Refresh

Refresh analytics when:

- Checking real-time traffic
- After publishing changes
- Monitoring campaigns
- Verifying tracking

## What Updates

Refreshing updates:

- Visitor counts
- Page views
- Traffic sources
- All metrics

## Tips

- Allow time for data processing
- Don't refresh excessively
- Check after major changes
- Use date range for historical data
