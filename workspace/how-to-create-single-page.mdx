title: "How to Create a Single-Page Website"
description: "Generating one-page websites"
---

Build a complete website on a single scrolling page.

## What is Single-Page

A single-page website has:
- All content on one page
- Smooth scroll navigation
- No page transitions
- Anchor links to sections

## When to Use

Single-page works best for:
- Landing pages
- Product launches
- Event pages
- Portfolios
- Coming soon pages

<Steps>
  <Step title="Write Your Prompt">
    Describe your website and mention "single page" or "landing page":

    > "Create a single-page landing page for my fitness app. Include hero, features, testimonials, pricing, and CTA sections."
  </Step>
  <Step title="Add Tools (Optional)">
    - Set language
    - Choose dark/light mode
    - Define colors
    - Add reference URL
  </Step>
  <Step title="Generate">
    Click send to create your website.
  </Step>
  <Step title="Review">
    Your single-page site is ready with all sections.
  </Step>
</Steps>
## Typical Sections

A good single-page site includes:
1. Hero (top banner)
2. Features/Benefits
3. About/Story
4. Testimonials
5. Pricing (if applicable)
6. CTA (call to action)
7. Footer

## Navigation

Single-page sites use anchor navigation:
- Menu links scroll to sections
- Smooth scrolling effect
- No page reloads

## Tips

- Keep it focused
- 5-7 sections is ideal
- Strong hero section
- Clear CTA at the end

## Single vs Multi-Page

| Single-Page | Multi-Page |
|-------------|------------|
| One scrolling page | Multiple pages |
| Anchor navigation | Page navigation |
| Focused message | More content |
| Faster to build | More comprehensive |
