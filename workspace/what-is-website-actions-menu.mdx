The actions menu provides quick access to website management options.

## Accessing the Menu

1. Hover over a website card
2. Actions menu appears
3. Click desired action

## Available Actions

### Edit

Opens the website in the editor.

### Rename

Change the website's display name:
- Click Rename
- Enter new name
- Save

### Clone

Create a copy of the website:
- Duplicates all pages
- Duplicates settings
- New website created

### Export

Download the website as a ZIP file:
- HTML/CSS files
- Images
- Ready for self-hosting

### Upgrade

Upgrade website to premium features:
- Custom domain
- Remove branding
- Advanced features

### Downgrade

Revert to free tier:
- Removes premium features
- Keeps content

### Save as Template

Create a reusable template:
- Save current design
- Use for future projects
- Available in templates panel

### Delete

Permanently remove the website:
- Cannot be undone
- All pages deleted
- Confirmation required

## Action Availability

Some actions depend on your plan:

| Action | Free | Pro | Business+ |
|--------|------|-----|-----------|
| Edit | ✓ | ✓ | ✓ |
| Rename | ✓ | ✓ | ✓ |
| Clone | ✓ | ✓ | ✓ |
| Export | - | ✓ | ✓ |
| Save as Template | - | - | ✓ |

## Tips

- Hover to reveal actions
- Clone before major changes
- Export for backups
- Delete with caution
