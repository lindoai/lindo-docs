Add Google Tag Manager to your workspace.

## What is GTM

Google Tag Manager lets you:
- Manage tracking codes
- Add marketing tags
- No code changes needed
- Centralized tag management

## Prerequisites

- Google Tag Manager account
- GTM container ID (GTM-XXXXXX)

<Steps>
  <Step title="Get Your Container ID">
    1. Log into tagmanager.google.com
    2. Select your container
    3. Copy the ID (format: GTM-XXXXXX)
  </Step>
  <Step title="Open Integrations">
    Go to Settings → White Label → Integrations tab.
  </Step>
  <Step title="Find GTM">
    Locate Google Tag Manager card.
  </Step>
  <Step title="Enter Container ID">
    Paste your GTM container ID.
  </Step>
  <Step title="Save">
    Click Save or Connect.
  </Step>
</Steps>
## Verification

After connecting:
- GTM loads on workspace pages
- Tags fire as configured
- Check GTM preview mode

## What Gets Tagged

GTM applies to:
- Workspace login page
- Client dashboard
- Workspace pages

Does NOT apply to:
- Individual websites (use website settings)

## Tips

- Test in GTM preview mode
- Start with basic tags
- Monitor for issues
- Use GTM for all tracking
