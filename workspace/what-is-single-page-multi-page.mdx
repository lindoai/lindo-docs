---
title: "What is Single-Page vs Multi-Page"
description: "Website structure options explained"
---

When creating a website, you can choose between single-page and multi-page structures.

## Single-Page Website

A single-page website:

- Has one long scrolling page
- All content on one URL
- Sections linked with anchors
- Good for simple sites

### Best For

- Landing pages
- Coming soon pages
- Simple portfolios
- Event pages

## Multi-Page Website

A multi-page website:

- Has multiple separate pages
- Each page has its own URL
- Navigation between pages
- Good for complex sites

### Best For

- Business websites
- E-commerce stores
- Blogs
- Large portfolios

## Choosing the Right Type

Consider:

- Amount of content
- User navigation needs
- SEO requirements
- Site complexity

## Tips

- Start simple, expand later
- Single-page is faster to create
- Multi-page better for SEO
- Can convert between types
