---
title: "How to Refresh Workflows"
description: "Update the workflow list with latest status"
---

Refresh the workflows list to see the current status of your running workflows.

## Refreshing Workflows

1. Open **Workflows** from sidebar
2. Click the **Refresh** button
3. List updates with latest status

## When to Refresh

Refresh workflows when:

- Checking on running workflows
- Waiting for completion
- Monitoring batch progress
- Verifying status changes

## Status Updates

After refreshing, you'll see:

- Current workflow status
- Progress counts
- Completion status
- Any errors

## Tips

- Don't refresh excessively
- Allow time for processing
- Check periodically for long batches
- Enable email notifications instead
