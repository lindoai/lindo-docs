Add custom code to the head section of workspace pages.

## What is Header Code

Code injected before `</head>`:
- Meta tags
- CSS stylesheets
- Analytics scripts
- Font imports

<Steps>
  <Step title="Open Custom Code">
    Go to Settings → White Label → Custom Code tab.
  </Step>
  <Step title="Find Header Section">
    Locate the "Header Code" textarea.
  </Step>
  <Step title="Add Your Code">
    Paste your code:

    ```html
    <!-- Example: Custom font -->
    <link href="https://fonts.googleapis.com/css2?family=Inter&display=swap" rel="stylesheet">

    <!-- Example: Meta tag -->
    <meta name="author" content="Your Agency">
    ```
  </Step>
  <Step title="Save">
    Click Save or Update.
  </Step>
</Steps>
## Common Uses

### Custom Fonts

```html
<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">
```

### Custom CSS

```html
<style>
  .custom-class {
    color: #333;
  }
</style>
```

### Meta Tags

```html
<meta name="robots" content="noindex">
```

## Tips

- Test code before adding
- Keep code minimal
- Use for CSS and meta tags
- JavaScript goes in body
