description: "Programmatic access to your workspace"
---

API keys provide programmatic access to your Lindo.ai workspace.

## What They Do

API keys allow:
- Automated operations
- Integration with other tools
- Custom workflows
- Programmatic control

## Use Cases

### Automation

- Create websites automatically
- Update content programmatically
- Batch operations

### Integration

- Connect to other platforms
- Sync data
- Trigger workflows

### Custom Tools

- Build custom dashboards
- Create internal tools
- Extend functionality

## Key Properties

- Unique identifier
- Bearer token format
- Workspace-specific
- Revocable

## Security

API keys are sensitive:
- Only shown once when created
- Store securely
- Don't share publicly
- Rotate regularly

## Availability

API access requires:
- Business plan or higher
- API enabled for workspace

## Tips

- Create separate keys for different uses
- Label keys descriptively
- Revoke unused keys
- Monitor usage
