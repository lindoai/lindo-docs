---
title: "How to Set a Color Palette"
description: "Selecting colors for your website design"
---

Define custom colors for your AI-generated website.

<Steps>
  <Step title="Open Add Tools Menu">
    Click the (+) button in the toolbar.
  </Step>
  <Step title="Select Color Palette Tool">
    Click "Color Palette" from the menu.
  </Step>
  <Step title="Choose Colors">
    Set your color scheme:
    - **Primary** - Main brand color (buttons, links)
    - **Secondary** - Accent color
    - **Background** - Page background
    - **Text** - Default text color
  </Step>
  <Step title="Use Color Picker">
    Click each color field to open the picker:
    - Select from palette
    - Enter hex code
    - Use eyedropper
  </Step>
  <Step title="Palette Tag Added">
    The colors appear as a tag in your prompt.
  </Step>
  <Step title="Generate">
    Your website uses the selected colors.
  </Step>
</Steps>
## Color Roles

| Color | Used For |
|-------|----------|
| Primary | Buttons, links, highlights |
| Secondary | Accents, hover states |
| Background | Page background |
| Text | Body text, headings |

## Tips

- Use brand colors for consistency
- Ensure contrast for readability
- Test dark text on light backgrounds
- Limit to 2-3 main colors

## Color Combinations

**Professional:**
- Primary: Navy blue
- Secondary: Gold
- Background: White

**Modern Tech:**
- Primary: Purple
- Secondary: Cyan
- Background: Dark gray

**Natural:**
- Primary: Forest green
- Secondary: Tan
- Background: Cream

## Changing Colors Later

Edit colors after generation:
1. Open website in editor
2. Go to Settings → Theme
3. Adjust color values
