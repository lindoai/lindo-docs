Access the complete API documentation to learn about available endpoints and integration options.

## Accessing API Docs

1. Open **Settings** in sidebar
2. Go to **API** tab
3. Click **API Docs** link
4. Documentation opens in new tab

## What's in the Docs

The API documentation includes:

- Authentication guide
- Available endpoints
- Request/response formats
- Code examples
- Error handling

## API Overview

The Lindo.ai API allows you to:

- List and manage websites
- Access client data
- Trigger workflows
- Retrieve analytics

## Getting Started

1. Generate an API key
2. Read the authentication section
3. Try a simple request
4. Build your integration

## Support

If you need help with the API:

- Check the documentation first
- Review code examples
- Contact support for complex issues
