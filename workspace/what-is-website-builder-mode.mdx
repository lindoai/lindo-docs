title: "Understanding Website Builder Mode"
description: "Full website generation mode explained"
---

Website Builder Mode generates complete websites from a single prompt.

## What It Does

When you enter a prompt in the main input area, the AI:
1. Analyzes your description
2. Plans the website structure
3. Generates all pages
4. Creates content and images
5. Applies styling and layout

## How It Works

### Single-Page Mode

Creates a one-page website with:
- Hero section
- Features
- About
- Testimonials
- CTA
- Footer

### Multi-Page Mode

Creates a complete website with:
- Home page
- About page
- Services/Features page
- Contact page
- Navigation between pages

## Using Website Builder Mode

1. Enter your prompt in the main input
2. Add optional tools (language, colors, mode)
3. Click the send button
4. Wait for generation (30-60 seconds)
5. Review your new website

## Credit Cost

Website generation costs **0 credits** for the initial generation.

Subsequent edits use different agents with their own costs.

## Best Prompts

Good prompts include:
- Business type
- Target audience
- Key features
- Desired style

**Example:**
> "Create a modern SaaS landing page for a project management tool. Include hero, features, pricing, testimonials, and CTA sections. Target audience is small business owners."

## Tips

- Be specific about your business
- Mention desired sections
- Include style preferences
- Specify single or multi-page
