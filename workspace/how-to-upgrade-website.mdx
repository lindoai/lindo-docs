---
title: "How to Upgrade a Website"
description: "Upgrading to premium features"
---

Unlock premium features for individual websites.

## Premium Features

Upgrading a website enables:
- Custom domain connection
- Remove Lindo.ai branding
- Priority support
- Advanced analytics

<Steps>
  <Step title="Find Your Website">
    Locate the website in your workspace grid.
  </Step>
  <Step title="Open Actions Menu">
    Hover over the website card.
  </Step>
  <Step title="Click Upgrade">
    Select "Upgrade" from the actions menu.
  </Step>
  <Step title="Review Features">
    See what's included in the upgrade.
  </Step>
  <Step title="Confirm">
    Complete the upgrade process.
  </Step>
</Steps>
## Upgrade vs Plan Upgrade

| Website Upgrade | Plan Upgrade |
|-----------------|--------------|
| Single website | All websites |
| Specific features | More limits |
| One-time or monthly | Monthly subscription |

## When to Upgrade

Upgrade a website when you need:
- Professional custom domain
- Client-facing site without branding
- Specific premium features

## Cost

Website upgrades are billed:
- Per website
- Monthly or one-time
- Based on features selected

## Tips

- Upgrade client-facing sites
- Keep personal projects on free
- Consider plan upgrade for multiple sites
