---
title: "How to Clone a Website"
description: "Duplicating a website"
---

Create a copy of an existing website.

<Steps>
  <Step title="Find the Website">
    Locate the website you want to clone in the Websites Grid.
  </Step>
  <Step title="Open Actions Menu">
    Hover over the website card to reveal the actions menu.
  </Step>
  <Step title="Click Clone">
    Click **Clone** from the actions menu.
  </Step>
  <Step title="Wait for Copy">
    The system creates a duplicate. This takes a few seconds.
  </Step>
  <Step title="Find the Clone">
    The cloned website appears in your grid with a modified name (e.g., "Original Name (Copy)").
  </Step>
</Steps>
## What Gets Cloned

- All pages
- All content and sections
- Theme settings
- SEO settings
- Products (if any)

## What Doesn't Get Cloned

- Custom domain settings
- Analytics data
- Customer/CRM data
- Published status (clone starts as draft)

## Use Cases

### Create Variations

Clone to test different designs:
1. Clone the original
2. Make changes to the clone
3. Compare results
4. Keep the better version

### Multi-Language Sites

Clone for translations:
1. Clone the English site
2. Use AI to translate: "Translate all content to Spanish"
3. Publish on different domain

### Client Templates

Clone your best work:
1. Create a great design
2. Clone for new clients
3. Customize for each client

### Backup Before Changes

Clone before major edits:
1. Clone the current version
2. Make risky changes to original
3. Revert to clone if needed

## Tips

- **Rename immediately** - Avoid confusion with original
- **Update content** - Don't leave duplicate content live
- **Check settings** - Review domain and SEO settings
