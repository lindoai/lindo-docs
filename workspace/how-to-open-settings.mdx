description: "Accessing the settings modal"
---

Access workspace configuration and settings.

<Steps>
  <Step title="Find Settings in Sidebar">
    Look for the **Settings** icon (gear) in the left sidebar.
  </Step>
  <Step title="Click Settings">
    Click to open the Settings modal.
  </Step>
  <Step title="Navigate Tabs">
    The settings modal has multiple tabs:
    - **White Label** - Branding and domain
    - **Members** - Team management
    - **Reseller** - Reseller features
    - **API** - API keys and webhooks
    - **Plans** - Subscription management
    - **Credits** - AI credit balance
  </Step>
</Steps>
## Settings Tabs Overview

### White Label

Configure branding:
- Workspace name
- Logo and favicon
- Custom domain
- Integrations
- Custom code

### Members

Manage team:
- Invite members
- Set roles
- Revoke access

### Reseller

Reseller features:
- Reseller website
- Workspace embed

### API

Developer access:
- Generate API keys
- Configure webhooks
- View documentation

### Plans

Subscription:
- Current plan
- Plan comparison
- Upgrade/downgrade

### Credits

AI credits:
- Current balance
- Usage history
- Purchase credits

## Keyboard Shortcuts

Quick tab switching:
- `Cmd/Ctrl + 1` - Tab 1
- `Cmd/Ctrl + 2` - Tab 2
- `Cmd/Ctrl + 3` - Tab 3
- `Cmd/Ctrl + 4` - Tab 4
- `Cmd/Ctrl + 5` - Tab 5
- `Cmd/Ctrl + 6` - Tab 6

## Closing Settings

- Click outside the modal
- Press `Escape`
- Click the X button
