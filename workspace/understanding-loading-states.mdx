Lindo.ai uses various loading states to indicate when content is being fetched or processed.

## Types of Loading States

### Skeleton Loaders

Gray placeholder shapes that show:

- Where content will appear
- Approximate layout
- Loading in progress

### Loading Spinners

Circular animations indicating:

- Action in progress
- Please wait
- Processing request

### Progress Indicators

Show completion status:

- Workflow progress
- Upload progress
- Generation progress

## Where You'll See Them

Loading states appear when:

- Opening modals
- Loading websites
- Fetching analytics
- Processing AI requests
- Uploading files

## Tips

- Wait for loading to complete
- Don't refresh during loading
- Check internet connection if stuck
- Contact support for persistent issues
