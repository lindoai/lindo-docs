---
title: "How to Use a Reference Website"
description: "Use an existing website as design inspiration"
---

Provide a reference website URL to guide the AI in generating designs similar to sites you admire.

## What is Reference Website?

The reference website tool lets you:

- Share a URL of a site you like
- AI analyzes the design style
- Generated website takes inspiration from it

## Using the Tool

1. Click the **(+)** button in the prompt builder
2. Select **Website URL** tool
3. Enter the reference website URL
4. Write your prompt
5. Generate your website

## What the AI Considers

From the reference site, the AI may consider:

- Layout structure
- Color scheme
- Typography style
- Section arrangement
- Overall aesthetic

## Tips for Best Results

- Choose sites with clear design
- Combine with specific instructions
- Reference isn't exact copying
- Add your own requirements

## Example

```
Reference: https://example.com

Create a SaaS landing page for "CloudSync" with similar
modern aesthetic, featuring hero, features, pricing,
and testimonials sections.
```

## Limitations

- AI interprets, doesn't copy exactly
- Some elements may not transfer
- Your content will be unique
- Results vary based on prompt
