Configure subdomains for your white label setup.

## Subdomain Types

You'll configure:

- **WebApp**: Where clients access dashboard (e.g., app.yourdomain.com)
- **Website**: Where websites are hosted (e.g., sites.yourdomain.com)
- **Email**: For email sending (e.g., mail.yourdomain.com)

## Configuring Subdomains

1. After entering main domain
2. Set webapp subdomain (e.g., "app")
3. Set website subdomain (e.g., "sites")
4. Set email subdomain (e.g., "mail")
5. Continue to DNS setup

## Subdomain Examples

| Type | Subdomain | Full URL |
|------|-----------|----------|
| WebApp | app | app.yourdomain.com |
| Website | sites | sites.yourdomain.com |
| Email | mail | mail.yourdomain.com |

## Tips

- Use clear, short subdomains
- Keep consistent naming
- Consider client experience
- Document your choices
