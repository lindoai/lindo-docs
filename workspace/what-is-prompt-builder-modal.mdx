---

The Prompt Builder Modal is a full-screen interface for constructing detailed prompts using pre-built components.

## Accessing the Modal

Click **✨ Prompt Builder** button below the main prompt input.

## Modal Sections

### Sample Prompts

Pre-written prompts for common website types:
- Agency
- SaaS Product
- Restaurant
- Portfolio
- E-commerce
- Real Estate
- Fitness Studio
- And more...

Click any sample to use it as a starting point.

### Sample Sections

Page sections you can include:
- Hero
- Features
- About
- Services
- Testimonials
- Pricing
- FAQ
- Team
- Contact
- Footer

Select multiple sections to include in your prompt.

### Design Styles

Visual style presets:
- **Modern** - Clean, contemporary design
- **Minimal** - Simple, whitespace-focused
- **Brutalist** - Bold, raw aesthetic
- **Glass** - Glassmorphism effects
- **Playful** - Fun, colorful design
- **Elegant** - Sophisticated, refined

Select a style to influence the design direction.

## Using the Modal

1. **Browse sample prompts** - Find one close to your needs
2. **Select sections** - Check boxes for sections you want
3. **Choose design style** - Pick a visual direction
4. **Click Apply** - Adds selections to your prompt
5. **Customize** - Edit the generated prompt as needed
6. **Generate** - Submit to create your website

## Example Workflow

1. Click **✨ Prompt Builder**
2. Select "SaaS Product" sample prompt
3. Check: Hero, Features, Pricing, Testimonials, FAQ
4. Select "Modern" design style
5. Click **Apply to Prompt**
6. Edit: Change product name and description
7. Click submit to generate

## Benefits

- **Faster prompts** - Don't start from scratch
- **Complete prompts** - Include all important sections
- **Consistent results** - Proven prompt structures
- **Design guidance** - Style presets for direction

## Tips

- **Combine samples** - Mix elements from different samples
- **Add specifics** - Always customize with your details
- **Iterate** - Use as starting point, refine after generation
