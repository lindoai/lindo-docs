title: "How to View Client Websites"
description: "See all websites owned by a specific client"
---

View all websites that belong to a specific client in your workspace.

## Accessing Client Websites

1. Open **CRM** from sidebar
2. Select the client
3. Go to **Websites** tab
4. View all their websites

## Website List

The websites tab shows:

- Website name
- Status (Published/Draft)
- Creation date
- Last modified

## Actions Available

From the client's website list:

- Click to open in editor
- View website status
- See website details

## Tips

- Review client websites regularly
- Check for inactive sites
- Monitor website limits
- Help clients optimize their sites
