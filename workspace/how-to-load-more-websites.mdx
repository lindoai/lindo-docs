---
title: "How to Load More Websites"
description: "View additional websites beyond the initial display"
---

Load more websites when you have many sites in your workspace.

## Loading More

1. Scroll to bottom of websites grid
2. Click **Load More** button
3. Additional websites appear

## Pagination

Websites load in batches:

- Initial load shows first batch
- Click to load more
- Continue until all loaded

## Alternative: Search

For many websites:

1. Use the search input
2. Type website name
3. Results filter instantly

## Tips

- Use search for faster access
- Organize with clear naming
- Consider archiving unused sites
- Monitor website limits
