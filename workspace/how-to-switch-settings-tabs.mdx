Quickly navigate settings tabs with keyboard.

## Shortcuts

| Keys | Tab |
|------|-----|
| Cmd/Ctrl + 1 | First tab |
| Cmd/Ctrl + 2 | Second tab |
| Cmd/Ctrl + 3 | Third tab |
| Cmd/Ctrl + 4 | Fourth tab |
| Cmd/Ctrl + 5 | Fifth tab |
| Cmd/Ctrl + 6 | Sixth tab |

## How to Use

1. Open Settings modal
2. Press Cmd (Mac) or Ctrl (Windows) + number
3. Tab switches instantly

## Tips

- Faster than clicking
- Works in settings modal
- Learn tab order
