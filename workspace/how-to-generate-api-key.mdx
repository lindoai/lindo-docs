---
title: "How to Generate API Key"
description: "Creating new API key"
---

Create an API key for programmatic access.

## Prerequisites

- Business plan or higher

<Steps>
  <Step title="Open Settings">
    Click **Settings** in the sidebar.
  </Step>
  <Step title="Go to API Tab">
    Select the **API & Webhook** tab.
  </Step>
  <Step title="Click Generate Key">
    Click the "Generate API Key" button.
  </Step>
  <Step title="Name Your Key">
    Enter a descriptive name:
    - "Production API"
    - "Development Key"
    - "Integration Key"
  </Step>
  <Step title="Copy the Key">
    **Important:** Copy the key immediately. It's only shown once.
  </Step>
  <Step title="Store Securely">
    Save the key in a secure location:
    - Password manager
    - Environment variables
    - Secrets manager
  </Step>
</Steps>
## Key Format

API keys look like:
```
lindo_sk_live_xxxxxxxxxxxxxxxxxxxx
```

## Tips

- Copy immediately (shown once)
- Use descriptive names
- Create separate keys per use
- Store securely
