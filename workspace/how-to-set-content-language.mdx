---
title: "How to Set Content Language"
description: "Configure the language for generated website content"
---

Set the content language so the AI generates website text in your preferred language.

## What is Content Language?

Content language determines:

- The language of generated text
- Headings, paragraphs, buttons
- All written content on the site

## Setting the Language

1. Click the **(+)** button in prompt builder
2. Select **Language** tool
3. Choose your language
4. Write your prompt
5. Generate the website

## Available Languages

Lindo.ai supports 11 languages:

- English
- Spanish
- French
- German
- Italian
- Portuguese
- Dutch
- Polish
- Swedish
- Norwegian
- Danish

## Auto-Detect Option

You can also let the AI detect language from your prompt. If you write in Spanish, it may generate Spanish content.

## Tips

- Set language explicitly for best results
- Match language to your target audience
- Consider SEO for local markets
- Review generated content for accuracy

## Changing Language Later

To change language after generation:

1. Open the website editor
2. Use AI chat to request translation
3. Or regenerate specific sections
