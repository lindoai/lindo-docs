Configure AI tools when creating workflows to customize generated websites.

## Available Tools

When creating a workflow, you can set:

- **Language**: Content language
- **Mode**: Light or dark theme
- **Color Palette**: Custom colors

## Setting Tools

1. Open **Workflows** from sidebar
2. Click **Create Workflow**
3. Enter your prompt
4. Configure AI tools:
   - Select language
   - Choose mode
   - Set colors (optional)
5. Start workflow

## How Tools Affect Output

- **Language**: All generated text in selected language
- **Mode**: Website uses light or dark theme
- **Colors**: Custom color scheme applied

## Tips

- Set language for target audience
- Match mode to brand preference
- Use consistent colors across batch
- Test with single workflow first
