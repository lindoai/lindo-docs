Learn how to optimize your Lindo.ai websites for search engines using built-in SEO tools.

## Understanding SEO in Lindo.ai

Every website includes:
- Auto-generated sitemap
- Configurable robots.txt
- Schema markup support
- Page-level SEO settings

<Steps>
  <Step title="Configure Page SEO">
    For each page:

    1. Open the page in the editor
    2. Click the **Page Settings** icon in the preview header
    3. Fill in SEO fields:
       - **Title** - Page title for search results
       - **Description** - Meta description (150-160 characters)
       - **OG Image** - Social sharing image
  </Step>
  <Step title="Set Up Schema Markup">
    Schema helps search engines understand your content:

    1. Go to **Settings** → **SEO**
    2. Find the **Schema Markup** section
    3. Add JSON-LD structured data
    4. Common schemas:
       - Organization
       - LocalBusiness
       - Product
       - Article

    #
  </Step>
  <Step title="Configure Robots.txt">
    Control how search engines crawl your site:

    1. Go to **Settings** → **SEO**
    2. Enable **Manual Robots.txt**
    3. Edit the content

    #
  </Step>
  <Step title="Submit Sitemap">
    1. Go to **Settings** → **SEO**
    2. Copy the sitemap URL
    3. Submit to Google Search Console
    4. Submit to Bing Webmaster Tools
  </Step>
  <Step title="Configure Blog SEO">
    For blog-heavy sites:

    1. Go to **Settings** → **SEO** → **Blog Config**
    2. Set the blog path (e.g., /blog)
    3. Configure canonical URL
    4. Set blog heading and description
  </Step>
</Steps>
## Best Practices

### Titles
- Include primary keyword
- Keep under 60 characters
- Make them compelling

### Descriptions
- Summarize page content
- Include call-to-action
- Keep 150-160 characters

### Images
- Use descriptive alt text
- Optimize file sizes
- Use WebP format when possible

### Content
- Use heading hierarchy (H1, H2, H3)
- Include internal links
- Write for users first, search engines second
