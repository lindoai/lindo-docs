Create an online store with products and checkout.

## What You'll Build

- Home page with featured products
- Product pages
- Stripe checkout
- Thank you page

<Steps>
  <Step title="Generate Website">
    Enter this prompt:

    > "Create an e-commerce website for handmade jewelry. Include hero, featured products grid, about the maker, and contact. Elegant, minimal design."
  </Step>
  <Step title="Connect Stripe">
    1. Go to Settings → Integrations
    2. Connect Stripe account
    3. Complete verification
  </Step>
  <Step title="Add Products">
    For each product:
    1. Go to Pages tab
    2. Click Add Product
    3. Set name and description
    4. Set price
    5. Upload image
    6. Configure checkout options
  </Step>
  <Step title="Configure Checkout">
    For each product:
    - Enable/disable discount codes
    - Set shipping address collection
    - Configure thank you message
  </Step>
  <Step title="Test Checkout">
    1. Preview product page
    2. Click buy button
    3. Test payment flow
    4. Verify thank you page
  </Step>
  <Step title="Publish">
    Click Publish to go live.
  </Step>
</Steps>
## Tips

- High-quality product photos
- Clear descriptions
- Test checkout thoroughly
- Set up email notifications
