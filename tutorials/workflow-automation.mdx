Learn how to leverage workflows for automated, batch operations in Lindo.ai.

## What are Workflows?

Workflows are background processes that:

- Generate multiple websites at once
- Create bulk blog content
- Run without blocking your work
- Send notifications on completion

## Types of Workflows

### Website Workflows
Generate complete websites from prompts in the background.

### Blog Pipeline Workflows
Create multiple blog posts from a list of topics.

### Page Workflows
Generate individual pages for existing websites.

<Steps>
  <Step title="Access Workflows">
    1. Click the **Workflows** icon in the sidebar
    2. View workflow statistics:
       - Total workflows
       - Running
       - Completed
       - Errored
  </Step>
  <Step title="Create a Website Workflow">
    1. Click **Create Workflow**
    2. Enter your website prompt
    3. Configure AI tools:
       - Language
       - Mode (light/dark)
       - Color palette
    4. Enable email notification (optional)
    5. Click **Start**
  </Step>
  <Step title="Batch Import Websites">
    For multiple websites:

    1. Click **Batch Import**
    2. Download sample CSV template
    3. Fill in your data:
       - Website name
       - Prompt
       - Settings
    4. Upload completed CSV
    5. Start batch workflow

    #
  </Step>
  <Step title="Monitor Progress">
    Track your workflows:

    - **Running**: Currently processing
    - **Completed**: Successfully finished
    - **Error**: Failed (check details)

    Click **Refresh** to update status.
  </Step>
  <Step title="Handle Errors">
    If a workflow fails:

    1. Click on the errored workflow
    2. Review error message
    3. Common issues:
       - Invalid prompt
       - Credit shortage
       - Rate limiting
    4. Fix and retry if needed
  </Step>
  <Step title="Access Results">
    Once complete:

    1. Find new websites in your grid
    2. Open to review and edit
    3. Publish when ready
  </Step>
</Steps>
## Best Practices

- Start with small batches to test
- Use clear, specific prompts
- Monitor credit usage
- Enable notifications for long batches
- Review results before publishing

## Credit Considerations

Each workflow item consumes credits:

- Website: 30 credits (Website Builder agent)
- Blog post: 30 credits (Blog Writer agent)
- Page: 30 credits (Page Builder agent)

Plan batch sizes according to available credits.

## Automation Tips

- Schedule workflows during off-hours
- Use consistent naming conventions
- Create prompt templates for efficiency
- Document successful prompts
- Build a library of reusable workflows
