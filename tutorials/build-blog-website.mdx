Build a complete blog website with category organization, featured posts, and newsletter signup.

## What You'll Create

A professional blog featuring:
- Featured post hero section
- Category navigation
- Post grid layout
- Author profiles
- Newsletter subscription

<Steps>
  <Step title="Generate the Blog">
    Enter this prompt in your workspace:

    ```
    Create a modern blog website for "The Daily Insight" with:
    - Hero featuring latest post with large image
    - Category navigation: Technology, Lifestyle, Business, Travel
    - Recent posts grid (6 posts with thumbnails)
    - Popular posts sidebar
    - Newsletter signup section
    - About the blog section
    - Footer with categories and social links
    - Clean, readable typography
    ```
  </Step>
  <Step title="Create Blog Posts">
    Navigate to Pages tab and click "Create Blog":

    ```
    Write a blog post about "10 Productivity Tips for Remote Workers" with:
    - Engaging introduction
    - Numbered list format
    - Practical actionable tips
    - Conclusion with call-to-action
    ```

    Use the Blog Writer agent for AI-assisted content creation.
  </Step>
  <Step title="Design Post Layout">
    Use Block Editor to customize the blog post template:

    ```
    Style the blog post layout with:
    - Large featured image at top
    - Post title in large heading
    - Author name, date, reading time
    - Category tag
    - Body text with good line spacing
    - Pull quotes for key points
    - Related posts at bottom
    ```
  </Step>
  <Step title="Add Category Pages">
    Create category landing pages:

    ```
    Create a category page for "Technology" with:
    - Category title and description
    - Filter by subcategory
    - Post grid showing all tech posts
    - Pagination for older posts
    ```
  </Step>
  <Step title="Set Up Newsletter">
    Add a newsletter signup section:

    ```
    Design a newsletter signup with:
    - Compelling headline: "Get Weekly Insights"
    - Brief description of what subscribers receive
    - Email input field
    - Subscribe button
    - Privacy note
    ```
  </Step>
  <Step title="Configure Blog Settings">
    Open Settings → SEO → Blog Config:

    1. Set blog path (e.g., `/blog`)
    2. Add blog heading
    3. Configure canonical URL
    4. Add menu links for categories
  </Step>
  <Step title="Create Blog Pipeline">
    For bulk content, use Workflows:

    1. Open Workflows from sidebar
    2. Click "Create Workflow"
    3. Select blog pipeline
    4. Enter multiple blog topics
    5. Enable email notification
    6. Start workflow
  </Step>
</Steps>
## Tips for Blog Websites

- Use consistent featured image sizes
- Write compelling meta descriptions
- Include internal links between posts
- Add social sharing buttons
- Optimize images for fast loading
