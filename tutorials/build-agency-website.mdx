Create a complete agency website with multiple pages.

## What You'll Build

- Home page with hero and services
- About page with team
- Services page
- Portfolio/Work page
- Contact page

<Steps>
  <Step title="Generate Website">
    Enter this prompt:

    > "Create a multi-page website for a digital marketing agency. Include Home, About, Services, Portfolio, and Contact pages. Modern, professional design with dark theme."
  </Step>
  <Step title="Review Pages">
    Check each generated page:
    - Home: Hero, services overview, testimonials
    - About: Story, team, values
    - Services: Service cards, details
    - Portfolio: Project showcase
    - Contact: Form, info
  </Step>
  <Step title="Customize Content">
    Edit each page with AI:
    - Update company name
    - Add real services
    - Include team photos
    - Add portfolio items
  </Step>
  <Step title="Configure Settings">
    1. Upload logo
    2. Set colors
    3. Add social links
    4. Configure SEO
  </Step>
  <Step title="Connect Domain">
    1. Go to Settings → Domains
    2. Add custom domain
    3. Configure DNS
    4. Verify connection
  </Step>
  <Step title="Publish">
    Click Publish to go live.
  </Step>
</Steps>
## Tips

- Use real content
- Add testimonials
- Include case studies
- Optimize for SEO
