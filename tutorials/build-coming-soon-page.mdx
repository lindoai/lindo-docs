---

Build a coming soon page that captures leads and builds excitement for your upcoming launch.

## What You'll Create

A launch page featuring:
- Countdown timer
- Email capture form
- Social media links
- Brand preview

<Steps>
  <Step title="Generate the Page">
    Enter this prompt in your workspace:

    ```
    Create a coming soon landing page for "Nexus App" with:
    - Large logo placeholder at top
    - Headline: "Something Amazing is Coming"
    - Subheadline with launch date hint
    - Countdown timer showing days, hours, minutes, seconds
    - Email signup form: "Be the first to know"
    - Social media icons
    - Dark gradient background with subtle animation
    ```

    Select **Single-Page** mode.
  </Step>
  <Step title="Design the Countdown">
    Use Element Editor to style the timer:

    ```
    Style the countdown timer with:
    - Large numbers in bold font
    - Labels below: Days, Hours, Minutes, Seconds
    - Subtle glow effect
    - Responsive sizing for mobile
    ```
  </Step>
  <Step title="Create Email Capture">
    Add a compelling signup form:

    ```
    Design the email signup with:
    - Input field with placeholder "Enter your email"
    - Submit button: "Notify Me"
    - Privacy note below
    - Success message on submit
    ```
  </Step>
  <Step title="Add Social Proof">
    Include early interest indicators:

    ```
    Add a section showing:
    - "Join 1,000+ people waiting"
    - Social media follow buttons
    - Share buttons to spread the word
    ```
  </Step>
  <Step title="Include Teaser Content">
    Give visitors a preview:

    ```
    Add a features preview section with:
    - 3 icon cards showing key features
    - Brief descriptions
    - "Learn more at launch" text
    ```
  </Step>
  <Step title="Set Up Notifications">
    Configure email collection:

    1. Connect an integration (HubSpot, etc.) in Settings → Integrations
    2. Or use the Website CRM to track signups
    3. Set up webhook for real-time notifications
  </Step>
  <Step title="Optimize for Sharing">
    In Settings → SEO, add Open Graph tags:

    - Compelling title
    - Description mentioning launch
    - Preview image
    - Social sharing optimization
  </Step>
</Steps>
## Tips for Coming Soon Pages

- Create urgency with countdown
- Offer early access incentive
- Keep the design simple
- Test on mobile devices
- Update regularly with progress
