Build a professional fitness studio website with class schedules, trainer profiles, and membership options.

## What You'll Create

A complete fitness website featuring:
- Dynamic hero with call-to-action
- Class schedule display
- Trainer profiles
- Membership pricing tiers
- Contact and trial signup forms

<Steps>
  <Step title="Generate the Website">
    Enter this prompt in your workspace:

    ```
    Create a modern fitness studio website for "FitLife Gym" with:
    - Hero section with motivational headline and "Start Free Trial" button
    - Class types: yoga, HIIT, spinning, strength training, pilates
    - Weekly class schedule grid
    - 4 trainer profiles with specialties
    - 3 membership tiers: Basic, Premium, Elite
    - Testimonials from members
    - Location with hours and contact form
    - Dark theme with energetic accent colors
    ```
  </Step>
  <Step title="Design Class Schedule">
    Use Block Creator to add a detailed schedule:

    ```
    Create a weekly class schedule table with:
    - Days as columns (Mon-Sun)
    - Time slots as rows (6am, 8am, 10am, 12pm, 5pm, 7pm)
    - Class name and instructor in each cell
    - Color-coded by class type
    - Hover effect showing class description
    ```
  </Step>
  <Step title="Create Membership Pricing">
    Add compelling pricing cards:

    ```
    Design 3 membership pricing cards:

    Basic ($29/month):
    - Access to gym floor
    - 2 group classes/week
    - Locker room access

    Premium ($59/month):
    - Unlimited classes
    - Personal training session monthly
    - Nutrition consultation
    - Sauna access

    Elite ($99/month):
    - All Premium benefits
    - Unlimited personal training
    - Priority class booking
    - Guest passes
    ```
  </Step>
  <Step title="Add Trainer Profiles">
    Enhance trainer section with the Block Editor:

    ```
    Update trainer cards to show:
    - Professional photo
    - Name and title
    - Certifications (ACE, NASM, etc.)
    - Specialties list
    - Years of experience
    - "Book Session" button
    ```
  </Step>
  <Step title="Create Trial Signup Form">
    Use Element Editor to build the form:

    ```
    Create a free trial signup form with:
    - Name field
    - Email field
    - Phone field
    - Fitness goals dropdown
    - Preferred class type
    - "Claim Free Week" button
    ```
  </Step>
  <Step title="Set Up Products">
    Navigate to Pages → Products and create membership products:

    1. Click "Create Product"
    2. Set name: "Premium Membership"
    3. Set price: $59
    4. Set frequency: Monthly
    5. Add description and image
  </Step>
</Steps>
## Tips for Fitness Websites

- Use action-oriented photography
- Highlight transformation stories
- Make class booking easy
- Show real member results
- Include COVID safety measures if applicable
