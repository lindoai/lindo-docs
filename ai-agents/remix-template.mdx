Transform your design with a new style.

## What It Does

Redesigns pages:
- New visual style
- Updated layout
- Fresh look
- Maintains content

## Credit Cost

**30 credits** per remix.

## When to Use

- Want a new look
- Refreshing design
- Trying different styles

## How to Use

1. Open page in editor
2. Select Remix Template agent
3. Describe new style
4. AI transforms design

## Best Prompts

Include:
- Desired style
- Reference examples
- What to keep

## Example

> "Remix this page with a minimalist, modern style using more whitespace and subtle colors"

## Tips

- Describe target style
- Keep content intact
- Review changes
