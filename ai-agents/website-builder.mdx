Generate complete websites from text descriptions.

## What It Does

Creates full websites including:
- Multiple pages
- All sections
- Content and images
- Navigation
- Styling

## Credit Cost

**0 credits** - Free to generate websites.

## When to Use

- Starting a new website
- Creating from scratch
- Full site generation

## How to Use

1. Go to workspace
2. Enter prompt in main input
3. Add optional tools
4. Click send
5. Website generates

## Best Prompts

Include:
- Business type
- Target audience
- Desired pages
- Style preferences

## Example

> "Create a modern SaaS landing page for a project management tool with hero, features, pricing, testimonials, and CTA sections"

## Tips

- Be specific
- Mention pages needed
- Include style preferences
