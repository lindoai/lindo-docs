Some AI agents have been retired and are no longer available.

## Retired Agents

The following agents have been removed:

- Duplicate Page Agent
- Product Page Agent
- Legacy Convertor Agent
- Page Editor Agent
- Regenerator (Remix Template) Agent
- Header Creator Agent
- Footer Creator Agent
- Advanced Page Builder Agent
- Advanced Website Agent
- Shop Agent
- Pricing Block Agent
- Support Agent
- Blog Research, Strategy, Writing, Critique, and Enhancement Agents
- Website Builder (single-page) Agent

## Available Alternatives

For most retired agent functions, use:

- Page Builder for new pages
- Block Creator for sections
- Block Editor for modifying sections
- Header Editor / Footer Editor for header and footer changes
- Multipage Website Builder for full websites
