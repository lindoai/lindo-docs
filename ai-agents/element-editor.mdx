Make precise changes to individual elements.

## What It Does

Edits specific elements:
- Text content
- Styling
- Colors
- Spacing
- Individual components

## Credit Cost

**10 credits** per edit.

## When to Use

- Fine-tuning details
- Changing specific text
- Adjusting individual elements
- Precise modifications

## How to Use

1. Select an element in preview
2. Choose Element Editor agent
3. Describe the change
4. AI updates element

## Best Prompts

Be specific:
- "Change this button to blue"
- "Make this text larger"
- "Add padding to this section"

## Tips

- Select element first
- Be precise
- One change at a time
