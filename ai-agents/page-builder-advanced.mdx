The Page Builder Agent (Advanced) creates complete pages with real-time streaming preview, showing you the page as it's being generated.

## Overview

- **Credit Cost**: 30 credits
- **Best For**: Creating new pages with live preview
- **Output**: Complete page with all sections

## How It Works

1. Select the Page Builder (Advanced) agent
2. Describe the page you want to create
3. Watch the page generate in real-time via streaming
4. The page appears section by section as it's built

## Streaming Preview

Unlike the standard Page Builder, the Advanced version shows you the page being built in real-time. This lets you:

- See progress as it happens
- Understand how the AI interprets your prompt
- Get visual feedback during generation

## When to Use

- Creating new pages from scratch
- When you want to see the generation process
- For complex pages with multiple sections
- When you need immediate visual feedback

## Example Prompts

- "Create a pricing page with three tiers: Basic, Pro, and Enterprise"
- "Build an about page with team section, company history, and values"
- "Generate a services page showcasing our web design offerings"

## Tips

- Be specific about sections you want included
- Mention the number of items (e.g., "3 pricing tiers", "4 team members")
- Include tone and style preferences
