Improve AI results with reference images.

## What It Does

Upload images to:
- Show design inspiration
- Provide visual reference
- Guide AI output
- Demonstrate desired style

## How to Use

1. Click image upload in chat
2. Select image file
3. Add text description
4. Send to AI

## Best Uses

### Design Reference

Upload a design you like:
> "Create a hero section similar to this style"

### Logo/Brand

Upload your logo:
> "Add my logo to the header"

### Style Guide

Upload brand assets:
> "Use these colors and style"

## Tips

- Clear, high-quality images
- Describe what to do with it
- Combine with text prompts
