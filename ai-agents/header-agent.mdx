Create website headers and navigation.

## What It Does

Generates headers:
- Navigation menus
- Logo placement
- CTA buttons
- Mobile menus

## Credit Cost

**10 credits** per header.

## When to Use

- Creating new header
- Redesigning navigation
- Adding header to page

## How to Use

1. Select Header agent
2. Describe header needs
3. AI generates header

## Best Prompts

Include:
- Navigation items
- Style preference
- CTA button
- Logo position

## Example

> "Create a modern header with logo left, navigation center, and 'Get Started' button right"

## Tips

- List nav items
- Specify style
- Include CTA
