Create website footers.

## What It Does

Generates footers:
- Navigation links
- Social icons
- Contact info
- Copyright
- Newsletter signup

## Credit Cost

**10 credits** per footer.

## When to Use

- Creating new footer
- Redesigning footer
- Adding footer to page

## How to Use

1. Select Footer agent
2. Describe footer needs
3. AI generates footer

## Best Prompts

Include:
- Link sections
- Social platforms
- Contact info
- Style preference

## Example

> "Create a footer with 4 columns: Company, Products, Resources, Contact. Include social icons and newsletter signup"

## Tips

- List link sections
- Include social links
- Add contact info
