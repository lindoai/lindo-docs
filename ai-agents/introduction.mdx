AI Agents are specialized models optimized for different website creation and editing tasks. Each agent handles a specific type of work—some create entire pages, some generate individual sections, some edit existing content, and some write blog posts.

## Available Agents

### Creation Agents

| Agent | Purpose | Credits |
|-------|---------|---------|
| Website Builder | Generate multi-page websites | 10x |
| Page Builder | Create page structure and sections | 10 |
| Block Creator | Generate new sections | 10 |
| Simple Blog | Write blog posts | 10 |

### Editing Agents

| Agent | Purpose | Credits |
|-------|---------|---------|
| Block Editor | Modify existing sections | 10 |
| Element Editor | Edit individual elements | 10 |

## How It Works

<Steps>
  <Step title="Describe what you want">
    Type your request in the AI Chat panel. Be specific about what you want to create or change.
  </Step>
  <Step title="System selects the agent">
    Based on your selection (block, element, or nothing) and request type (add, edit, modify), the system automatically chooses the appropriate agent.
  </Step>
  <Step title="Agent processes your request">
    The selected agent generates or modifies content based on your description.
  </Step>
  <Step title="Result appears in preview">
    The changes show up in the live preview. You can continue refining with follow-up requests.
  </Step>
</Steps>

## Automatic vs Manual Selection

The system usually picks the right agent automatically. If you want to override:

1. Look for the agent selector in the chat panel
2. Click to see available agents
3. Choose the one you want

## Best Practices

- **Let the system choose** — It usually picks correctly based on context
- **Be specific** — Clear requests get better results
- **Select first** — Selecting elements before typing helps the AI understand context
- **Iterate** — Refine with follow-up requests rather than starting over
