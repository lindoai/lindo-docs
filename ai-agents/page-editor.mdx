Make comprehensive changes to entire pages.

## What It Does

Edits full pages:
- Multiple sections
- Content updates
- Layout changes
- Style modifications

## Credit Cost

**15 credits** per edit.

## When to Use

- Major page updates
- Multiple section changes
- Page-wide modifications

## How to Use

1. Open page in editor
2. Select Page Editor agent
3. Describe changes
4. AI updates page

## Best Prompts

Include:
- What to change
- Sections affected
- Desired outcome

## Example

> "Update this page to have a more modern look with larger images and more whitespace"

## Tips

- Describe overall changes
- Be clear about scope
- Review all sections
