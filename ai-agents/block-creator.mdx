The Block Creator Agent generates new sections (blocks) for your website pages.

## Overview

When you need to add a new section—like a features grid, testimonials carousel, or pricing table—the Block Creator handles it.

## Credit Cost

**10 credits** per block generated.

## What It Creates

Each block is a complete HTML `<section>` element with:
- Responsive container (max-w-6xl mx-auto)
- Light and dark mode styling
- Scroll-triggered animations (data-motion)
- High-quality images from Pexels
- Icons from Iconify
- Mobile-friendly design

## When to Use

Use Block Creator when you want to:
- Add a new section to a page
- Create a features grid
- Build a testimonials section
- Add a pricing table
- Create an FAQ accordion
- Build a team members section
- Add a stats/metrics display

## How to Use

### Option 1: Direct Request

Type in the AI Chat:
```
Add a testimonials section with 3 customer reviews
```

### Option 2: Be Specific

For better results:
```
Add a features section with 6 cards in a 3-column grid.
Each card should have an icon, title, and short description.
Features: Fast Performance, Easy Setup, 24/7 Support,
Secure Data, Custom Themes, API Access.
```

## Example Prompts

**Features section:**
```
Add a features section with 4 benefit cards showing
why customers should choose us
```

**Testimonials:**
```
Add a testimonials carousel with 3 customer quotes
including photos and company names
```

**Pricing:**
```
Add a pricing section with 3 tiers: Basic ($9),
Pro ($29), Enterprise (Contact Us)
```

**FAQ:**
```
Add an FAQ accordion with 5 common questions about
our service
```

**Team:**
```
Add a team section showing 4 team members with
photos, names, and titles
```

## Tips

- **Specify numbers** - "4 features" not just "features"
- **Mention layout** - "3-column grid" or "carousel"
- **Include content** - Provide actual text when possible
- **Reference style** - "Modern" or "minimal"

## Technical Details

The agent:
- Uses Tailwind CSS v4 utility classes
- Applies neutral color palette
- Generates unique section IDs
- Includes hover animations
- Ensures proper spacing (py-20 sm:py-24)
