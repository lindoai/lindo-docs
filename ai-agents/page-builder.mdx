Create complete pages with real-time preview.

## What It Does

Generates full pages:
- All sections
- Content
- Images
- Layout
- Styling

## Credit Cost

**30 credits** per page.

## When to Use

- Adding new pages
- Creating landing pages
- Building complete pages

## How to Use

1. Open website editor
2. Select Page Builder agent
3. Describe the page
4. Watch it generate

## Best Prompts

Include:
- Page purpose
- Sections needed
- Content focus
- Style preferences

## Example

> "Create an About page with company story, team section, values, and timeline"

## Tips

- Describe sections
- Be specific about content
- Watch stream preview
