Pick the best agent for your task.

## Quick Guide

| Task | Agent | Credits |
|------|-------|---------|
| New website | Website Builder | 10x (10 per page) |
| New page | Page Builder | 10 |
| New section | Block Creator | 10 |
| Edit section | Block Editor | 10 |
| Edit element | Element Editor | 10 |
| Quick style | Tailwind Class | 10 |
| Blog post | Blog Writer | 10 |
| New header | Header Agent | 10 |
| New footer | Footer Agent | 10 |
| Page-wide edit | Page Editor | 10 |
| New style | Remix Template | 10 |
| Refresh look | Block Editor | 10 |

## Decision Tree

### Creating New Content

- Full website → Website Builder
- Full page → Page Builder
- New section → Block Creator
- Blog post → Blog Writer

### Editing Existing

- Whole section → Block Editor
- Single element → Element Editor
- Quick style → Tailwind Class
- Full page → Page Editor

### Redesigning

- New style → Remix Template
- Refresh section → Block Editor

## Tips

- Start with cheapest option
- Use specific agents
- Match agent to task
