The Block Editor Agent modifies existing sections on your page.

## Overview

When you need to change an existing section—update text, change layout, modify styling—the Block Editor handles it.

## Credit Cost

**10 credits** per edit.

## What It Does

- Updates text content
- Changes images
- Modifies layout
- Adjusts styling
- Adds/removes elements within a section
- Changes colors and spacing

## When to Use

Use Block Editor when you want to:
- Change headlines or copy
- Update images in a section
- Modify the layout (columns, spacing)
- Change colors or styling
- Add elements to existing sections
- Remove elements from sections

## How to Use

### Option 1: Select and Edit

1. Switch to **Select** mode
2. Click on the section to edit
3. Type your edit request:
```
Change the headline to "Build Faster"
```

### Option 2: Describe the Section

Without selecting:
```
In the hero section, change the headline to "Build Faster"
```

## Example Prompts

**Change text:**
```
Update the hero headline to "Your Success Starts Here"
```

**Change images:**
```
Replace the hero image with a photo of a modern office
```

**Modify layout:**
```
Change the features from 3 columns to 2 columns
```

**Add elements:**
```
Add a "Watch Demo" button next to the main CTA
```

**Change styling:**
```
Add a gradient background to this section
```

**Update content:**
```
Update the testimonials with these new quotes: [quotes]
```

## Tips

- **Select first** - Helps the AI know which section
- **Be specific** - "The second button" not "a button"
- **One change at a time** - Easier to review
- **Reference existing** - "Keep the layout but change..."

## Block Editor vs Block Creator

| Block Editor | Block Creator |
|--------------|---------------|
| Modifies existing | Creates new |
| Preserves structure | Builds from scratch |
| Updates content | Generates content |
| 10 credits | 10 credits |
