# What's New in Lindo

<Note>
**July 2026** - Latest Release
</Note>

Lindo now supports native forms powered by FormBnB. Connect FormBnB as an MCP connector, then ask the website agent to connect all native, agent-designed forms to your FormBnB account with one prompt.

---

## July 2026

### 🧾 Native Forms with [FormBnB](https://formbnb.com)

Lindo now supports native website forms that can be connected directly to **[FormBnB](https://formbnb.com)**. Add FormBnB as an MCP connector in the website editor, then use one prompt — “Connect all of my website forms to my FormBnB account.” — and the agent will wire native, agent-designed forms to your FormBnB account.

This gives agencies a simple form workflow without embeds or custom backend code: the agent designs the forms, FormBnB handles submissions, spam protection, email alerts, and downstream workflows.

---

## June 2026

### ✨ Lindo V3 Vibe Coding

A completely new Lindo experience. Open a website and just tell the agent what you want — it understands your whole site and makes the change. Preview-first editing, MCP data connectors, one-click website cloning, and simple credit-based pricing while we roll it out.

### 🤖 Website-Aware Agent

Our agent now understands your **entire website** — every page, your `DESIGN.md`, and all website settings. You can ask it to update the current page, make changes across all pages, change branding colors, or even redesign the full website, all in a single conversation.

### 👁️ Preview-First Editor

When you open a website in the editor, it now starts in **Preview mode**. You can navigate between pages just like a published website and check every change naturally. You don't even need to activate the editor anymore — just ask the agent what you want changed, mention the section or area, and it's smart enough to find it. You can still use **Editor mode**, **Select mode**, and element/block selection when you want more control.

### 🔌 MCP Data Connectors

You can now connect **Supabase, Airtable, Stripe, and Shopify** as MCP connectors when creating a website or inside the editor. This means you can ask the agent to build or update websites based on connected data — create client dashboards with Supabase, build storefronts from your Shopify products, and much more.

### 📑 Website Cloning

You can now clone any website, including **Lindo Core/Legacy** websites. It's a great way to move your existing projects into the new Agentic experience — you can import all your client websites from other platforms into Lindo in a few hours.

### 🪙 Credit-Based Pricing

Lindo is credit-based: instead of fixed per-action prices, credits are consumed
according to how much work the agent does and how complex the task is. A small
edit uses far fewer credits than building a full page or an entire multi-page
website. Our goal is to make Lindo credits cheaper over time, not more expensive
— so this is a great time to migrate your Legacy or client websites to Agentic.

### 🧠 Lindo MCP Server, CLI & Automations

We've added the **Lindo MCP server** and **CLI**, and you can now build agency automations with **n8n** and **Zapier**. Connect your lead forms to your workspace and automatically create websites for clients when they submit a form. The MCP server exposes Lindo's tools to Claude, ChatGPT, Perplexity, Cursor, and any MCP-compatible client.

## April 2026

### 📅 Scheduled Workflows

Workflows now support scheduling. Set a future date and time when creating a website, page, or blog workflow — it will run automatically at the scheduled time. Edit or cancel scheduled workflows anytime before execution. Batch CSV imports also support a `schedule_at` column for bulk scheduling. [Learn more →](/workspace/what-are-workflows)

### 🔍 Website SEO

- **Workflow table improvements** — Workflow list now shows Created date, Schedule date, and Duration columns.
- **Edit scheduled workflows** — Change the prompt or reschedule date for pending workflows.

### Store-level Shop Activation

You can now enable **Shop** directly from each website's Settings modal. This makes ecommerce activation website-specific instead of workspace-wide, which is much better for agencies managing mixed portfolios.

### Shop Documentation & Navigation

The docs now include a dedicated **Shop** section in Website Editor documentation. Products now live under Shop so the full commerce flow is easier to follow from activation to checkout configuration.

### Stripe Coupons Guide

Shop settings now guide you to Stripe coupons, making it easier to create discounts and then activate discount-code collection on the right products.

### Stripe Inventory Workflow

Inventory guidance now points merchants to **Stockify** on the Stripe Marketplace for stock management and operational workflows.

### Stripe Shipping Rates in Checkout

If a checkout includes at least one product that requires shipping, Lindo now passes all active shipping rates from the connected Stripe account into Stripe Checkout.

### 🌐 Global Theme & Blog Header/Footer

- **Global fonts & dark/light mode** — Set your website's font and color mode once, and all published pages follow it automatically. No need to republish — changes take effect instantly at the edge.
- **Website header/footer for blogs** — Blog posts and the blog list page can now use your website's global header and footer instead of the default blog navigation. Enable it from Blog Config in website settings, or leave the blog menu links empty to use the website header/footer by default. Already-published blogs update automatically.

### 📊 Analytics Improvements

- **Bot traffic filtering** — A new "Hide bots" toggle in both workspace and website analytics filters out bot/crawler traffic so you only see real human visitors. Enabled by default.
- **Website filter in workspace analytics** — Search and select a specific website from the workspace analytics modal to view its individual traffic, top pages, countries, browsers, and more without leaving the overview.
- **Horizontal scroll on data tables** — Analytics tables with long domain names or referrer URLs now scroll horizontally instead of clipping content.

### ✏️ Editor Improvements

- **Font name labels** — Font options in the editor theme panel now display the font name below each preview, so you can identify fonts at a glance.
- **SEO character guidance** — Page title and meta description fields in page settings now show a live character counter with recommended ranges (15–70 for titles, 120–170 for descriptions). Fields are capped at the max to prevent over-length entries.

### 🔄 Page Persistence on Refresh

Refreshing the website editor no longer resets you to the first page. The currently selected page is now persisted in the URL, so you stay on the same page after a browser refresh.

### 🤖 Semi-Agentic Support

Our support system is now powered by AI agents. When you submit a feature request, bug report, or need guidance, our agents immediately start preparing answers and update our internal technical logs so the team is notified. Once we verify the response, the agents send the reply and begin preparing any related product updates. This cuts our response workflow by roughly 90%. We're just getting started.

### 🌐 Domain Connect for Domains

Domains now support Domain Connect for one-click connection. Cloudflare is currently supported, so you can connect and configure DNS without manually copying records.

### 🏷️ Simpler White Label Domain Setup

White label domain configuration now needs only 5 DNS records instead of 9, making branded client portals faster to launch and easier to maintain.

### 🔗 Leaner Website Domain Configuration

Website domain setup now needs only 2 DNS records instead of 3, so pointing live sites is simpler and less error-prone.
